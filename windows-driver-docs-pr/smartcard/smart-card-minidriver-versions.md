# <a name="smart-card-minidriver-versions"></a>智能卡微型驱动程序版本


智能卡微型驱动程序编写为与特定版本。 每个版本中介绍了新特性和功能。 若要符合特定版本，微型驱动程序将支持其规定的版本与所有以前版本中包括的功能。

# <a name="minidriver-version-707-features"></a>微型驱动程序版本 7.07 功能


在此版本中引入以下功能。

## <a name="span-idgenericinboxminidriverthatsupportsthegidscardedgespanspan-idgenericinboxminidriverthatsupportsthegidscardedgespanspan-idgenericinboxminidriverthatsupportsthegidscardedgespan-generic-inbox-minidriver-that-supports-the-gids-card-edge"></a><span id="_Generic_Inbox_minidriver_that_supports_the_GIDS__card_edge"></span><span id="_generic_inbox_minidriver_that_supports_the_gids__card_edge"></span><span id="_GENERIC_INBOX_MINIDRIVER_THAT_SUPPORTS_THE_GIDS__CARD_EDGE"></span> 泛型收件箱微型驱动程序支持 GID 卡边缘


-   泛型收件箱微型驱动程序支持通用标识设备规范 (GIDS) 卡边缘

    Windows 8 包含泛型微型驱动程序支持 GID 卡边缘与 Microsoft 电气配置文件。 请参阅[Windows 收件箱智能卡微型驱动程序](windows-inbox-smart-card-minidriver.md)过程的详细信息对于该 Windows 遵循来识别并对收件箱微型驱动程序的 GID 网卡。 泛型微型驱动程序也会提供从 Windows 7 Service Pack 1 (SP1) 开始。

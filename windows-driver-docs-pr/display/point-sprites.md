# <a name="point-sprites"></a>点 Sprite


## <span id="ddk_point_sprites_gg"></span><span id="DDK_POINT_SPRITES_GG"></span>


DirectX 8.0 引入了对点 sprite 的支持。 点 sprite 是点的允许的大小来呈现状态或顶点组件为指定的基本点呈现的扩展。 加速时, 点 sprite 屏幕空间四边形格式正确的两个三角形，因为在硬件中呈现和呈现状态，如使用纹理和混合。

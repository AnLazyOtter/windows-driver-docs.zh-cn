# <a name="span-iddisplaywindows8in-boxgraphicsdriverpreferredspanwindows-8-in-box-graphics-driver-preferred"></a><span id="display.windows_8_in-box_graphics_driver_preferred_"></span>首选的 Windows 8 中框图形驱动程序


在此方案中，Windows 8 中框图形驱动程序是首选通过 Windows 7 或较旧的图形驱动程序。

在 Windows 8 升级安装中，如果图形硬件涵盖的 Windows 8 内置驱动程序，从早期的 Windows 版本的图形驱动程序将不迁移到 Windows 8。 甚至当较旧的图形驱动程序是 4 个部分匹配项 （图形硬件 + 模型特定的 pc） 和 Windows 8 中框图形驱动程序是只有两个部分构成匹配项 （仅针对图形硬件），也是如此。 这是因为 Windows 8 中现成驱动程序具有较好功能的分数比任何内置驱动程序或它们在以前的 Windows 版本。 若要了解驱动程序选择条件，请参阅[驱动程序匹配条件](driver-matching-criteria.md)。 如果将 Windows 8 认证的驱动程序在 Windows 7 上安装 Windows 8 升级安装之前，将迁移该驱动程序。

**请注意**  现成图形驱动程序将永远不会将迁移到 Windows 8 中，即使没有图形硬件不 Windows 8 中框涵盖的 Windows 7。 在这种情况下，Windows 8 使用 Microsoft 基本显示驱动程序 (MSBDD)。

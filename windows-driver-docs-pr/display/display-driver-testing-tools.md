# <a name="display-driver-testing-tools"></a>显示驱动程序测试工具


## <span id="ddk_display_driver_testing_tools_gg"></span><span id="DDK_DISPLAY_DRIVER_TESTING_TOOLS_GG"></span>


除了提供调试程序，Windows 2000 和更高版本的操作系统版本提供以下用于测试和调试您的显示驱动程序的工具：

-   *newdisp* -可以动态地重新加载您的驱动程序，而无需重新启动系统。 请参阅[NewDisp:显示驱动程序的动态重新加载](newdisp--dynamic-reload-of-a-display-driver.md)有关详细信息。

-   在控制面板中的显示计划-中的滑块控件，可以动态更改 GDI 将在您的驱动程序中调用加速功能。 请参阅[动态更改的允许驱动程序加速程序](dynamic-change-of-permitted-driver-accelerations.md)有关详细信息。

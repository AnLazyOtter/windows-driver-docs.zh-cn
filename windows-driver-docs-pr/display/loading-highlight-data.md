# <a name="loading-highlight-data"></a>正在加载突出显示数据


## <span id="ddk_loading_highlight_data_gg"></span><span id="DDK_LOADING_HIGHLIGHT_DATA_GG"></span>


[ **DXVA\_突出显示**](https://msdn.microsoft.com/library/windows/hardware/ff563979)结构指定的子画面，突出显示的矩形区域，并与 DCCMD 数据和 DPXD 面一起使用来创建一个 alpha 值混合处理的图面。 突出显示数据的格式与 DVD ROM 规范兼容的方式。 DVD 子画面定义和数据字段的解释的进一步说明，请参阅*DVD 规范为只读磁盘：第 3 部分-视频规范 (v。1.11，1999 年 5)*。

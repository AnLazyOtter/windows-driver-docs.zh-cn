# <a name="general-install-section-directives"></a>常规安装部分指令


这是对开箱或生产/发布的二进制文件、 服务、 regadd 或 delreg 部分通常属于零售 WHQL 驱动程序包的所有引用程序未都列在 Windows 中内置驱动程序包的常规提醒。

不引用任何内容所需的 OpenGL ICDs、 OpenCL、 控制面板、 帮助文件、 现成可用的服务、 轮询应用程序和等等。

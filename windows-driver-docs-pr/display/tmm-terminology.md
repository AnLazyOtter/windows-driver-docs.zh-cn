# <a name="tmm-terminology"></a>TMM 术语


以下是用于 TMM 主要术语：

<span id="Clone_View"></span><span id="clone_view"></span><span id="CLONE_VIEW"></span>**克隆视图**  
显示模式，其中附加到图形适配器的所有活动的监视器显示主显示器。

<span id="COM"></span><span id="com"></span>**COM**  
组件对象方法-提供二进制标准，用于链接组件。

<span id="External_Only"></span><span id="external_only"></span><span id="EXTERNAL_ONLY"></span>**仅外部**  
显示在其中会显示在外部显示器设备而不是内部的显示设备的便携式计算机很常见的配置。

<span id="Single_View"></span><span id="single_view"></span><span id="SINGLE_VIEW"></span>**单一视图**  
其中只有一个视图显示在一个监视器显示模式。

<span id="Topology"></span><span id="topology"></span><span id="TOPOLOGY"></span>**拓扑**  
指定哪些源显示在图形适配器的目标上的信息。

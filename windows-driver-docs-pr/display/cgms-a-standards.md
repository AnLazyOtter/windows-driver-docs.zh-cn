# <a name="cgms-a-standards"></a>CGMS-A 标准


多个标准定义了内容的生成管理系统模拟 (CGMS A) 保护类型。 不同国家和地区使用各种版本的 CGMS a。 硬件供应商必须确保其显示微型端口驱动程序支持相应 CGMS 一个版本。 例如，若要在日本使用的图形适配器的驱动程序可能应支持关联的单选行业和企业 (ARIB) TR B15 标准，它是数字卫星广播该操作准则。 但是，若要在美国使用的图形适配器的驱动程序应支持国际电工委员会 (IEC) 61880 标准或使用者电子设备关联 (CEA) CEA-608-B 标准。 图形适配器的显示微型端口驱动程序支持的标准取决于适配器传输的信号的类型。 以下列表介绍了定义的各种标准 CGMS a。 目前，重新分发控件定义只在 CEA 805 一个标准。

<span id="CEA-805-A"></span><span id="cea-805-a"></span>CEA-805-A  
视频的组件接口上的数据

定义如何 CGMS A 和重新分发控件信息应编码中模拟 480p、 720p 或 1080i 信号传输从组件视频输出 （Y/Pb/Pr 输出）。

CEA 发布此标准。 CEA 有关详细信息，请参阅[消费电子设备关联](https://go.microsoft.com/fwlink/p/?linkid=71276)网站。

<span id="CEA-608-B_and_EIA-608-B"></span><span id="cea-608-b_and_eia-608-b"></span><span id="CEA-608-B_AND_EIA-608-B"></span>CEA-608-B 和 EIA-608-B  
行 21 数据服务

定义如何 CGMS-一个应从复合，RF 传输 480i 信号中编码的信息或 S-视频输出。

CEA 和电子组件行业关联 (ECIA) 被发布此标准。 有关 ECIA 详细信息，请参阅[电子组件行业关联](https://go.microsoft.com/fwlink/p/?linkid=71278)网站。

<span id="EN_300_294_V1.3.2__1998-04_"></span><span id="en_300_294_v1.3.2__1998-04_"></span>EN 300 294 V1.3.2 (1998-04)  
电视系统;625 行电视-宽屏幕信号 (WSS)

定义如何 CGMS A 在用内存 (SECAM) 信号应 576i 阶段替换行 (PAL) 或顺序颜色中进行编码。

此标准被发布的欧洲电信标准协会 (ETSI)。 有关此标准的详细信息，请参阅[ETSI](https://go.microsoft.com/fwlink/p/?linkid=26364)网站。

<span id="IEC_-_61880_-_First_edition_-_Video_systems__525_60_"></span><span id="iec_-_61880_-_first_edition_-_video_systems__525_60_"></span><span id="IEC_-_61880_-_FIRST_EDITION_-_VIDEO_SYSTEMS__525_60_"></span>IEC-61880-第一版-视频系统 (525/60)  
视频和旁边都有使用消隐-模拟接口的数据

从模拟或数字的视频输出传输 480i 视频信号中的 CGMS 的信息进行编码的方法。

此方法发布的 IEC。 IEC 有关详细信息，请参阅[IEC](https://go.microsoft.com/fwlink/p/?linkid=8732)网站。

<span id="IEC_-_61880-2_-_First_edition_-_Video_systems__525_60__"></span><span id="iec_-_61880-2_-_first_edition_-_video_systems__525_60__"></span><span id="IEC_-_61880-2_-_FIRST_EDITION_-_VIDEO_SYSTEMS__525_60__"></span>IEC-61880-第 2-第一版-视频系统 (525/60)   
视频和旁边都有数据，使用消隐-模拟接口-第 2 部分：525 渐进式扫描系统

从模拟或数字的视频输出传输 480p 视频信号中的 CGMS 的信息进行编码的方法。

<span id="IEC_-_62375_-_Video_systems__625_50_progressive_"></span><span id="iec_-_62375_-_video_systems__625_50_progressive_"></span><span id="IEC_-_62375_-_VIDEO_SYSTEMS__625_50_PROGRESSIVE_"></span>IEC-62375-视频系统 （625/50 渐进式）  
视频和旁边都有使用消隐-模拟接口的数据

从模拟或数字的视频输出传输 576 p 视频信号中的 CGMS 的信息进行编码的方法。

<span id="ARIB_TR-B15"></span><span id="arib_tr-b15"></span>ARIB TR B15  
有关数字卫星广播的操作指南

定义如何 CGMS 一个信息应以模拟 480i、 480p、 720p 或 1080i 信号的视频输出从传输编码。

此标准仅适用于日本。

ARIB 发布此标准。 有关 ARIB 详细信息，请参阅[ARIB 英语](https://go.microsoft.com/fwlink/p/?linkid=71283)网站。

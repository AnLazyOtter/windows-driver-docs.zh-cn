# <a name="alpha-blend-combination-set"></a>Alpha 混合组合集


## <span id="ddk_alpha_blend_combination_set_gg"></span><span id="DDK_ALPHA_BLEND_COMBINATION_SET_GG"></span>


适用于 alpha blend 组合的最小互操作性配置集包括[**DXVA\_ConfigAlphaCombine**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dxva/ns-dxva-_dxva_configalphacombine)结构的**bConfigBlendType**成员，该成员根据[受限制的配置文件选择值](restricted-profiles.md)正在实现。

# <a name="gdi"></a>GDI


## <span id="ddk_gdi_gg"></span><span id="DDK_GDI_GG"></span>


在本部分中的主题描述图形设备接口 (GDI) 以及它与打印机驱动程序的关系，并显示适配器驱动程序。

显示适配器驱动程序在 Windows Vista 上运行，可以遵循两个模型之一： Windows Vista 显示器驱动程序模型或 Windows 2000 显示器驱动程序模型。 在本部分中的主题适用于打印机驱动程序和驱动程序在 Windows 2000 显示器驱动程序模型，但它们不适用于 Windows Vista 显示器驱动程序模型中的驱动程序。

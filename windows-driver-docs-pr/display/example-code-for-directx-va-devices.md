# <a name="example-code-for-directx-va-devices"></a>适用于 DirectX VA 设备的示例代码


## <span id="ddk_example_code_for_directx_va_devices_gg"></span><span id="DDK_EXAMPLE_CODE_FOR_DIRECTX_VA_DEVICES_GG"></span>


本部分中提供的代码示例演示的实现[运动补偿](motion-compensation-callbacks.md)代码模板和认证的输出保护协议 (COPP) 微型端口驱动程序模板。 运动补偿代码模板用于访问 ProcAmp 控件、 去隔行和 COPP 功能。 COPP 微型端口驱动程序模板用于访问 COPP 功能。 使用这些模板可以简化您的显示器驱动程序和视频的微型端口驱动程序开发。 但是，不需要的驱动程序才能正常工作这种方式实现对 ProcAmp 控件、 去隔行和 COPP 功能的访问。

本部分包括：

[运动补偿代码模板](motion-compensation-code-template.md)

[COPP 视频微型端口驱动程序模板](copp-video-miniport-driver-template.md)

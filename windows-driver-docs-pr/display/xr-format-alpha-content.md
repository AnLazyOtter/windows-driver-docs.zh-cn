# <a name="xr-format-alpha-content"></a>XR 格式 Alpha 内容


本部分仅适用于 Windows 7 和更高版本的操作系统。

可以从表中的格式名称在确定[扩展格式的详细信息](details-of-the-extended-format.md)部分中，alpha 通道解释为处于范围内的 UNORM 值\[0，1\]并且应将其视为与现有格式 DXGI 的 alpha 通道相同\_格式\_R10G10B10A2\_UNORM。

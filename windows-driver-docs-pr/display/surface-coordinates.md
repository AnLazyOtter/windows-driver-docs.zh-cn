# <a name="surface-coordinates"></a>图面上坐标


## <span id="ddk_surface_coordinates_gg"></span><span id="DDK_SURFACE_COORDINATES_GG"></span>


设备面是 2²⁸ 2²⁸ 像素数组的子集。 这些像素都由对 28 位有符号数字，设备表面的左上方像素坐标 (0，0)。 设备表面存在于此坐标空间，这两个坐标的非负的右下象限中。

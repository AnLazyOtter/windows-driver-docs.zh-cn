# <a name="windows-8-in-box-graphics-drivers-treated-as-generic-drivers"></a>Windows 8 中框图形驱动程序视为通用驱动程序


在此方案中，Windows 8 中框图形驱动程序，包括 MS 基本显示驱动程序 (MSBDD)，全都被视为等通用驱动程序通过 Windows 和 Windows Update。

这意味着，作为提供任何更高版本匹配的图形的驱动程序包在 Windows 更新*重要说明*更新。

如果要使用 Windows 8 Windows 更新的默认设置，重要的驱动程序更新下载并安装而无需用户干预，并经常注意到的用户的情况下，此更新出现。

Windows 8 附带，现成图形驱动程序时，显著早于 Windows 更新上的最新驱动程序更新。 此行为可确保用户体验 Windows 8 使用最新/最好图形驱动程序可用。

**注意**  
Windows 8 认证的 OEM 销售与预安装的 Windows 8 的新计算机上所提供的图形驱动程序不被视为泛型。 Windows Update 上的较新，则匹配图形驱动程序将作为可选提供在这些情况下更新。 用户必须主动选择安装的可选驱动程序更新。

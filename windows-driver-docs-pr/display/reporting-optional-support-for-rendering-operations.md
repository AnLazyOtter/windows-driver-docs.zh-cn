# <a name="reporting-optional-support-for-rendering-operations"></a>适用于呈现操作的报告的可选支持


## <span id="ddk_introduction_to_command_and_dma_buffers_gg"></span><span id="DDK_INTRODUCTION_TO_COMMAND_AND_DMA_BUFFERS_GG"></span>


从 Windows 7 开始，显示微型端口驱动程序可以设置其他成员[ **DXGK\_PRESENTATIONCAPS** ](https://msdn.microsoft.com/library/windows/hardware/ff562004)结构，以指示驱动程序可以某些呈现操作或不能支持。

有关可用的呈现功能设置的详细信息，请参阅[ **DXGK\_PRESENTATIONCAPS**](https://msdn.microsoft.com/library/windows/hardware/ff562004)。

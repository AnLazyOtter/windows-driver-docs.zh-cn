# <a name="pixel-formats-for-uncompressed-video"></a>未压缩视频的像素格式


## <span id="ddk_pixel_formats_for_uncompressed_video_gg"></span><span id="DDK_PIXEL_FORMATS_FOR_UNCOMPRESSED_VIDEO_GG"></span>


为了使应用程序以使用未压缩的已解码的图片，图片必须生成已知视频的像素格式。 支持的任何 DirectX VA 快捷键的未压缩的图片格式的列表必须包含至少一种像素格式中所述[4:2:2 视频像素格式](4-2-2-video-pixel-formats.md)或[4:2:0 视频像素格式](4-2-0-video-pixel-formats.md)。

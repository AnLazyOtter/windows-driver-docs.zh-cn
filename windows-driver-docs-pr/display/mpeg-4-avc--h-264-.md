# <a name="mpeg-4-avc-h264"></a>MPEG-4 AVC (H.264)


Mpeg-4 高级视频编码 (AVC)，也称为 ITU-T H.264 视频，是一种标准可提供很好的视频质量比以往的标准 （例如，一半或更低的比特率 mpeg-2、 H.263 或 MPEG 4） 的显著降低比特率的视频压缩。 Mpeg-4 AVC 提供此视频质量而无需增加了设计复杂性如此，这可以使 MPEG 4 实现不切实际或成本过高。 Mpeg-4 AVC 可以应用于各种各样的各种网络和系统，包括最低和最高的比特率、 低和高分辨率视频、 广播、 DVD 存储、 RTP 和 IP 数据包网络和 ITU-T 多媒体电话服务系统上的应用程序。 DirectX VA 如何对 mpeg-4 AVC 进行解码的信息，请下载[DirectX 视频加速规范 H.264/AVC 解码](https://go.microsoft.com/fwlink/p/?linkid=141799)。

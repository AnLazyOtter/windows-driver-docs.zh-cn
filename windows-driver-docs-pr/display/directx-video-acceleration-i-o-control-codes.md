本部分仅适用于 Windows Server 2003 SP1 和更高版本和 Windows XP SP2 和更高版本。

本部分提供示例认证输出保护协议 (COPP) I/O 控制 (IOCTL) 处理的请求的微型端口驱动程序用于引用页面。 您的驱动程序可以重复使用这些 COPP Ioctl，也可以创建相应的新闻。

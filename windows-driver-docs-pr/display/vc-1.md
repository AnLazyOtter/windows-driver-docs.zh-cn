# <a name="vc-1"></a>VC-1


VC 1 是基于离散的余弦值转换 (DCT)，并且还在 H.261、 H.263、 mpeg-1、 MPEG-2 和 MPEG 4 中找到的传统视频编解码器设计演变而来。 VC-1 是一种替代方法[mpeg-4 AVC (H.264)](mpeg-4-avc--h-264-.md)标准的视频编解码器。 Vc-1 包含交错视频序列以及渐进式编码的编码工具。 Vc-1 开发和标准化的主要目标是支持交错内容的压缩，而不事先将内容转换为渐进式。 此支持使 vc-1 广播和视频行业专业人员更具吸引力。 DirectX VA 如何解码 vc-1 的信息，请下载[DirectX 视频加速规范的 Windows Media Video v8、 v9 和 vA 解码 (包括 SMPTE 421m"VC-1")](https://go.microsoft.com/fwlink/p/?linkid=141800)。

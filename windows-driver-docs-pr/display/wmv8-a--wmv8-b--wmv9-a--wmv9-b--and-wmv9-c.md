WMV8\_A，WMV8\_B，WMV9\_A，WMV9\_B 和 WMV9\_C 受限制的配置文件包含一系列的功能支持的 Windows Media 视频所需的版本 8 和 9。 有关这些配置文件的详细信息，请下载[DirectX 视频加速规范的 Windows Media Video v8、 v9 和 vA 解码 (包括 SMPTE 421m"VC-1")](https://go.microsoft.com/fwlink/p/?linkid=141800)。

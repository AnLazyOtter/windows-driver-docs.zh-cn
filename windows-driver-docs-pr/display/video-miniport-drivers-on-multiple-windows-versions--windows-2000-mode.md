# <a name="video-miniport-drivers-on-multiple-windows-versions-windows-2000-model"></a>在多个 Windows 版本 （Windows 2000 模式） 的视频微型端口驱动程序

如果您计划修改为更高版本的基于 NT 的操作系统版本上运行的 Windows NT 4.0 编写的微型端口驱动程序请参阅[将 Windows NT 4.0 微型端口驱动程序转换为 Windows 2000](converting-a-windows-nt-4-0-miniport-driver-to-windows-2000.md)。 您可以开发微型端口驱动程序在当前版本的 Windows，但使用它在早期基于 NT 的 Windows 版本上运行。 有关详细信息，请参阅[使用 VideoPortGetProcAddress](using-videoportgetprocaddress.md)。

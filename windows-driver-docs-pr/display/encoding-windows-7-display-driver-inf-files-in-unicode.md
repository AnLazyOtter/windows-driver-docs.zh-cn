# <a name="encoding-windows-7-display-driver-inf-files-in-unicode"></a>Windows 7 的显示器驱动程序 INF 文件以 unicode 格式进行编码


所有内置 INF 文件必须都编码为 Unicode;它们不能 ANSI。

若要编码为 Unicode （或以验证是否以 unicode 格式编码的 INF 文件） 的 INF 文件，请执行以下步骤：

1.  使用记事本打开 INF 文件。

2.  在文件菜单上单击另存为。

3.  如果"ANSI"出现在对话框中的编码字段中，更改编码为"Unicode"并保存新名称下的文件。

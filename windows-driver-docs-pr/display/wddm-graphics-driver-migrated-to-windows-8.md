# <a name="wddm-graphics-driver-migrated-to-windows-8"></a>WDDM 图形驱动程序迁移到 Windows 8


如果有 Windows 8 升级安装中的图形硬件没有 Window 8 现成覆盖率，WDDM 1.1 或 WDDM 1.0 图形驱动程序使用的以前版本的 Windows 将迁移到 Windows 8 中。

Windows 8 安装程序可能会阻止某些问题的驱动程序。 此类驱动程序不会迁移到 Windows 8。 驱动程序以使用 Windows 遥测报告的问题为基础的此类块进行标识。 在这种情况下，Windows 8 使用 MSBDD 之前通过 Windows 更新或由 OEM/IHV 支持站点的用户安装较新的驱动程序。

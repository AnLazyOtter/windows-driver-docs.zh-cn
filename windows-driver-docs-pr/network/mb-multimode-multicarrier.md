# <a name="mb-multimode-multicarrier"></a>MB 多模多载波


本主题介绍 MB 多模式多运营商，为 Windows 8 集成到移动宽带的支持。 Windows 8 引入了允许的运营商解锁移动宽带设备用户切换 （大多数移动宽带用户在美国以外），您可以选择并连接到任何支持的承运人从在 Windows 用户界面内该支持承运人的本机支持。

这些功能面向 Ihv 以使他们能够实现对此方案的支持。 它们来补充使用 Cid 和支持 MB 多模式运营商情形所需设备行为的 USB NCM (MBIM) 规范 （当前版本 1.0）。

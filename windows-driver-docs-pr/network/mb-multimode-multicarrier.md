# <a name="mb-multimode-multicarrier"></a>MB 多模多载波


本主题介绍对已集成到 Windows 8 移动宽带的 MB 多模式多的支持。 Windows 8 引入了本机支持，允许运营商解锁的移动宽带设备（支持电信公司切换功能，大多数移动宽带用户）的用户选择并连接到 Windows 用户界面中的任何受支持的运营商。

这些功能面向 Ihv，使其能够实现对此方案的支持。 它们对使用支持 MB 多模式多方案所需的 Cid 和设备行为的 USB NCM （MBIM）规范（当前版本1.0）进行了补充。

有关详细信息，请参阅以下主题：

-   [MB 提供程序操作](mb-provider-operations.md)

-   [OID\_WWAN\_首选\_多\_提供程序](https://docs.microsoft.com/windows-hardware/drivers/network/oid-wwan-preferred-multicarrier-providers)

-   [ **\_WWAN\_首选\_多\_提供程序的 NDIS\_状态**](https://docs.microsoft.com/windows-hardware/drivers/network/ndis-status-wwan-preferred-multicarrier-providers)

-   [**NDIS\_WWAN\_首选\_多\_提供程序**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndiswwan/ns-ndiswwan-_ndis_wwan_preferred_multicarrier_providers)

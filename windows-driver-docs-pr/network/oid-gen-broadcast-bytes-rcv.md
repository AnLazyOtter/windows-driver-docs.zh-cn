# <a name="oidgenbroadcastbytesrcv"></a>OID\_GEN\_BROADCAST\_BYTES\_RCV


为查询，OID\_GEN\_广播\_字节\_RCV OID 收到未出现错误的广播数据包中指定的字节数。

**版本信息**

<a href="" id="windows-vista-and-later-versions-of-windows"></a>Windows Vista 和更高版本的 Windows  
已过时。

<a href="" id="ndis-6-0-and-later-drivers"></a>NDIS 6.0 和更高版本的驱动程序  
未请求。 使用[OID\_代\_统计信息](oid-gen-statistics.md)相反。

可选。

支持。

可选。

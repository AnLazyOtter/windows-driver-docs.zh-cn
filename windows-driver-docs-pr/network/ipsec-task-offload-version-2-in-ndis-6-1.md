# <a name="ipsec-task-offload-version-2-in-ndis-61"></a>在 NDIS 6.1 的 IPsec 任务卸载版本 2

\[IPsec 任务卸载功能已弃用，不应使用。\]




IPsec 任务卸载提供卸载服务 IPsec 的网络数据处理到 IPsec 卸载功能的网络接口卡 (Nic)。 NDIS 6.1 和更高版本支持 IPsec 卸载版本 2 (IPsecOV2)。 IPsecOV2 扩展了支持其他加密算法、 IPv6，以及大量发送卸载 (LSO) 与共存。

IPsecOV2 使用 NDIS 6.1 直接 OID 请求接口。 有关直接 OID 请求接口的详细信息，请参阅[NDIS 6.1 中直接 OID 请求接口](direct-oid-request-interface-in-ndis-6-1.md)。

有关 IPsecOV2 详细信息，请参阅[IPsec 卸载版本 2](ipsec-offload-version-2.md)。

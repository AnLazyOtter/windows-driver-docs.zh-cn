# <a name="wditlvcipherkeyihvkey"></a>WDI\_TLV\_CIPHER\_KEY\_IHV\_KEY


WDI\_TLV\_密码\_密钥\_IHV\_键是包含 IHV 密钥 TLV。

## <a name="tlv-type"></a>TLV 类型


0x118

## <a name="length"></a>长度


UINT8 元素的数组大小 （以字节为单位）。 该数组必须包含一个或多个元素。

## <a name="values"></a>值


| 在任务栏的搜索框中键入      | 描述                                            |
|-----------|--------------------------------------------------------|
| UINT8\[\] | 指定 IHV 键 UINT8 元素的数组。 |

# <a name="tearing-down-a-call"></a>放弃呼叫





面向连接的客户端可以发起传出呼叫所做的或已接受的传入呼叫的拆解。 同样，远程方都可以启动传出呼叫所做的或已接受的传入呼叫的拆解。

本部分包括：

[关闭调用客户端启动请求](client-initiated-request-to-close-a-call.md)

[关闭调用传入请求](incoming-request-to-close-a-call.md)

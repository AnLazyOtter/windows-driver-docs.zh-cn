# <a name="receiving-data-in-an-intermediate-driver"></a>在中间驱动程序中接收数据





中间驱动程序收到其协议下边缘上的数据，方法与协议驱动程序大致相同。

有关管理接收网络中间驱动程序中的数据，请参阅[中间驱动程序网络数据管理](intermediate-driver-network-data-management.md)。 有关在 NDIS 协议驱动程序中接收数据的详细信息，请参阅[协议驱动程序发送和接收操作](protocol-driver-send-and-receive-operations.md)。

# <a name="kernel-mode-sdk-topics-for-network-drivers"></a>内核模式网络驱动程序的 SDK 主题

此部分列出了标头文件和内核模式 Windows 网络驱动程序的主题。 在本部分中的标头文件则也会使用用户模式应用程序的网络共享所包含在 Windows 软件开发工具包 (SDK) 而不是 Windows Driver Kit (WDK)。

> [!IMPORTANT]
> 本部分的标头主题包含用于定义、 Oid、 状态指示和不属于网络驱动程序参考其他数据结构的页。 参考主题包括结构、 枚举、 函数和回调。 
>
> 有关适用于这些标头的网络驱动程序参考的详细信息，请参阅[网络 SDK 标头文件中的驱动程序参考](https://msdn.microsoft.com/library/windows/hardware/mt808525)。

本部分包含：

* [Mstcpip.h](mstcpip-h.md)
* [Ntddndis.h](ntddndis-h.md)
* [Ws2def.h](ws2def-h.md)

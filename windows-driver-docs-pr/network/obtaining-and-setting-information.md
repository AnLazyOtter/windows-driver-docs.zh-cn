# <a name="obtaining-and-setting-information"></a>获取和设置的信息





面向连接的客户端、 呼叫管理器或 MCM 驱动程序可使用查询或设置由一个绑定上的其他驱动程序维护的信息。 此外，微型端口驱动程序或 MCM 驱动程序可以指示微型端口驱动程序状态设置为绑定上的其他驱动程序。

本部分包括：

[查询或设置信息](querying-or-setting-information.md)

[指示微型端口驱动程序状态](indicating-miniport-driver-status.md)

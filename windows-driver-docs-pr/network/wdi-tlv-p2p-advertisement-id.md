# <a name="wditlvp2padvertisementid"></a>WDI\_TLV\_P2P\_ADVERTISEMENT\_ID


WDI\_TLV\_P2P\_播发\_ID 是 TLV 包含唯一标识服务实例的 ID。

## <a name="tlv-type"></a>TLV 类型


0xEA

## <a name="length"></a>长度


UINT32 大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入   | 描述                                        |
|--------|----------------------------------------------------|
| UINT32 | 唯一地标识服务实例的 ID。 |

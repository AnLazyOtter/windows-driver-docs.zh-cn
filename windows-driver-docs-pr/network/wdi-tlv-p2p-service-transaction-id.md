# <a name="wditlvp2pservicetransactionid"></a>WDI\_TLV\_P2P\_SERVICE\_TRANSACTION\_ID


WDI\_TLV\_P2P\_服务\_事务\_ID 是包含要用于 ANQP 查询请求的服务事务 ID TLV。

## <a name="tlv-type"></a>TLV 类型


0x116

## <a name="length"></a>长度


超出 UINT8 的大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入  | 描述                                                       |
|-------|-------------------------------------------------------------------|
| UINT8 | 要用于 ANQP 查询请求的服务事务 ID。 |

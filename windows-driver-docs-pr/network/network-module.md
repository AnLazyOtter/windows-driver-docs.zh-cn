# <a name="network-module"></a>网络模块


一个*网络模块*是一个软件模块，在网络堆栈，如数据中实现特定的函数链接接口、 传输协议或网络应用程序。 可以是网络模块[提供程序模块](provider-module.md)即[客户端模块](client-module.md)，或两个，具体取决于网络堆栈中位于何处。

# <a name="wdi-tlv-generatorparser"></a>WDI TLV 生成器/分析器


TLV 生成器，并且分析器共享的库允许 IHV 驱动程序以正确地分析 TLVs 为强类型化的 C/c + + 结构，或与之相反的结构中生成的 TLV 字节 blob。 它还处理版本控制语义，因此 IHV 不需要为。

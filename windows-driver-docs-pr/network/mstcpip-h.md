# <a name="mstcpiph"></a>Mstcpip.h

本部分包含 Mstcpip.h 标头的内核模式网络驱动程序主题。 此标头包括在 Windows SDK 中，因为它也与用户模式应用程序的网络共享。

Mstcpip.h 标头包含 Microsoft 专用扩展核心 Winsock 定义的定义。

# <a name="wditlvcommunicationcapabilities"></a>WDI\_TLV\_通信\_功能


WDI\_TLV\_通信\_功能是 TLV，指定通信功能。

## <a name="tlv-type"></a>TLV 类型


0xE

## <a name="length"></a>长度


UINT32 大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入   | 描述                        |
|--------|------------------------------------|
| UINT32 | 命令最大大小 （字节）。 |

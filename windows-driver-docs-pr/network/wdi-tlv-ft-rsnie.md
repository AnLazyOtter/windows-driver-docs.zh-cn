# <a name="wditlvftrsnie"></a>WDI\_TLV\_FT\_RSNIE


WDI\_TLV\_FT\_RSNIE 是 TLV 包含快速转换 RSN IE 字节 blob。

## <a name="tlv-type"></a>TLV 类型


0x10C

## <a name="length"></a>长度


UINT8 元素的数组大小 （以字节为单位）。 该数组必须包含一个或多个元素。

## <a name="values"></a>值


| 在任务栏的搜索框中键入      | 描述                                                                    |
|-----------|--------------------------------------------------------------------------------|
| UINT8\[\] | UINT8 元素数组，其中包含快速转换 RSN IE 字节 blob。 |

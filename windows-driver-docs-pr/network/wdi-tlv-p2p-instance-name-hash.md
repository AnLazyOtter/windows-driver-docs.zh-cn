# <a name="wditlvp2pinstancenamehash"></a>WDI\_TLV\_P2P\_INSTANCE\_NAME\_HASH


WDI\_TLV\_P2P\_实例\_名称\_哈希是包含的哈希 TLV"实例名称，服务类型"。

**请注意**  此 TLV 添加 Windows 10，版本 1607，WDI 版本 1.0.21 中。

 

## <a name="tlv-type"></a>TLV 类型


0x12C

## <a name="length"></a>长度


大小 （以字节为单位） [ **WDI\_P2P\_服务\_名称\_哈希**](https://msdn.microsoft.com/library/windows/hardware/dn926103)结构。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                                                    | 描述                                |
|-------------------------------------------------------------------------|--------------------------------------------|
| [**WDI\_P2P\_SERVICE\_NAME\_HASH**](https://msdn.microsoft.com/library/windows/hardware/dn926103) | "实例名称，服务类型"的哈希值。 |

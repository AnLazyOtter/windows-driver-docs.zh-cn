# <a name="wditlvindicationcansustainap"></a>WDI\_TLV\_指示\_可以\_SUSTAIN\_亚太


WDI\_TLV\_指示\_可以\_SUSTAIN\_AP 是 TLV 包含为可承受亚太可能的原因。

## <a name="tlv-type"></a>TLV 类型


0xE7

## <a name="length"></a>长度


UINT32 大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入   | 描述                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------|
| UINT32 | 可以承受 AP 原因。 请参阅[ **WDI\_可以\_SUSTAIN\_AP\_原因**](https://msdn.microsoft.com/library/windows/hardware/dn897797)可能的原因值。 |

# <a name="ndis-filter-drivers"></a>NDIS 筛选器驱动程序





NDIS 筛选器驱动程序支持 NDIS 6.0 及更高版本。 以下各节介绍筛选器驱动程序，并描述如何编写和安装 NDIS 筛选器驱动程序：

-   [NDIS 筛选器驱动程序开发的路线图](roadmap-for-developing-ndis-filter-drivers.md)
-   [NDIS 筛选器驱动程序简介](introduction-to-ndis-filter-drivers.md)
-   [编写 NDIS 筛选器驱动程序](writing-ndis-filter-drivers.md)
-   [NDIS 筛选器驱动程序安装](ndis-filter-driver-installation.md)

**请注意**  应而不是 NDIS 筛选器中间驱动程序使用 NDIS 筛选器驱动程序。 NDIS 筛选器中间驱动程序已过时。

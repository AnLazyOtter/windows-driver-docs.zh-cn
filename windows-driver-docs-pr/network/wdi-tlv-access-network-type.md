# <a name="wditlvaccessnetworktype"></a>WDI\_TLV\_ACCESS\_NETWORK\_TYPE


WDI\_TLV\_访问权限\_网络\_类型是 TLV，其中包含访问网络类型。

## <a name="tlv-type"></a>TLV 类型


0x100

## <a name="length"></a>长度


超出 UINT8 的大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入  | 描述                                                                              |
|-------|------------------------------------------------------------------------------------------|
| UINT8 | 若要在探测请求中使用的连接到的网络访问网络类型。 |

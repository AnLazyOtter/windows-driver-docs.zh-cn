# <a name="overview-of-ndis-network-interfaces"></a>NDIS 网络接口概述





NDIS 网络接口提供对所有 Microsoft Windows 支持的各种网络接口的一致表示形式。 不使用 NDIS 网络接口服务，所有网络接口不都是计算机管理员，可以看到和都可见的接口不一定支持管理信息基础 (MIB)。 此外，NDIS 网络接口服务启用对管理员可见的接口之间的分层关系。

本部分包括：

[NDIS 网络接口服务](ndis-network-interface-services.md)

[NDIS 网络接口体系结构](ndis-network-interface-architecture.md)

[NDIS 接口提供程序操作](ndis-interface-provider-operations.md)

[NDIS 接口类型](ndis-interface-types.md)

> [!NOTE]
> NDIS 6.0 网络接口部分引用许多征求意见 (文档 Rfc) 从 Internet 工程任务组 (IETF)。 若要查看 IETF RFC，请访问[IETF 请求注释](https://go.microsoft.com/fwlink/p/?linkid=45661)网站，然后搜索在 RFC **RFC 编号**下框**IETF 存储库检索**。

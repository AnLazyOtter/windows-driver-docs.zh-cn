有关移植 NDIS 信息 5.x 或更早 NDIS 驱动程序 6.x，请参阅[到 NDIS 6.0 移植 NDIS 5.x 驱动程序](https://docs.microsoft.com/previous-versions/windows/hardware/network/porting-ndis-5-x-drivers-to-ndis-6-0)。

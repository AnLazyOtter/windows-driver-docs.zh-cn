# <a name="wditlvp2pservicestatus"></a>WDI\_TLV\_P2P\_服务\_状态


WDI\_TLV\_P2P\_服务\_状态是包含服务的服务状态 TLV。

## <a name="tlv-type"></a>TLV 类型


0xFB

## <a name="length"></a>长度


超出 UINT8 的大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入  | 描述                      |
|-------|----------------------------------|
| UINT8 | 服务的服务状态。 |

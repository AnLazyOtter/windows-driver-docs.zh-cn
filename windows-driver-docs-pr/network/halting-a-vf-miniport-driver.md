# <a name="halting-a-vf-miniport-driver"></a>停止 VF 微型端口驱动程序


VF 微型端口驱动程序遵循相同的步骤为任何 NDIS 微型端口驱动程序时其[ *MiniportHaltEx* ](https://msdn.microsoft.com/library/windows/hardware/ff559388)调用函数。 有关这些步骤的详细信息，请参阅[停止微型端口适配器](halting-a-miniport-adapter.md)。

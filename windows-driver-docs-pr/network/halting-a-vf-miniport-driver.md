# <a name="halting-a-vf-miniport-driver"></a>停止 VF 微型端口驱动程序


调用[*MiniportHaltEx*](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndis/nc-ndis-miniport_halt)函数时，VF 微型端口驱动程序与任何 NDIS 微型端口驱动程序遵循相同的步骤。 有关这些步骤的详细信息，请参阅[停止微型端口适配器](halting-a-miniport-adapter.md)。

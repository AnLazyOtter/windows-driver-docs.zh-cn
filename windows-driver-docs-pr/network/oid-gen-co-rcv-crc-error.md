# <a name="oidgencorcvcrcerror"></a>OID_GEN_CO_RCV_CRC_ERROR

OID_GEN_CO_RCV_CRC_ERROR OID 指定编号的 Pdu 接收与循环冗余检查 (CRC) 错误。

## <a name="requirements"></a>要求

| | |
| --- | --- |
| 版本 | Windows Vista 及更高版本 |
| Header | Ntddndis.h （包括 Ndis.h） |

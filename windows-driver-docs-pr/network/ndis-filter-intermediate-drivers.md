# <a name="ndis-filter-intermediate-drivers"></a>NDIS 筛选器中间驱动程序





**请注意**  NDIS 6.0 及更高版本不支持筛选器中间驱动程序。 应改为使用 NDIS 筛选器驱动程序接口。 NDIS 筛选器驱动程序有关的详细信息，请参阅[NDIS 筛选器驱动程序](ndis-filter-drivers.md)。

 

在 NDIS 5。*x*，则*NDIS 筛选器中间驱动程序*公开的每个绑定到中间驱动程序的较低 （协议） 边缘的基础的微型端口驱动程序的一个虚拟微型端口。

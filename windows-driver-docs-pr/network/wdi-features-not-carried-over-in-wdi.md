# <a name="features-not-carried-over-in-wdi"></a>不能在 WDI 中的功能


以下功能可在先前的本机 WLAN 驱动程序模型，但在 WDI 中不可用。

-   软亚太
-   IBSS
-   网络监视器

以下软 AP 所需功能移植到 Wi-Fi Direct。

-   添加了 802.11 b。
-   移动到 Wi-Fi Direct 软亚太的所有 tethering 要求。

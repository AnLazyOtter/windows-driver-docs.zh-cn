# <a name="ndisstatusmediabusy"></a>NDIS\_状态\_媒体\_忙


NDIS\_状态\_媒体\_忙碌状态指示 IRDA 媒体正忙。

<a name="remarks"></a>备注
-------

详细了解 NDIS\_状态\_媒体\_繁忙状态，请参阅[OID\_IRDA\_媒体\_忙](https://msdn.microsoft.com/library/windows/hardware/ff560284)。

# <a name="sample-ndis-miniport-drivers"></a>示例 NDIS 微型端口驱动程序





[网络驱动程序示例](https://go.microsoft.com/fwlink/p/?LinkId=616034)中[Windows 驱动程序示例](https://go.microsoft.com/fwlink/p/?LinkId=616507 )GitHub 上的存储库包括用于管理多个类型的网卡的微型端口驱动程序的示例代码。 您可以根据需要修改这些示例驱动程序。 示例驱动程序包含可适用于新但类似的驱动程序的函数。 始终是必须编写依赖于硬件的函数。 但是，许多函数是相当标准。 例如，通常标准是使用 NDIS 库而不是网络接口卡进行通信的函数。 对于这些驱动程序函数，示例驱动程序中的代码可能会使用很少或没有修改可用。

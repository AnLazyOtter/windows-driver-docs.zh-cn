# <a name="oidtunnelinterfacesetoid"></a>OID_TUNNEL_INTERFACE_SET_OID

OID_TUNNEL_INTERFACE_SET_OID 对象标识符 (OID) 保留供系统使用。 不要使用它在您的驱动程序中。

## <a name="requirements"></a>要求

| | |
| --- | --- |
| Version | Windows 7 及更高版本 |
| Header | Ntddndis.h （包括 Ndis.h） |

# <a name="vmq-driver-configuration"></a>VMQ 驱动程序配置





本部分介绍 NDIS 提供，以确定网络适配器的虚拟机队列 (VMQ) 功能的功能。

微型端口驱动程序在网络适配器初始化期间报告到 NDIS 其 VMQ 功能和基础驱动程序。

标准化的 INF 文件条目可以启用或禁用 VMQ 功能时安装的驱动程序。

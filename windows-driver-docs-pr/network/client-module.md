# <a name="client-module"></a>客户端模块


一个*客户端模块*是[网络模块](network-module.md)的支持，并实现的客户端[网络编程接口 (NPI)](network-programming-interface.md)。 客户端模块向作为网络模块注册机构中注册自身*客户端*NPI 它所支持。 客户端模块可以注册其自身的多个 NPI 客户端。 网络模块可以是客户端模块和一个[提供程序模块](provider-module.md)。

# <a name="wditlvbeaconies"></a>WDI\_TLV\_信号\_导致浏览器


WDI\_TLV\_发信号的\_导致浏览器是包含从关联的信号 IEs TLV。

## <a name="tlv-type"></a>TLV 类型


0x78

## <a name="length"></a>长度


UINT8 元素的数组大小 （以字节为单位）。 该数组必须包含一个或多个元素。

## <a name="values"></a>值


| 在任务栏的搜索框中键入      | 描述                         |
|-----------|-------------------------------------|
| UINT8\[\] | 从关联信号导致浏览器。 |

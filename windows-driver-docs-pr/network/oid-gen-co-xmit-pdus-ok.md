# <a name="oidgencoxmitpdusok"></a>OID_GEN_CO_XMIT_PDUS_OK

OID_GEN_CO_XMIT_PDUS_OK OID 指定 Pdu 错误没有传输的数。

## <a name="requirements"></a>要求

| | |
| --- | --- |
| Version | Windows Vista 及更高版本 |
| Header | Ntddndis.h （包括 Ndis.h） |

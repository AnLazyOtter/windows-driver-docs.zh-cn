# <a name="offloading-tcpip-connections"></a>卸载 TCP/IP 连接





NDIS 6.0 及更高版本支持卸载 TCP/IP 连接。

NDIS TCP/IP 连接卸载接口使服务，例如 TCP 烟囱卸载。

有关卸载的 TCP/IP 连接的详细信息，请参阅概述的 TCP 烟囱卸载中[完整 TCP 卸载](full-tcp-offload.md)。

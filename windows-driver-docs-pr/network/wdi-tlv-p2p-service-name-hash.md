# <a name="wditlvp2pservicenamehash"></a>WDI\_TLV\_P2P\_SERVICE\_NAME\_HASH


WDI\_TLV\_P2P\_服务\_名称\_哈希是包含服务名称的哈希 TLV。

## <a name="tlv-type"></a>TLV 类型


0xED

## <a name="length"></a>长度


大小 （以字节为单位） [ **WDI\_P2P\_服务\_名称\_哈希**](https://msdn.microsoft.com/library/windows/hardware/dn926103)结构。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                                                    | 描述                      |
|-------------------------------------------------------------------------|----------------------------------|
| [**WDI\_P2P\_SERVICE\_NAME\_HASH**](https://msdn.microsoft.com/library/windows/hardware/dn926103) | WFDS 服务名称的哈希值。 |

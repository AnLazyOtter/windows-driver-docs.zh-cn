# <a name="oidgencovendordriverversion"></a>OID_GEN_CO_VENDOR_DRIVER_VERSION

网卡驱动程序从供应商指派的版本号。

此 OID 是两个字节的长度;返回值的低位部分指定的次版本，而高序位后半部分中指定的主版本。

## <a name="requirements"></a>要求

| | |
| --- | --- |
| 版本 | Windows Vista 及更高版本 |
| Header | Ntddndis.h （包括 Ndis.h） |

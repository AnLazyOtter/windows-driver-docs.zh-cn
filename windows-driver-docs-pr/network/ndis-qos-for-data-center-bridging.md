# <a name="ndis-qos-for-data-center-bridging"></a>数据中心桥接的 NDIS QoS


本部分介绍了 IEEE 802.1 数据中心桥接 (DCB) 接口的 NDIS 服务质量 (QoS)。 微型端口驱动程序使用 NDIS QoS 通信优先级的传输，或*出口*，DCB 兼容的网络适配器上的数据包。

DCB 的 NDIS QoS 引入了 Windows Server 2012 中的 NDIS 6.30。

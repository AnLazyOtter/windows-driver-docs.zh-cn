# <a name="introduction-to-ndis-qos-for-data-center-bridging"></a>用于数据中心桥接的 NDIS QoS 简介


本部分介绍 IEEE 802.1 数据中心桥接 (DCB) 接口的 NDIS 服务质量 (QoS)。 小型端口驱动程序使用 NDIS QoS 实现流量优先级, 即通过 DCB 兼容网络适配器传输或*传出*数据包。

Windows Server 2012 中的 NDIS 6.30 引入了 DCB 的 NDIS QoS。

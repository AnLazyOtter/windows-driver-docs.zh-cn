# <a name="wditlvoperationmode"></a>WDI\_TLV\_操作\_模式


WDI\_TLV\_操作\_模式是 TLV 包含所需的操作模式。

## <a name="tlv-type"></a>TLV 类型


0x95

## <a name="length"></a>长度


UINT32 大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入   | 描述                                                                                        |
|--------|----------------------------------------------------------------------------------------------------|
| UINT32 | 所需的操作模式，如中所定义[ **WDI\_操作\_模式**](https://msdn.microsoft.com/library/windows/hardware/dn926085)。 |

# <a name="wditlvchannelnumber"></a>WDI\_TLV\_通道\_数


WDI\_TLV\_通道\_数是包含频道号 TLV。

## <a name="tlv-type"></a>TLV 类型


0x121

## <a name="length"></a>长度


超出 UINT8 的大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入  | 描述         |
|-------|---------------------|
| UINT8 | 频道号。 |

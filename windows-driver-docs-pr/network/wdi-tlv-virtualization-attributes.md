# <a name="wditlvvirtualizationattributes"></a>WDI\_TLV\_虚拟化\_属性


WDI\_TLV\_虚拟化\_属性是包含虚拟化属性 TLV。

## <a name="tlv-type"></a>TLV 类型


0x24

## <a name="length"></a>长度


所有的大小 （以字节为单位） 总和包含 TLVs。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                                                                  | 允许多个 TLV 实例 | 可选 | 描述                      |
|---------------------------------------------------------------------------------------|--------------------------------|----------|----------------------------------|
| [**WDI\_TLV\_虚拟化\_功能**](wdi-tlv-virtualization-capabilities.md) |                                |          | 虚拟化功能。 |

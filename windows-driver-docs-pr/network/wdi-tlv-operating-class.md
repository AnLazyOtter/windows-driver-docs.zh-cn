# <a name="wditlvoperatingclass"></a>WDI\_TLV\_OPERATING\_类


WDI\_TLV\_OPERATING\_类是包含通道频段 TLV。

## <a name="tlv-type"></a>TLV 类型


0xFA

## <a name="length"></a>长度


超出 UINT8 的大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入  | 描述                       |
|-------|-----------------------------------|
| UINT8 | 表示通道的频率带区。 |

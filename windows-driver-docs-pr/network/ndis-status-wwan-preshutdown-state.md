# <a name="ndisstatuswwanpreshutdownstate"></a>NDIS\_状态\_WWAN\_PRESHUTDOWN\_状态


NDIS\_状态\_WWAN\_PRESHUTDOWN\_状态通知是向主机 MBB 驱动程序中的单向通知。 调制解调器完成关闭之前所需的所有操作时，此通知将发送 MBB 驱动程序。

使用此通知[ **NDIS\_WWAN\_PRESHUTDOWN\_状态**](https://msdn.microsoft.com/library/windows/hardware/mt593234)结构。

# <a name="ndis_status_wwan_preshutdown_state"></a>\_WWAN\_PRESHUTDOWN\_状态的 NDIS\_状态


\_WWAN\_PRESHUTDOWN\_状态通知的 NDIS\_状态是从 MBB 驱动程序到主机的单向通知。 MBB 驱动程序在调制解调器完成所有需要的操作后发送此通知。

此通知使用[**NDIS\_WWAN\_PRESHUTDOWN\_状态**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndiswwan/ns-ndiswwan-_ndis_wwan_preshutdown_state)结构。

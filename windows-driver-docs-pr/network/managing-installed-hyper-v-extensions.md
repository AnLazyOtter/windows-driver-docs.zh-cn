# <a name="managing-installed-hyper-v-extensible-switch-extensions"></a>管理已安装的 Hyper-V 可扩展交换机扩展


从 Windows Server 2012 开始，可以通过 PowerShell cmdlet 管理的 HYPER-V 可扩展交换机扩展在每个可扩展交换机实例。 通过这些 cmdlet，扩展可以启用或禁用，并可以在可扩展交换机驱动程序堆栈中更改同一类中的扩展的顺序。 这样，每个可扩展交换机实例具有一组不同的已启用的扩展和扩展的同一个类的唯一排序。

本部分包括以下主题描述如何使用 PowerShell cmdlet 来管理的 HYPER-V 可扩展交换机扩展：

-   [枚举的 HYPER-V 可扩展交换机实例](enumerating-hyper-v-extensibility-switch-instances.md)
-   [枚举的 HYPER-V 可扩展交换机扩展](enumerating-installed-hyper-v-extensibility-switch-extensions.md)
-   [启用的 HYPER-V 可扩展交换机扩展](enabling-hyper-v-extensibility-switch-extensions.md)
-   [禁用 HYPER-V 可扩展交换机扩展](disabling-hyper-v-extensibility-switch-extensions.md)
-   [重新排序的 HYPER-V 可扩展交换机扩展](reordering-hyper-v-extensibility-switch-extensions.md)

# <a name="wmi-support-for-header-data-split"></a>标头数据拆分的 WMI 支持





NDIS 6.1 和更高版本的支持标头数据拆分为 WMI 应用程序的 Guid 来查询和设置的标头数据拆分配置并提供更改通知。

以下 Guid 支持标头数据拆分：

-   GUID\_NDIS\_HD\_拆分\_当前\_配置

-   GUID\_NDIS\_HD\_拆分\_参数

-   GUID\_NDIS\_状态\_HD\_拆分\_当前\_配置

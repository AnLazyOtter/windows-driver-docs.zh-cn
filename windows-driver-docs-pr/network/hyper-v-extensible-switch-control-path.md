# <a name="hyper-v-extensible-switch-control-path"></a>Hyper-V 可扩展交换机控制路径


本部分讨论的控制路径的 HYPER-V 可扩展交换机对象标识符 (OID) 请求和 NDIS 状态指示在之间移动。

本部分包括以下主题：

[HYPER-V 可扩展交换机的 OID 请求的控制路径](hyper-v-extensible-switch-control-path-for-oid-requests.md)

[HYPER-V 可扩展交换机控制 NDIS 状态指示路径](hyper-v-extensible-switch-control-path-for-ndis-status-indications.md)

**请注意**  可扩展交换机接口，在 NDIS 筛选器驱动程序被称为*可扩展交换机扩展*驱动程序堆栈称为*可扩展交换机驱动程序堆栈*.

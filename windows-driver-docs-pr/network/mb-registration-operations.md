# <a name="mb-registration-operations"></a>MB 注册操作


微型端口驱动程序使用[OID\_WWAN\_注册\_状态](https://msdn.microsoft.com/library/windows/hardware/ff569834)若要处理两者*查询*并*设置*注册操作。 例如，这些操作可能要使用网络提供程序注册、 查询与提供程序的网络设备的注册状态和注册状态更改时发送通知。

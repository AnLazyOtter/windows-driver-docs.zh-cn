# <a name="hyper-v-extensible-switch-extensions"></a>Hyper-V 可扩展交换机扩展


本部分介绍的 NDIS 筛选器驱动程序中的可扩展交换机接口支持的类型。 这些筛选器驱动程序被称为*可扩展交换机扩展*。

本部分包括以下主题：

[捕获扩展](capturing-extensions.md)

[筛选扩展](filtering-extensions.md)

[转发扩展](forwarding-extensions.md)

**请注意**  相同类型的多个扩展绑定到可扩展交换机实例，可以修改在可扩展交换机驱动程序堆栈内的分层的顺序。 有关详细信息，请参阅[重新排序的 HYPER-V 可扩展交换机扩展](reordering-hyper-v-extensibility-switch-extensions.md)。

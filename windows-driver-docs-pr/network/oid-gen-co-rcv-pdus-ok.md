# <a name="oid_gen_co_rcv_pdus_ok"></a>OID_GEN_CO_RCV_PDUS_OK

OID_GEN_CO_RCV_PDUS_OK OID 指定 NIC 接收的且没有错误并指示绑定协议的 Pdu 数。

| | |
| --- | --- |
| Version | Windows Vista 及更高版本 |
| Header | Ntddndis （包括 Ndis .h） |

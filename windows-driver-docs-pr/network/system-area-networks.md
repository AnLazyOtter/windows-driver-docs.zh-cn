# <a name="system-area-networks"></a>系统区域网络





一个*系统区域网络*(SAN) 是一组链接的高速、 高性能的连接的设备。 SAN 连接将使用 Internet 协议 (IP) 地址，分配 TCP/IP 通过给每个 SAN 网络接口控制器 (NIC)，以确定数据的路由。 它还使用内置 SAN 的可靠传输来执行数据传送。 例如，SAN 可以包括客户端和服务器计算机群集。

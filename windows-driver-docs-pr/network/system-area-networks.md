# <a name="system-area-networks-overview"></a>系统区域网络概述

*系统区域网络*(SAN) 是一组通过高速、高性能连接进行链接的设备。 SAN 连接使用 Internet 协议 (IP) 地址, 这些地址通过 TCP/IP 分配给每个 SAN 网络接口控制器 (NIC) 来确定数据路由。 它还使用 SAN 中内置的可靠传输来执行数据传输。 例如, SAN 可以包括客户端和服务器计算机的群集。

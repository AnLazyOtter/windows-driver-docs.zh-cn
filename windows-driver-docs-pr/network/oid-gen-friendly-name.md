# <a name="oidgenfriendlyname"></a>OID\_GEN\_友好\_名称


为查询，OID\_GEN\_友好\_名称 OID 返回微型端口适配器的友好名称。

<a name="remarks"></a>备注
-------

OID\_GEN\_友好\_名称 OID 返回微型端口适配器的友好名称。

友好名称旨在帮助用户快速、 准确地识别微型端口适配器-例如，"PCI 以太网适配器"和"虚拟专用网络适配器"将被视为友好名称。

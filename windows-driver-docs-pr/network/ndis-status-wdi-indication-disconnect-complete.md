# <a name="ndisstatuswdiindicationdisconnectcomplete"></a>NDIS\_状态\_WDI\_指示\_断开连接\_完成


微型端口驱动程序使用 NDIS\_状态\_WDI\_指示\_断开连接\_完成，以指示完成[OID\_WDI\_任务\_断开连接](oid-wdi-task-disconnect.md)。

| Object |
|--------|
| 端口   |

此指示不包含任何其他数据。 标头中的数据就足够了。

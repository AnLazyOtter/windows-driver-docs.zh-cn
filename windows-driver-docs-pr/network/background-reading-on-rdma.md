# <a name="background-reading-on-rdma"></a>背景阅读过有关 RDMA


RDMA 是一种可提供最大程度减少 CPU 使用情况的高吞吐量、 低延迟通信的网络技术。 NDK 目前支持以下 RDMA 技术：

-   Infiniband (IB)
-   Internet 宽区域 RDMA 协议 (iWARP)
-   RDMA over 汇聚 Ethernet (RoCE)

有关 RDMA、 Infiniband、 iWARP 和 RoCE 的详细信息，请参阅以下资源：

-   [RFC 5040:远程直接内存访问协议规范](http://tools.ietf.org/html/rfc5040)
-   [RFC 5041:直接数据放置在可靠的传输通道](http://tools.ietf.org/html/rfc5041)
-   [RFC 5042:直接数据放置协议 (DDP) / 远程直接内存访问协议 (RDMAP) 安全性](http://tools.ietf.org/html/rfc5042)
-   [RFC 5043:Stream 控制传输协议 (SCTP) 直接数据放置 (DDP) 自适应](http://tools.ietf.org/html/rfc5043)
-   [RFC 5044:标记 PDU 对齐 TCP 规范的帧](http://tools.ietf.org/html/rfc5044)
-   [RDMA 联盟](http://www.rdmaconsortium.org/)
-   [Internet 草案：RDMA 协议谓词规范](http://tools.ietf.org/html/draft-hilland-rddp-verbs-00)
-   [Infiniband 贸易协会 （适用于 Infiniband 和 RoCE 可下载规范）](http://www.infinibandta.org/)

# <a name="pldr"></a>PLDR


## <a name="recovery-of-pldr"></a>PLDR 的恢复


意外删除后的驱动程序 （UE 和 LE） 必须在实际应用，以便可以删除并重新枚举 （由总线） 的设备对象释放所有资源。 如果这不会发生，设备将不重新枚举，并因此未恢复。

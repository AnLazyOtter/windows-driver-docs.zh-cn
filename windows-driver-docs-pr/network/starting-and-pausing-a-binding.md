# <a name="starting-and-pausing-a-binding"></a>启动和暂停绑定





NDIS 暂停停止数据流可能会干扰插即用 (PnP) 操作，例如，在添加或删除筛选器模块的驱动程序堆栈，或者，若要添加新的绑定的绑定。 有关如何修改正在运行的驱动程序堆栈的详细信息，请参阅[修改运行驱动程序堆栈](modifying-a-running-driver-stack.md)。

NDIS 启动从绑定*已暂停*状态。 该绑定将进入*已暂停*绑定操作完成后或暂停操作完成后状态。

# <a name="parties"></a>参与方





一个*参与方*表示的可能是多个叶之一*多点到点连接*。 在进行传出调用，面向连接的客户端可以指定参与方。 这样，在调用 multipoint 调用，与客户端充当*根*的调用，并作为远程参与方*叶*。 然后，客户端可以请求其他远程参与方添加为*叶节点*到调用。

有关将参与方添加到多点到点调用的详细信息，请参阅[将参与方添加到 Multipoint 调用](adding-a-party-to-a-multipoint-call.md)。 从点 multipoint 调用删除参与方的信息，请参阅[Multipoint 调用正在删除参与方](dropping-a-party-from-a-multipoint-call.md)。

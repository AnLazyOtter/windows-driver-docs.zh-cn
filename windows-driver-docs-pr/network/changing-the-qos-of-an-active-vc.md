# <a name="changing-the-qos-of-an-active-vc"></a>更改活动 VC 的 QoS





面向连接的客户端可以请求的服务质量 (QoS) 上的活动的虚拟连接 (VC) 传出或传入的调用中的更改。 远程客户端还可以进行此类请求。 在这种情况下，呼叫管理器或 MCM 驱动程序指示远程客户端的传入请求，若要更改 QoS。

本部分包括：

[客户端发出的请求以更改调用参数](client-initiated-request-to-change-call-parameters.md)

[传入请求，更改调用参数](incoming-request-to-change-call-parameters.md)

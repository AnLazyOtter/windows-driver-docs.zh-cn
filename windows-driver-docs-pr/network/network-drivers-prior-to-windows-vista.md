# <a name="network-drivers-prior-to-windows-vista"></a>Windows Vista 以前的网络驱动程序

若要访问 Windows 2000 和 Windows XP 的网络驱动程序的设计指南和参考主题，请参阅[网络驱动程序之前 Windows Vista](https://docs.microsoft.com/previous-versions/windows/hardware/network/ff562407(v=vs.85))。 

在 Windows Vista 之前的网络驱动程序使用 NDIS 5.1，已取代在 Windows Vista 和更高版本的 NDIS 6.X。

# <a name="mb-interface-terms"></a>MB 接口术语


在移动宽带 (MB) 文档中使用了以下术语：

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th align="left">术语</th>
<th align="left">描述</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td align="left"><p>MBIM</p></td>
<td align="left"><p>移动宽带接口模型中，移动宽带设备的 USB 设备使用组 (DWG) 规范。</p></td>
</tr>
<tr class="even">
<td align="left"><p>MBIM 函数</p></td>
<td align="left"><p>符合 MBIM 规范的 USB 设备内的某个 USB 函数。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>移动宽带设备</p></td>
<td align="left"><p>这两种单个函数的 USB 设备或多正常工作。 在单个函数的情况下，该函数应 MBIM 函数。 在多函数事例中，其中一个函数是 MBIM 函数。 这可能是在其中配置一个包含 MBIM 函数的多配置设备。</p></td>
</tr>
<tr class="even">
<td align="left"><p>NCM2</p></td>
<td align="left"><p>MBIM 规范早期名称。 某些关系图仍然作为 NCM2 函数称为 MBIM 功能。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>虚拟 CD-ROM</p></td>
<td align="left"><p>CD-ROM 函数不具有的物理 CD-ROM 驱动器。</p></td>
</tr>
<tr class="even">
<td align="left"><p><a href="https://go.microsoft.com/fwlink/p/?linkid=308934" data-raw-source="[IAD](https://go.microsoft.com/fwlink/p/?linkid=308934)">IAD</a></p></td>
<td align="left"><p>关联描述符 (Iad) 到函数中使用组接口的接口。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>WMC UFD</p></td>
<td align="left"><p>联合功能描述符 (Ufd) 的无线移动通信 (WMC) 规范中所述。 Ufd 用于分组到函数的接口。 这是使用 Iad 的替代方法。</p></td>
</tr>
<tr class="even">
<td align="left"><p>变形</p></td>
<td align="left"><p>若要公开一组不同的 USB 函数比当前公开的内容 USB 设备的能力。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>驱动程序</p></td>
<td align="left"><p>Windows 以使用 USB 函数所需的软件。</p></td>
</tr>
<tr class="even">
<td align="left"><p>收件箱驱动程序</p></td>
<td align="left"><p>由 Microsoft 提供的 USB 函数的驱动程序。 这些驱动程序都存在于 Windows。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>IHV 驱动程序</p></td>
<td align="left"><p>独立硬件供应商 (IHV) 提供的 USB 函数的不具有收件箱驱动程序的驱动程序。</p></td>
</tr>
<tr class="even">
<td align="left"><p>IHV 驱动程序包</p></td>
<td align="left"><p>由 IHV 提供所有 IHV 驱动程序的集合。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>USBHUB</p></td>
<td align="left"><p>Microsoft USB 集线器驱动程序。</p></td>
</tr>
<tr class="even">
<td align="left"><p>USBCCGP</p></td>
<td align="left"><p>用于复合的 USB 设备的 Microsoft 驱动程序。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>MBCD</p></td>
<td align="left"><p>移动宽带类驱动程序，Windows 8 中的收件箱驱动程序符合 MBIM 规范的 USB 函数。</p></td>
</tr>
</tbody>
</table>

# <a name="windows-sockets-direct"></a>Windows Sockets Direct





Microsoft Windows 套接字直接是一种技术，如果可能，请在高速度和以通过系统区域网络 (SAN) 而不是 TCP/IP 传输和 NDIS 微型端口驱动程序的 Windows 套接字应用程序的高性能路由数据。

Windows 套接字直接包括在：

-   Microsoft Windows 2000 Datacenter Server

-   Microsoft Windows 2000 Advanced Server SP2

-   Microsoft Windows 2000 Server 设备工具包 SP2

-   Microsoft Windows Server 2003

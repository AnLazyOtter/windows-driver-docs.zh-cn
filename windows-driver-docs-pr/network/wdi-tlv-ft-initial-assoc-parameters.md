# <a name="wditlvftinitialassocparameters"></a>WDI\_TLV\_FT\_初始\_ASSOC\_参数


WDI\_TLV\_FT\_初始\_ASSOC\_参数是快速转换为包含初始关联参数 TLV。

## <a name="tlv-type"></a>TLV 类型


0x105

## <a name="length"></a>长度


所有包含的元素的大小的总和 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                        | 描述                |
|---------------------------------------------|----------------------------|
| [**WDI\_TLV\_FT\_MDE**](wdi-tlv-ft-mde.md) | BSS 条目的 MDIE。 |

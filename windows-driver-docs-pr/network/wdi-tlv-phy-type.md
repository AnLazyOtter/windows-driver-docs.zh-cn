# <a name="wditlvphytype"></a>WDI\_TLV\_PHY\_类型


WDI\_TLV\_PHY\_类型是包含 PHY 类型 TLV。

## <a name="tlv-type"></a>TLV 类型


0x122

## <a name="length"></a>长度


UINT32 大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                                 | 描述   |
|------------------------------------------------------|---------------|
| [**WDI\_PHY\_TYPE**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wditypes/ne-wditypes-_wdi_phy_type) (UINT32) | PHY 类型中。 |

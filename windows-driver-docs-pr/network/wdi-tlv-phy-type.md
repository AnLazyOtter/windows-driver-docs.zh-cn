# <a name="wdi_tlv_phy_type"></a>WDI\_TLV\_PHY\_类型


WDI\_TLV\_PHY\_类型是包含 PHY 类型的 TLV。

## <a name="tlv-type"></a>TLV 类型


0x122

## <a name="length"></a>长度


UINT32 的大小（以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                                 | 描述   |
|------------------------------------------------------|---------------|
| [**WDI\_PHY\_类型**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wditypes/ne-wditypes-_wdi_phy_type)（UINT32） | PHY 类型。 |

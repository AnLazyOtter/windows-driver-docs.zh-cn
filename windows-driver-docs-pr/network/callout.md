# <a name="callout"></a>标注


一个*标注*提供的功能，扩展了 Windows 筛选平台的功能。 标注的一组组成[标注函数](callout-function.md)和一个用于唯一地标识标注 GUID 键。 有多个[内置标注](https://msdn.microsoft.com/library/windows/hardware/ff543857)附带 Windows 筛选平台。 可以使用添加其他标注[标注驱动程序](callout-driver.md)。

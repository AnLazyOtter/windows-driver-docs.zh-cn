# <a name="learning-about-intermediate-drivers"></a>学习有关中间驱动程序





您可以编写无连接或面向连接的下边缘中间驱动程序。 以下列表描述了 WDK 文档中的哪些部分，应，具体取决于你正在编写的中间驱动程序的类型：

<a href="" id="intermediate-drivers-that-have-a-connectionless-lower-edge"></a>**中间有无连接的下边缘的驱动程序**  
如果你正在编写的下边缘提供无连接的微型端口驱动程序的接口的中间驱动程序，请阅读：

-   [中间的 NDIS 驱动程序](ndis-intermediate-drivers.md)

<a href="" id="intermediate-drivers-that-have-a-connection-oriented-lower-edge"></a>**中间有面向连接的下边缘的驱动程序**  
如果你正在编写的下边缘提供面向连接的微型端口驱动程序的接口的中间驱动程序，请阅读：

-   [中间的 NDIS 驱动程序](ndis-intermediate-drivers.md)

-   [面向连接的 NDIS](connection-oriented-ndis.md)

# <a name="starting-and-pausing-a-filter-module"></a>启动和暂停筛选器模块





NDIS 暂停停止数据流可能会干扰插操作，例如，在添加或删除筛选器模块的驱动程序堆栈，或添加新的绑定的筛选器模块。 有关如何修改正在运行的驱动程序堆栈的详细信息，请参阅[修改运行驱动程序堆栈](modifying-a-running-driver-stack.md)。

NDIS 开始提供的筛选器模块*已暂停*状态。 筛选器模块进入*已暂停*附加操作完成后或暂停操作完成后状态。

[暂停筛选器模块](pausing-a-filter-module.md)

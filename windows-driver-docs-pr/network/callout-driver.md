# <a name="callout-driver"></a>标注驱动程序


一个*标注驱动程序*是一个内核模式驱动程序实现一个或多个[标注](callout.md)。 标注驱动程序注册使用其标注[筛选器引擎](filter-engine.md)，以便可以调用筛选器引擎[标注函数](callout-function.md)标注时在计算机处理连接或多个数据包。

# <a name="ndis-qos-architecture-for-data-center-bridging"></a>数据中心桥接的 NDIS QoS 体系结构


本部分介绍 NDIS 服务质量 (QoS) 体系结构的 IEEE 802.1 数据中心桥接 (DCB) 的一部分的各种组件。 这些组件在下图中显示。

![设备安装组件](images/dcb.png)

在关系图中的无阴影的框表示 Windows 操作系统提供，包括支持 DCB 的组件的组件。 有关这些组件的详细信息，请参阅[系统提供 DCB 组件](system-provided-dcb-components.md)。

阴影框中的关系图表示 DCB 组件的独立硬件供应商 (Ihv) 和原始设备制造商 (Oem) 提供。 有关这些组件的详细信息，请参阅[Vendor-Provided DCB 组件](vendor-provided-dcb-components.md)。

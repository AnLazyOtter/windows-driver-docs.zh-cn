# <a name="other-ndis-features-available-to-condis-wan-drivers"></a>提供的 CoNDIS WAN 驱动程序的其他 NDIS 功能





WAN 的 CoNDIS 微型端口驱动程序或 MCMs 可以充分利用以下功能：

-   插即用 (PnP) 事件通知的微型端口驱动程序

-   64 位的统计计数器

-   取消发送数据包

-   注册[ *MiniportShutdownEx* ](https://msdn.microsoft.com/library/windows/hardware/ff559449)函数

-   新的微型端口驱动程序属性

-   安全函数

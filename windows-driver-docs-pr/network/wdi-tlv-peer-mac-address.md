# <a name="wditlvpeermacaddress"></a>WDI\_TLV\_对等方\_MAC\_地址


WDI\_TLV\_对等方\_MAC\_地址是 TLV 包含对等的 MAC 地址。

## <a name="tlv-type"></a>TLV 类型


0x4C

## <a name="length"></a>长度


大小 （以字节为单位） [ **WDI\_MAC\_地址**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/dot11wdi/ns-dot11wdi-_wdi_mac_address)结构。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                              | 描述                                  |
|---------------------------------------------------|----------------------------------------------|
| [**WDI\_MAC\_ADDRESS**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/dot11wdi/ns-dot11wdi-_wdi_mac_address) | 指定对等方的 Wi-fi MAC 地址。 |

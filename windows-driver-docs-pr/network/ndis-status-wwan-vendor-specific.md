# <a name="ndisstatuswwanvendorspecific"></a>NDIS\_STATUS\_WWAN\_VENDOR\_SPECIFIC


微型端口驱动程序使用 NDIS\_状态\_WWAN\_供应商\_特定的通知实现供应商特定操作或特定于供应商的事务完成响应更改通知。

微型端口驱动程序还可以发送未经请求的事件与该通知。

使用此通知[ **NDIS\_WWAN\_供应商\_特定**](https://msdn.microsoft.com/library/windows/hardware/ff567947)结构。

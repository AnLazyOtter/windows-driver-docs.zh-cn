# <a name="issuing-commands-to-communicate-with-devices"></a>发出命令以便与设备通信





使用 WDM 下边缘，用于控制设备的总线上的微型端口驱动程序与这些设备进行通信的总线接口向发出命令。 有关详细信息，请参阅[处理 Irp](https://docs.microsoft.com/windows-hardware/drivers/kernel/handling-irps)和特定的总线驱动程序接口的文档。

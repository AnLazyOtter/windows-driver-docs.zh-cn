# <a name="oidgencorcvpduserror"></a>OID_GEN_CO_RCV_PDUS_ERROR

OID_GEN_CO_RCV_PDUS_ERROR OID 指定 Pdu NIC 接收，但未指示绑定协议由于错误数。

## <a name="requirements"></a>要求

| | |
| --- | --- |
| Version | Windows Vista 及更高版本 |
| Header | Ntddndis.h （包括 Ndis.h） |

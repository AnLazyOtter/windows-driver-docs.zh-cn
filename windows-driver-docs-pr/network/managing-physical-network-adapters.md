# <a name="managing-physical-network-adapters"></a>管理物理网络适配器


本部分中描述的 HYPER-V 可扩展交换机扩展可以对基础绑定到可扩展交换机外部网络适配器的物理适配器执行的操作。

这些操作允许要转发或源于对基础物理网络适配器的对象标识符 (OID) 请求的扩展。 扩展还可以将转发或源自从可扩展交换机驱动程序堆栈上的物理网络适配器的 NDIS 状态指示。

**请注意**  这类操作只能由转发扩展的可扩展交换机。 有关此类型的扩展的详细信息，请参阅[转发扩展](forwarding-extensions.md)。

 

本部分包括以下主题：

[管理物理网络适配器连接状态](forwarding-packets-to-physical-network-adapters.md)

[将数据包转发到物理网络适配器](forwarding-packets-to-physical-network-adapters.md)

[管理对物理网络适配器的 OID 请求](managing-oid-requests-to-physical-network-adapters.md)

[管理从物理网络适配器的 NDIS 状态指示](managing-ndis-status-indications-from-physical-network-adapters.md)

# <a name="wdi_tlv_configured_mac_address"></a>WDI\_TLV\_配置\_MAC\_地址


WDI\_TLV\_配置的\_MAC\_地址是包含自定义 MAC 地址的 TLV。

## <a name="tlv-type"></a>TLV 类型


0x99

## <a name="length"></a>长度


[**WDI\_MAC\_地址**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dot11wdi/ns-dot11wdi-_wdi_mac_address)结构的大小（以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                              | 描述                                       |
|---------------------------------------------------|---------------------------------------------------|
| [**WDI\_MAC\_地址**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dot11wdi/ns-dot11wdi-_wdi_mac_address) | 端口应使用的 MAC 地址。 |

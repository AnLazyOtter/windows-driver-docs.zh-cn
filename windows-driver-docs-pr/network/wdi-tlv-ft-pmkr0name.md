# <a name="wdi_tlv_ft_pmkr0name"></a>WDI\_TLV\_FT\_PMKR0NAME


WDI\_TLV\_FT\_PMKR0NAME 为 TLV，其中包含 PMKR0Name 或 PMKR1Name （802.11 r）。

## <a name="tlv-type"></a>TLV 类型


0x107

## <a name="length"></a>长度


WDI 的大小（以字节为单位） [ **\_类型\_PMK\_名称**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wditypes/ns-wditypes-_wdi_type_pmk_name)结构。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                                   | 描述                         |
|--------------------------------------------------------|-------------------------------------|
| [**WDI\_类型\_PMK\_名称**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wditypes/ns-wditypes-_wdi_type_pmk_name) | PMKR0Name 或 PMKR1Name （802.11 r）。 |

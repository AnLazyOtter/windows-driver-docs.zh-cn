# <a name="wditlvproberesponseframe"></a>WDI\_TLV\_探测\_响应\_帧


WDI\_TLV\_探测\_响应\_帧是包含的探测响应帧 TLV。

## <a name="tlv-type"></a>TLV 类型


0x9

## <a name="length"></a>长度


UINT8 元素的数组大小 （以字节为单位）。 该数组必须包含一个或多个元素。

## <a name="values"></a>值


| 在任务栏的搜索框中键入      | 描述                                                         |
|-----------|---------------------------------------------------------------------|
| UINT8\[\] | 指定的探测响应框架 UINT8 元素的数组。 |

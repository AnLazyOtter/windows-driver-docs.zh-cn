# <a name="wditlvp2plistenstate"></a>WDI\_TLV\_P2P\_LISTEN\_STATE


WDI\_TLV\_P2P\_侦听\_状态是 TLV，其中包含 Wi-Fi Direct 侦听状态。

## <a name="tlv-type"></a>TLV 类型


0x81

## <a name="length"></a>长度


所有包含的元素的大小的总和 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入                                                         | 描述                            |
|--------------------------------------------------------------|----------------------------------------|
| [**WDI\_P2P\_LISTEN\_STATE**](https://msdn.microsoft.com/library/windows/hardware/dn926097) | 所需的 Wi-Fi Direct 侦听状态。 |

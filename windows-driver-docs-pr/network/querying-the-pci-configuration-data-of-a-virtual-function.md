# <a name="querying-the-pci-configuration-data-of-a-virtual-function"></a>查询虚函数的 PCI 配置的数据


本部分介绍通过其基础驱动程序可以获得信息从 PCI 配置空间为 PCI Express (PCIe) 虚拟函数 (VF) 的方法。 VF 公开支持单个根 I/O 虚拟化 (SR-IOV) 的网络适配器。

**请注意**  这些方法只能由过量的 HYPER-V 父分区在管理操作系统中运行的驱动程序。

 

本部分包括以下主题：

[PCI 配置空间查询的虚函数](querying-the-pci-configuration-space-for-a-virtual-function.md)

[虚拟函数的查询的 PCI 供应商和设备标识符](querying-the-pci-vendor-and-device-identifiers-for-a-virtual-function.md)

[查询 PCI 基址注册的虚拟函数](querying-the-pci-base-address-registers-of-a-virtual-function.md)

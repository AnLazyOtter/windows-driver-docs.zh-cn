# <a name="oidgencodriverversion"></a>OID_GEN_CO_DRIVER_VERSION

网卡驱动程序使用 NDIS 版本。 此 OID 是两个字节的长度;高字节是主版本号，低字节是次版本号。

## <a name="requirements"></a>要求

| | |
| --- | --- |
| Version | Windows Vista 及更高版本 |
| Header | Ntddndis.h （包括 Ndis.h） |

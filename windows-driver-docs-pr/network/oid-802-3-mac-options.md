# <a name="oid8023macoptions"></a>OID\_802\_3\_MAC\_选项





一种协议可以使用此 OID 以确定由无法模拟以太网基础驱动程序支持的功能。

基础驱动程序将返回零，表明它支持任何选项。

**请注意**  此 OID 已过时的 NDIS 6 驱动程序。

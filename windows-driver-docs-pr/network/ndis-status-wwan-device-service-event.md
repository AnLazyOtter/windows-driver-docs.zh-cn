# <a name="ndisstatuswwandeviceserviceevent"></a>NDIS\_状态\_WWAN\_设备\_服务\_事件


微型端口驱动程序使用 NDIS\_状态\_WWAN\_设备\_服务\_事件表示要通知的设备服务更改 MB 服务。

微型端口驱动程序只能使用此通知将发送未经请求的事件。

使用此通知[ **NDIS\_WWAN\_设备\_服务\_事件**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ndiswwan/ns-ndiswwan-_ndis_wwan_device_service_event)结构。

# <a name="wditlvbandcapabilities"></a>WDI\_TLV\_外\_功能


WDI\_TLV\_外\_功能是 TLV 包含带区的功能。

## <a name="tlv-type"></a>TLV 类型


0x1A

## <a name="length"></a>长度


所有包含的元素的大小的总和 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入   | 描述                                   |
|--------|-----------------------------------------------|
| UINT32 | 带区的标识符。                  |
| UINT8  | 指定是否启用带区。 |

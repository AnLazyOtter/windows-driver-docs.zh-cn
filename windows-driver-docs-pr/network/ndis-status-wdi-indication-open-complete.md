# <a name="ndisstatuswdiindicationopencomplete"></a>NDIS\_状态\_WDI\_指示\_打开\_完成


微型端口驱动程序使用 NDIS\_状态\_WDI\_指示\_打开\_完成，以指示完成[OID\_WDI\_任务\_打开](oid-wdi-task-open.md)。

|---------|
| 适配器 |

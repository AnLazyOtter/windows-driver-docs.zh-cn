# <a name="hyper-v-extensible-switch-status-indications"></a>Hyper-V 可扩展交换机状态指示


本部分介绍以下 NDIS 状态指示可发出或由 HYPER-V 可扩展交换机扩展：

-   [**NDIS\_状态\_交换机\_NIC\_状态**](ndis-status-switch-nic-status.md)
-   [**NDIS\_状态\_交换机\_端口\_删除\_VF**](ndis-status-switch-port-remove-vf.md)

有关详细信息如何扩展发出或处理可扩展切换扩展状态指示，请参阅[HYPER-V 可扩展交换机控制路径 NDIS 状态指示](https://msdn.microsoft.com/library/windows/hardware/hh598165)。

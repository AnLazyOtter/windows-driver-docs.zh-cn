# <a name="ndisstatuswwansethomeprovidercomplete"></a>NDIS\_状态\_WWAN\_设置\_主页\_提供程序\_完成

使用此通知[ **NDIS\_WWAN\_设置\_主页\_提供程序**](https://msdn.microsoft.com/library/windows/hardware/hh439841)结构。

# <a name="ndis_status_wwan_set_home_provider_complete"></a>\_WWAN\_设置\_HOME\_提供程序的 NDIS\_状态\_完成


微型端口驱动程序使用 NDIS\_状态\_WWAN\_设置\_HOME\_提供程序\_完成通知，通知 MB 服务完成[OID\_WWAN\_HOME\_提供程序](https://docs.microsoft.com/windows-hardware/drivers/network/oid-wwan-home-provider)集发出.

此通知使用[**NDIS\_WWAN\_集\_HOME\_提供程序**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndiswwan/ns-ndiswwan-_ndis_wwan_set_home_provider)结构。

# <a name="adding-and-dropping-parties"></a>添加和删除参与方





面向连接的客户端可以向使用[**NdisClMakeCall**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndis/nf-ndis-ndisclmakecall)启动的传出 multipoint 调用添加参与方。 面向连接的客户端也可以自行决定从 multipoint 调用中删除参与方，也可以在要删除的远程方请求时删除。

本部分包括：

[将参与方添加到 Multipoint 调用](adding-a-party-to-a-multipoint-call.md)

[从 Multipoint 调用中删除参与方](dropping-a-party-from-a-multipoint-call.md)

[从 Multipoint 调用中删除参与方的传入请求](incoming-request-to-drop-a-party-from-a-multipoint-call.md)

# <a name="sim-locking"></a>SIM 锁定


## <a name="supported-sim-lock-features"></a>支持的 SIM 锁定功能


Windows 10 移动版支持 PIN1、 pin2 码和这些个人锁定功能：

-   3GPP 网络锁定

-   3GPP 网络子集锁

-   3GPP 服务提供程序锁

-   3GPP 公司锁

-   3GPP SIM/USIM 锁

-   3GPP 2 网络类型 1 锁 （MCC，mnc）

-   3GPP 2 网络键入 2 锁 （IRM 基于最小值）

-   3GPP 2 HRPD 锁

-   3GPP 2 服务提供程序锁

-   3GPP 2 企业锁

-   3GPP 2 R UIM 锁

Microsoft 提供了 SIM 锁定 UI。 OEM 自定义项不需要为要显示此 UI。

**请注意**   SIM 锁 UI 不是由 Oem 进行配置。

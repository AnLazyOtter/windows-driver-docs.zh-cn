# <a name="wditlvbandid"></a>WDI\_TLV\_BANDID


WDI\_TLV\_BANDID 是 TLV，其中包含一个带区 id。

## <a name="tlv-type"></a>TLV 类型


0x39

## <a name="length"></a>长度


UINT32 大小 （以字节为单位）。

## <a name="values"></a>值


| 在任务栏的搜索框中键入   | 描述                  |
|--------|------------------------------|
| UINT32 | 带区的标识符。 |

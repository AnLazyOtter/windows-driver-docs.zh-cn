# <a name="avstream-driver-samples"></a>AVStream 驱动程序示例


此目录中的 AVStream 驱动程序示例用于编写自定义流式处理媒体设备的驱动程序将提供一个起始点。

## <a name="avstream"></a>AVStream


| 示例名称                    | 解决方案                                                           | 描述                                                                                                                                                           |
|--------------------------------|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| AVStream                       | [avshws](https://go.microsoft.com/fwlink/p/?LinkId=620185)          | 提供的 pin 以中心捕获驱动程序来模拟 AV 捕获硬件的模拟的硬件示例驱动程序。                                                            |
| AVSSamp                        | [avssamp](https://go.microsoft.com/fwlink/p/?LinkId=620186)         | 筛选器以中心 AVStream 模拟功能音频捕获示例驱动程序。                                                                                    |
| 驱动程序设备转换示例 | [sampledevicemft](https://go.microsoft.com/fwlink/p/?LinkId=620187) | 驱动程序设备转换在流式处理使用媒体基础的基于 Avstream 的照相机设备的进程中加载的演示示例。                         |
| Driver MFT                     | [SampleMft0](https://go.microsoft.com/fwlink/p/?LinkId=617126)      | 驱动程序 MFT 用于相机的 UWP 设备应用程序。 驱动程序 MFT 是捕获视频时使用特定照相机使用媒体基础转换。 |

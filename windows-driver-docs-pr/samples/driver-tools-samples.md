# <a name="tools-samples"></a>工具示例


此目录中的驱动程序示例包含用于显示功能和特性的 Static Driver Verifier (SDV) 的有意代码错误。

SDV 是系统地分析 Windows 内核模式驱动程序的源代码的静态验证工具。 SDV 包含在 Windows Driver Kit (WDK) 中，可以从 Microsoft Visual Studio 中运行。

## <a name="tools"></a>工具


| 示例名称             | 解决方案                                                                   | 描述                                                            |
|-------------------------|----------------------------------------------------------------------------|------------------------------------------------------------------------|
| SDV-FailDriver-KMDF     | [SDV-FailDriver-KMDF](https://go.microsoft.com/fwlink/p/?LinkId=617993)     | 此示例演示如何 SDV 可以找到错误 KMDF 驱动程序中。     |
| SDV-FailDriver-NDIS     | [SDV-FailDriver-NDIS](https://go.microsoft.com/fwlink/p/?LinkId=617995)     | 此示例演示如何 SDV 可以找到错误 NDIS 驱动程序中。     |
| SDV-FailDriver-Storport | [SDV-FailDriver-STORPORT](https://go.microsoft.com/fwlink/p/?LinkId=617997) | 此示例演示如何 SDV 可以找到错误 Storport 驱动程序中。 |
| SDV-FailDriver-WDM      | [SDV-FailDriver-WDM](https://go.microsoft.com/fwlink/p/?LinkId=617999)      | 此示例演示如何 SDV 可以找到错误 WDM 驱动程序中。      |

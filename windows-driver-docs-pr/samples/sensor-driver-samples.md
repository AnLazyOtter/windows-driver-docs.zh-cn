# <a name="sensor-driver-samples"></a>传感器驱动程序示例


此目录中的驱动程序示例为编写你的设备的自定义传感器驱动程序提供的起始点。

## <a name="sensors"></a>传感器


| 示例名称                   | 解决方案                                                                         | 描述                                                                                   |
|-------------------------------|----------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| 活动                      | [活动](https://go.microsoft.com/fwlink/p/?LinkId=617956)                      | 演示如何编写 UMDF v2 驱动程序来控制虚拟活动传感器。              |
| ADXL345Accelerometer          | [ADXL345Acc](https://go.microsoft.com/fwlink/p/?LinkId=617957)                    | 演示如何编写 UMDF v2 驱动程序来控制 ADXL345 加速感应器芯片。          |
| CustomSensors                 | [CustomSensors](https://go.microsoft.com/fwlink/p/?LinkId=617958)                 | 演示如何编写 UMDF v2 驱动程序来控制虚拟 CO2 传感器。                   |
| 步程计                     | [计步器](https://go.microsoft.com/fwlink/p/?LinkId=617959)                     | 演示如何编写 UMDF v2 驱动程序来控制虚拟计步器传感器。             |
| SimpleDeviceOrientationSensor | [SimpleDeviceOrientationSensor](https://go.microsoft.com/fwlink/p/?LinkId=617960) | 演示如何编写 UMDF v2 传感器驱动程序以输出简单设备方向的值。 |

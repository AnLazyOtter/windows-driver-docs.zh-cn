# <a name="serial-driver-samples"></a>串行驱动程序示例


此目录中的驱动程序示例为编写你的设备的自定义串行驱动程序提供的起始点。

## <a name="serial"></a>序列


| 示例名称      | 解决方案                                                         | 描述                                                                                                                                   |
|------------------|------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| Serenum          | [serenum](https://go.microsoft.com/fwlink/p/?LinkId=617961)       | 枚举即插即用 n 播放 RS-232 符合即插即用和播放外部 COM 设备的当前修订版本的设备。                      |
| 序列           | [serial](https://go.microsoft.com/fwlink/p/?LinkId=617962)        | 序列 (16550 基于 RS-232) 示例驱动程序是收件箱 Serial.sys 驱动程序的 WDF 版本。                                                |
| 虚拟序列   | [VirtualSerial](https://go.microsoft.com/fwlink/p/?LinkId=617963) | 此示例演示串行驱动程序。 它包括的简单虚拟串行驱动程序 (ComPort) 和控制器的无调制解调器驱动程序 (FakeModem)。 |
| 虚拟串行 2 | [VirtualSerial](https://go.microsoft.com/fwlink/p/?LinkId=617965) | 此示例演示串行驱动程序。 它包括的简单虚拟串行驱动程序 (ComPort) 和控制器的无调制解调器驱动程序 (FakeModem)。 |

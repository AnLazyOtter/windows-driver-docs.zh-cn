# <a name="wiadpczoomposition"></a>WIA\_DPC\_缩放\_位置


WIA\_DPC\_缩放\_位置的属性由 Microsoft 保留供将来使用，未实现这一次。

## <span id="ddk_wia_dpc_zoom_position_si"></span><span id="DDK_WIA_DPC_ZOOM_POSITION_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

应使用[ **WIA\_DPC\_焦点\_长度**](wia-dpc-focal-length.md)视觉缩放的属性和[ **WIA\_DPC\_数字\_缩放**](wia-dpc-digital-zoom.md)数字缩放调整的属性。

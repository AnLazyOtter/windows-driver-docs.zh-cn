# <a name="wiaipsmirror"></a>WIA\_IP\_镜像


WIA\_IP\_镜像属性保留供将来使用由 Microsoft 和未实现这一次。

## <span id="ddk_wia_ips_mirror_si"></span><span id="DDK_WIA_IPS_MIRROR_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

WIA\_IPS\_镜像属性不与相关[ **WIA\_IP\_方向**](wia-ips-orientation.md)属性。 WIA\_IPS\_方向指定要在与扫描的方向的关系; 与之相反，WIA 扫描的文档的方向\_IP\_镜像指定是要应用于的操作在扫描后的图像。

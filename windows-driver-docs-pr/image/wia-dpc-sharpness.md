# <a name="wiadpcsharpness"></a>WIA\_DPC\_锐度


WIA\_DPC\_清晰度属性指示的感知的清晰度的捕获的映像。

## <span id="ddk_wia_dpc_sharpness_si"></span><span id="DDK_WIA_DPC_SHARPNESS_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_列表或 WIA\_PROP\_范围

访问权限：读取/写入

<a name="remarks"></a>备注
-------

WIA\_DPC\_清晰度属性可以使用一系列值或一系列值。 最小值表示最少量的清晰度，并最大值表示最大的清晰度。 通常情况下，值范围表示正常，或默认的清晰度的中间。

# <a name="wiadpcpicturestaken"></a>WIA\_DPC\_图片\_录


WIA\_DPC\_图片\_录属性包含的照相机已采取的照片的数量。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dpc_pictures_taken_si"></span><span id="DDK_WIA_DPC_PICTURES_TAKEN_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

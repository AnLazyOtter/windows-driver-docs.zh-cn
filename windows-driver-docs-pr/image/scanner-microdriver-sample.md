# <a name="scanner-microdriver-sample"></a>扫描仪微驱动程序示例





*Microdrv* WDK 中的目录包含扫描仪示例 WIA microdriver。 Microdriver 更轻松地开发比完整 WIA 微型驱动程序，但只提供基本的功能。 请参阅[开发 WIA 扫描程序驱动程序](developing-a-wia-scanner-driver.md)microdriver 限制。 此示例演示如何编写适用于平板扫描仪 WIA microdriver。 此示例驱动程序可作为起始点进行开发，但您的驱动程序应通过提供与 Windows 的内核模式驱动程序之一访问扫描仪硬件。

# <a name="wiaipsautodeskew"></a>WIA\_IPS\_自动\_反扭曲


WIA\_IPS\_自动\_反扭曲属性指示设备应使用偏差的自动修正。 WIA 微型驱动程序创建并维护此属性。

属性类型：VT\_I4

有效值：WIA\_PROP\_列表

访问权限：读取/写入

<a name="remarks"></a>备注
-------

下表描述了有效使用 WIA 的常量\_IPS\_自动\_反扭曲属性。

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th>ReplTest1</th>
<th>定义</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>WIA_AUTO_DESKEW_ON</p></td>
<td><p>使用倾斜的自动修正。</p></td>
</tr>
<tr class="even">
<td><p>WIA_AUTO_DESKEW_OFF</p></td>
<td><p>不使用偏差的自动修正。</p></td>
</tr>
</tbody>
</table>

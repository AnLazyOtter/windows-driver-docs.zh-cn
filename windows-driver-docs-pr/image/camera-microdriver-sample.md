# <a name="camera-microdriver-sample"></a>相机 Microdriver 示例





中的 microdrv 目录[WIA 驱动程序示例](https://go.microsoft.com/fwlink/p/?linkid=256210)的数码照相机示例 WIA microdriver。

此示例演示如何编写照相机 WIA 用户模式下的 microdriver。 它通过从硬盘上的目录中读取图像中模拟照相机。 此示例驱动程序可作为起始点进行开发，但您的驱动程序应通过提供与 Windows 的内核模式驱动程序之一访问照相机硬件。

# <a name="rotation-element"></a>旋转元素


可选**旋转**元素指定要扫描的文档的每个图像的旋转的量。

<a name="usage"></a>用法
-----

```xml
<wscn:Rotation wscn:MustHonor=""               wscn:Override=""               wscn:UsedDefault=""
  MustHonor = "xs:string"
  Override = "xs:string"
  UsedDefault = "xs:string">
  text
</wscn:Rotation wscn:MustHonor=""               wscn:Override=""               wscn:UsedDefault="">
```

<a name="remarks"></a>备注
-------

整数值**旋转**元素指定要将应用于扫描的文档的每个映像的顺时针旋转。 所有 WSD 扫描服务必须都支持值 0 （即，没有旋转）。

客户端可以指定可选**MustHonor**属性时，才**旋转**元素包含在**CreateScanJobRequest**层次结构。 有关详细信息**MustHonor**及其使用情况，请参阅[ **CreateScanJobRequest**](createscanjobrequest.md)。

WSD 扫描服务可以指定可选**重写**并**UsedDefault**属性时，才**旋转**元素包含在**DocumentFinalParameters**层次结构。 有关详细信息**重写**并**UsedDefault**及其使用情况，请参阅[ **DocumentFinalParameters**](documentfinalparameters.md)。

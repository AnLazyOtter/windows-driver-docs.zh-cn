WIA 应用程序枚举扫描程序项树来确定扫描程序的受支持的功能中的顶级项。 然后，应用程序使用的顶级项作为扫描的源。 用于平板扫描仪项从平板扫描、 送纸器项用于文档送纸器，从扫描和其他操作。 送纸器扫描程序通常还具有这样的内容适用于平板扫描仪大部分也适用于送纸器扫描程序扫描图面上，平板。 有关基本与平板扫描仪扫描的其他信息，请参阅[基本扫描平板扫描仪](basic-scanning-for-flatbed-scanners.md)。

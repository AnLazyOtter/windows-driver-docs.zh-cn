# <a name="wiadpsusername"></a>WIA\_DPS\_USER\_NAME


WIA\_DPS\_用户\_WIA 驱动程序的根项的 NAME 属性允许驱动程序以获取的域名或计算机名称和时间访问给定时刻 WIA 设备的用户的用户名。 WIA 服务创建并维护此属性。

属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：只读

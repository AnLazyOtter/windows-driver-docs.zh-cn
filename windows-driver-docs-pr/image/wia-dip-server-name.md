# <a name="wiadipservername"></a>WIA\_DIP\_SERVER\_NAME


WIA\_DIP\_SERVER\_NAME 属性包含的服务器上运行 WIA 微型驱动程序的名称。

## <span id="ddk_wia_dip_server_name_si"></span><span id="DDK_WIA_DIP_SERVER_NAME_SI"></span>


属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

默认值的 WIA\_DIP\_SERVER\_名称是"本地"。 当应用程序连接到同一台计算机上的设备时，此属性应包含字符串"local"。

# <a name="windows-image-acquisition-drivers"></a>Windows 图像采集驱动程序

Microsoft Windows 图像采集 (WIA) 编程接口是 Windows 仍映像 (STI) 体系结构。 本文档说明了如何开发 WIA 驱动程序支持与如数字照相机和扫描仪的图像处理设备的通信。

WIA 是一个全面映像体系结构已从 Windows STI 模型演变而来的。 WIA 提供可靠性、 可靠性、 一致的用户体验和内置的驱动程序支持行业标准协议如图片传输协议 (PTP) 的照相机。

有关对映像应用程序进行编程的信息，请参阅[WIA 应用程序编程接口](https://docs.microsoft.com/windows/desktop/wia/-wia-wia-application-programming-interface)。

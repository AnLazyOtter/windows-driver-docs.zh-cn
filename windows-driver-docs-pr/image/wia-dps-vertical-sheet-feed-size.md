# <a name="wiadpsverticalsheetfeedsize"></a>WIA\_DPS\_垂直\_工作表\_馈送\_大小


WIA\_DPS\_垂直\_工作表\_馈送\_大小属性包含文档送纸器的扫描仪，但在一英寸的千分之几秒的物理垂直尺寸 (。 001)。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dps_vertical_sheet_feed_size_si"></span><span id="DDK_WIA_DPS_VERTICAL_SHEET_FEED_SIZE_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

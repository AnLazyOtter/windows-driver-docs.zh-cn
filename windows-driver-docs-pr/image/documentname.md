# <a name="documentname-element"></a>文档名元素


所需**文档名**元素包含客户端提供的文档的名称。

<a name="usage"></a>用法
-----

```xml
<wscn:DocumentName>
  text
</wscn:DocumentName>
```

<a name="attributes"></a>特性
----------

没有特性。

<a name="text-value"></a>文本值
----------

必需。 任何字符的字符串。

## <a name="child-elements"></a>子元素


没有子元素。

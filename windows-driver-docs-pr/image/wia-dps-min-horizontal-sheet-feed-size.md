# <a name="wiadpsminhorizontalsheetfeedsize"></a>WIA\_DPS\_MIN\_水平\_表\_馈送\_大小


WIA\_DPS\_MIN\_水平\_表\_馈送\_大小属性包含的最小页的扫描可以扫描程序的文档送纸器，请在物理的水平尺寸千分之一英寸 (。 001)。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dps_min_horizontal_sheet_feed_size_si"></span><span id="DDK_WIA_DPS_MIN_HORIZONTAL_SHEET_FEED_SIZE_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

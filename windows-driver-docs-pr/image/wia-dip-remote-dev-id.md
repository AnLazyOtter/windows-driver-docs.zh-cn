# <a name="wiadipremotedevid"></a>WIA\_DIP\_远程\_开发人员\_ID


WIA\_DIP\_远程\_开发人员\_ID 属性包含在远程计算机安装的 WIA 设备的设备标识符 (ID)。 WIA 服务创建并维护此属性。

## <span id="ddk_wia_dip_remote_dev_id_si"></span><span id="DDK_WIA_DIP_REMOTE_DEV_ID_SI"></span>


属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

WIA 服务使用 WIA\_DIP\_远程\_开发人员\_仅在内部 ID 属性。

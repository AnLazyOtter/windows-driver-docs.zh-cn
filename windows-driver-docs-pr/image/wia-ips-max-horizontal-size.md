# <a name="wiaipsmaxhorizontalsize"></a>WIA\_IPS\_最大\_水平\_大小


WIA\_IPS\_最大\_水平\_SIZE 属性包含物理扫描仪的平板、 千分之几秒的英寸为单位中的水平维度 (。 001)。 WIA 微型驱动程序创建并维护此属性。

属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

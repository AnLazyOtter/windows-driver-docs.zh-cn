# <a name="wiaipsopticalyres"></a>WIA\_IPS\_光学\_YRES


WIA\_IPS\_光学\_YRES 属性包含最高支持垂直光学设备的分辨率，以每英寸点数 (dpi)。 WIA 微型驱动程序创建并维护此属性。

属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

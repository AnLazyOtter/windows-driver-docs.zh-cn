# <a name="wiadipbaudrate"></a>WIA\_DIP\_BAUDRATE


WIA\_DIP\_BAUDRATE 属性包含当前波特率设置设备。 WIA 服务创建并维护此属性。

## <span id="ddk_wia_dip_baudrate_si"></span><span id="DDK_WIA_DIP_BAUDRATE_SI"></span>


属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

值 WIA\_DIP\_BAUDRATE 属性应为"空"，如果设备未连接的串行电缆。

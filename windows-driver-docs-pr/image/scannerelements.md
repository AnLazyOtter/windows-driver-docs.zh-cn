# <a name="scannerelements-element"></a>ScannerElements 元素


所需**ScannerElements**包含所有要返回到客户端的扫描程序信息。

<a name="usage"></a>用法
-----

```xml
<wscn:ScannerElements>
  child elements
</wscn:ScannerElements>
```

<a name="attributes"></a>特性
----------

没有特性。

# <a name="user-interface-extensions"></a>用户界面扩展





许多照相机和扫描程序供应商可能提供不是通过默认 WIA 用户界面 (UI) 可访问其设备上的功能。 此类功能可能包括专有格式中存储的图像的操作，如自身的专用的设备操作测试或缩放，等等。 为了适应用户界面使用了非标准功能和专用的设备和用户组，WIA 提供了 UI 扩展功能。 此功能允许供应商扩展的上下文菜单，属性表和系统提供用户界面的图像应用程序对话框。

本部分的其余部分包含以下主题：

[上下文菜单扩展](context-menu-extensions.md)

[控件面板向导替换](control-panel-wizard-replacement.md)

[映像应用程序对话框扩展](image-application-dialog-extensions.md)

[属性表扩展插件](property-sheet-extensions.md)

[用户界面扩展插件注册表项](user-interface-extension-registry-entries.md)

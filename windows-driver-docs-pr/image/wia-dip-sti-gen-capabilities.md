# <a name="wiadipstigencapabilities"></a>WIA\_DIP\_STI\_常规\_功能


WIA\_DIP\_STI\_常规\_功能属性包含对于设备，一般 STI 功能，从驱动程序的 INF 文件中获取。 WIA 服务创建并维护此属性。

## <span id="ddk_wia_dip_sti_gen_capabilities_si"></span><span id="DDK_WIA_DIP_STI_GEN_CAPABILITIES_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

应用程序读取 WIA\_DIP\_STI\_常规\_功能属性来确定设备的通用 STI 功能。

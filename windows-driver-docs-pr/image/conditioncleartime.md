# <a name="conditioncleartime-element"></a>ConditionClearTime 元素


所需**ConditionClearTime**元素指定的日期和时间扫描设备条件已被清除。

<a name="usage"></a>用法
-----

```xml
<wscn:ConditionClearTime>
  text
</wscn:ConditionClearTime>
```

<a name="attributes"></a>特性
----------

没有特性。

<a name="text-value"></a>文本值
----------

必需。 DateTime 类型的任何有效的值。 有关日期时间的详细信息，请参阅 XML 架构第 2 部分：数据类型第二版。**dateTimedateTime**

## <a name="child-elements"></a>子元素


没有子元素。

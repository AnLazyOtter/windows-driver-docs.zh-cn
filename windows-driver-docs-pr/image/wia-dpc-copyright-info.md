# <a name="wiadpccopyrightinfo"></a>WIA\_DPC\_版权所有\_信息


WIA\_DPC\_版权\_属性包含版权的通知的信息。

## <span id="ddk_wia_dpc_copyright_info_si"></span><span id="DDK_WIA_DPC_COPYRIGHT_INFO_SI"></span>


属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：读取/写入

<a name="remarks"></a>备注
-------

设备使用 WIA\_DPC\_版权所有\_信息属性来填充它会捕获每个 EXIF 图像中的版权字段。

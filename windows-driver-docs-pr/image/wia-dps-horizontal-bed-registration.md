# <a name="wiadpshorizontalbedregistration"></a>WIA\_DPS\_水平\_平台\_注册


WIA\_DPS\_水平\_平台\_注册属性中包含注册或水平对齐方式，为放置在扫描程序的平台的文档。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dps_horizontal_bed_registration_si"></span><span id="DDK_WIA_DPS_HORIZONTAL_BED_REGISTRATION_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

# <a name="wiaipaitemsstored"></a>WIA\_IPA\_项\_存储


WIA\_IPA\_项\_存储属性指定多少个项存储在存储中 (WIA\_类别\_文件夹) 项。 WIA 微型驱动程序创建并维护此 WIA 属性。

属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

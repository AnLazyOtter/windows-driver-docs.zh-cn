# <a name="sample-wia-drivers"></a>示例 WIA 驱动程序





本部分介绍了在 Windows 驱动程序开发工具包 (DDK)，Windows Driver Kit (WDK) 的 WIA 示例并处于联机状态。 大多数这些 WIA 示例适用于 Windows XP 和更高版本操作系统和不符合 Windows me 一起提供。 将相应说明的是 Windows Me 兼容样本。 虽然 Windows XP 的 WIA 驱动程序可能无法正常工作在 Windows Me，WIA 驱动程序为 Windows 编写我应在 Windows XP 上运行。

有关 WIA 示例的详细信息，请参阅[Windows 图像采集 (WIA) 驱动程序示例](https://go.microsoft.com/fwlink/p/?LinkId=618052)。

此外可以查找中的其他示例[创建 Hello World WIA 微型驱动程序](creating-a---hello-world---wia-minidriver.md)。

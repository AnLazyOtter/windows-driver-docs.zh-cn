# <a name="wiaipsprinterendorsernumlines"></a>WIA\_IPS\_打印机\_印记签署器\_NUM\_行


**WIA\_IPS\_打印机\_印记签署器\_NUM\_行**属性指定的每一侧的最大的可打印或认可的文本行数印刷器/印记签署器单位的文档。 此属性已初始化并维护 WIA 微型驱动程序，并且可与 Windows 8 和更高版本的 Windows。

属性类型：VT\_UI4

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

**WIA\_IPS\_打印机\_印记签署器\_NUM\_行**属性是可选的印刷器/印记签署器项。 时不支持此属性，则只有一行文本允许每侧的文档。

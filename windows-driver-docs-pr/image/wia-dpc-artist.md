# <a name="wiadpcartist"></a>WIA\_DPC\_艺术家


WIA\_DPC\_艺术家属性包含设备的所有者 （这是当前用户） 的名称。 设备使用此属性来填充它会捕获每个 EXIF 图像中的艺术家字段。

## <span id="ddk_wia_dpc_artist_si"></span><span id="DDK_WIA_DPC_ARTIST_SI"></span>


属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：读取/写入

# <a name="wia-property-mapping"></a>WIA 属性映射


本部分介绍自动回归树模型项和属性的映射。 从 Windows XP 到 Windows Vista 的对应的 WIA 属性此映射是 WIA 兼容性层的一部分。

请注意，Windows XP 属性标记为已过时，保留的还是被忽略，未映射到 Windows Vista 的属性。

此部分包含有关以下主题的信息：

[具有不同的 Id 和名称的 WIA 属性映射](mapping-wia-properties-with-different-ids-and-names.md)

[具有 Id 相同，但不同的名称的映射 WIA 属性](mapping-wia-properties-with-the-same-ids-but-different-names.md)

[具有相同的 Id 和名称的 WIA 属性映射](mapping-wia-properties-with-the-same-ids-and-names.md)

[不需要进行任何更改的 WIA 属性映射](mapping-wia-properties-that-need-no-changes.md)

[不需要进行任何更改的特殊情况的 WIA 属性映射](mapping-wia-properties-that-need-no-changes---special-cases.md)

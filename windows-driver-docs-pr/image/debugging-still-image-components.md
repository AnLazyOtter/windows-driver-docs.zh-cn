# <a name="debugging-still-image-components"></a>调试静态图像组件

为了帮助调试的供应商提供静止图像组件，可以使用命令行选项，请使用来修改静止图像事件监视器的行为**运行**的选项**启动**菜单。 你可使用以下选项：

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th>命令行选项</th>
<th>定义</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p><strong>stimon /h</strong></p></td>
<td><p>隐藏消息窗口。</p></td>
</tr>
<tr class="even">
<td><p><strong>Stimon /r</strong></p></td>
<td><p>刷新事件监视程序的设备列表。</p></td>
</tr>
<tr class="odd">
<td><p><strong>Stimon /t</strong><em>数</em></p></td>
<td><p>将修改为指定的秒数的轮询间隔<em>数</em>。 通常用于增加轮询间隔。</p></td>
</tr>
<tr class="even">
<td><p><strong>Stimon /v</strong></p></td>
<td><p>使窗口可见，它显示事件监视器消息。</p></td>
</tr>
</tbody>
</table>

事件监视器可以停止和启动通过使用计算机管理窗口。 事件监视器被列为"静止图像服务"。

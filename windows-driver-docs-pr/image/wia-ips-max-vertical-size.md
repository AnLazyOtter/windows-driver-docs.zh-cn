# <a name="wiaipsmaxverticalsize"></a>WIA\_IPS\_MAX\_VERTICAL\_SIZE


WIA\_IPS\_最大\_垂直\_大小属性包含扫描仪的平板、 千分之几秒的英寸为单位中的物理垂直尺寸 (。 001)。 WIA 微型驱动程序创建并维护此属性。

属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

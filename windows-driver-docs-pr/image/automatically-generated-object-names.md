# <a name="automatically-generated-object-names"></a>自动生成的对象名称





如果**文件名**ObjectInfo 数据集的字段为空白，Microsoft PTP WIA 微型驱动程序将创建基于 ObjectFormat 和 ObjectHandle 字段的名称。 例如，具有 ObjectHandle 值 5 的映像称为 Image5 Windows 的英语版本上。

# <a name="maxvalue-element"></a>MaxValue 元素


所需**MaxValue**元素指定扫描设备支持需要的值范围的扫描程序配置元素的最大值。

<a name="usage"></a>用法
-----

```xml
<wscn:MaxValue>
  text
</wscn:MaxValue>
```

<a name="attributes"></a>特性
----------

没有特性。

<a name="text-value"></a>文本值
----------

必需。 有关可能的值，请参阅特定父元素。

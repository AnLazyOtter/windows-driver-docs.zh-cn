# <a name="scannerstatus-element"></a>ScannerStatus 元素


所需**ScannerStatus**元素包含与扫描程序相关的信息的当前状态自动机时 （例如时间以及更改扫描程序中的条件） 控制。

<a name="usage"></a>用法
-----

```xml
<wscn:ScannerStatus>
  child elements
</wscn:ScannerStatus>
```

<a name="attributes"></a>特性
----------

没有特性。

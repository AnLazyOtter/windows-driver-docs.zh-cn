# <a name="wiadpafirmwareversion"></a>WIA\_DPA\_固件\_版本


WIA\_DPA\_固件\_属性包含设备固件版本的版本。 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dpa_firmware_version_si"></span><span id="DDK_WIA_DPA_FIRMWARE_VERSION_SI"></span>


属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

值 WIA\_DPA\_固件\_版本属性必须是字符串值，如"1.0.4"或"1.0abc"。

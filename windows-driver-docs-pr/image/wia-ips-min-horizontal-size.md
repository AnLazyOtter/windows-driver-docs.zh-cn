# <a name="wiaipsminhorizontalsize"></a>WIA\_IPS\_MIN\_水平\_大小


WIA\_IPS\_MIN\_水平\_SIZE 属性包含的扫描程序的文档送纸器可以扫描，以英寸为单位的千分之几秒的最小页物理的水平尺寸 (。 001)。 WIA 微型驱动程序创建并维护此属性。

属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

# <a name="wia-scanner-storage-compatibility-for-windows-xp-and-windows-me"></a>Windows XP 和 Windows Me 的 WIA 扫描仪存储兼容性


Windows Vista 中的 WIA 服务包含一个内部的兼容性层，以便与新的 Windows Vista 应用程序和驱动程序，可以使用旧的 Windows XP 驱动程序和应用程序。 兼容性层提供平板和送纸器项翻译。 因为存储项对其进行*不*存在有关 Windows XP 或 Windows Me 和是否未翻译的兼容性层，存储的项将不可用的旧 WIA 应用程序。

有关 Windows XP 和 Windows Me 兼容性的其他说明，请参阅[WIA 平板扫描仪兼容性的 Windows Me 和 Windows XP](wia-flatbed-scanner-compatibility-for-windows-xp-and-windows-me.md)。

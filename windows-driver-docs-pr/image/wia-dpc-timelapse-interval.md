# <a name="wiadpctimelapseinterval"></a>WIA\_DPC\_TIMELAPSE\_INTERVAL


WIA\_DPC\_TIMELAPSE\_间隔属性包含的时间，以毫秒为单位，慢镜头捕获操作中的映像捕获。

## <span id="ddk_wia_dpc_timelapse_interval_si"></span><span id="DDK_WIA_DPC_TIMELAPSE_INTERVAL_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE、 WIA\_PROP\_列表中或 WIA\_PROP\_范围

访问权限：读取/写入

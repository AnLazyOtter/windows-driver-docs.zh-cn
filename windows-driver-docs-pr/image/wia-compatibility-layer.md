# <a name="wia-compatibility-layer"></a>WIA 兼容性层


WIA 兼容性层允许旧版应用程序和设备与 Windows Vista 的使用。

有两个函数的 WIA 兼容性层：

1.  不同的项树体系结构和属性的旧应用程序和 Windows Vista 驱动程序和 Windows Vista 应用程序和旧驱动程序之间的映射。

2.  Windows Vista 之间的映射，基于流的数据传输模型和旧的基于 TYMED 的模型。

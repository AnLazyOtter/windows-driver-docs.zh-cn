# <a name="wiaipsminverticalsize"></a>WIA\_IPS\_MIN\_VERTICAL\_SIZE


WIA\_IPS\_MIN\_垂直\_大小属性包含可以扫描扫描程序的文档送纸器，以英寸为单位的千分之几秒的最小页的物理垂直尺寸 (。 001)。 WIA 微型驱动程序创建并维护此属性

属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

# <a name="wia-scanner-storage"></a>WIA 扫描程序存储


本部分介绍在 Windows Vista 中引入的扫描仪的配备了能够图像和其他类型的获得的数据仍保留的数字存储 WIA 项树状结构。

此类存储可以固定大小 （例如硬盘驱动器） 或可移动 （例如内存 Compact Flash 卡）。 此存储可以添加在内部或外部连接到扫描程序;它可能包含其他类型的数据以及扫描生成的映像。

[扫描程序存储体系结构](scanner-storage-architecture.md)

[扫描程序存储项树上的实现说明](implementing-scanner-storage-item-trees.md)

[WIA 扫描程序存储兼容性的 Windows XP 和 Windows Me](wia-scanner-storage-compatibility-for-windows-xp-and-windows-me.md)

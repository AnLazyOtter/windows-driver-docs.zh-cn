# <a name="64-bit-wia-driver-components"></a>64 位 WIA 驱动程序组件


64 位 WIA 微型驱动程序被加载到 WIA 服务的过程中，这是在 64 位版本的 Windows 操作系统的 64 位进程。 因此，针对基于 x64 的版本的 Windows XP、 Windows Server 2003 和 Windows Vista 的 WIA 驱动程序可以包含仅 64 位驱动程序组件。 例如，它必须包含 64 位微型驱动程序、 64 位 WIA 微型驱动程序用户界面扩展和可能的 64 位内核模式驱动程序。

# <a name="wiadpcburstnumber"></a>WIA\_DPC\_迸发\_数


WIA\_DPC\_迸发\_NUMBER 属性包含的设备尝试迸发操作期间捕获的映像数量。

## <span id="ddk_wia_dpc_burst_number_si"></span><span id="DDK_WIA_DPC_BURST_NUMBER_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE、 WIA\_PROP\_列表中或 WIA\_PROP\_范围

访问权限：读取/写入

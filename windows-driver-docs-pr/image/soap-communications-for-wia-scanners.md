# <a name="soap-communications-for-wia-scanners"></a>WIA 扫描仪的 SOAP 通信


使用 WSD 的 WIA 驱动程序通过使用简单对象访问协议 (SOAP)，这一种协议，窗体的 web 服务堆栈的基础层与扫描程序设备进行通信。

该驱动程序必须使用功能发现服务以与设备初始化 SOAP 通信。

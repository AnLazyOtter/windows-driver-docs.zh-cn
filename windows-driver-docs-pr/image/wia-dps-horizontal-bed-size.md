# <a name="wiadpshorizontalbedsize"></a>WIA\_DPS\_水平\_平台\_大小


WIA\_DPS\_水平\_平台\_SIZE 属性包含物理扫描仪的平板、 千分之几秒的英寸为单位中的尺寸 (。 001)。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dps_horizontal_bed_size_si"></span><span id="DDK_WIA_DPS_HORIZONTAL_BED_SIZE_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

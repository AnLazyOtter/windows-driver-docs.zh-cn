# <a name="command-processing-and-event-reporting"></a>命令处理和事件报告





WIA 微型驱动程序可以支持从 WIA 应用程序，或直接从 WIA 服务发出的命令请求。 此外，某些静止图像设备具有的功能将通知发送到用户操作结果的驱动程序。 例如，具有一个或多个前面板按钮的扫描仪可以通知该驱动程序在用户按这些按钮之一时。 此功能，驱动程序采取某些操作。 当驱动程序通知 WIA 服务已按下某个特定按钮时，WIA 服务可以启动以前注册的应用程序处理的事件。

本部分包含以下主题：

[命令处理](command-handling.md)

[事件报告](event-reporting.md)

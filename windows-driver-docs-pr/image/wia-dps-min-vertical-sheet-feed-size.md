# <a name="wiadpsminverticalsheetfeedsize"></a>WIA\_DPS\_MIN\_VERTICAL\_SHEET\_FEED\_SIZE


WIA\_DPS\_MIN\_垂直\_表\_馈送\_SIZE 属性包含的最小页的扫描可以扫描程序的文档送纸器，请在物理垂直维度千分之一英寸 (。 001)。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dps_min_vertical_sheet_feed_size_si"></span><span id="DDK_WIA_DPS_MIN_VERTICAL_SHEET_FEED_SIZE_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

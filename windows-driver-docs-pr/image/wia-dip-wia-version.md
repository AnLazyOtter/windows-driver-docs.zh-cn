# <a name="wiadipwiaversion"></a>WIA\_DIP\_WIA\_版本


WIA\_DIP\_WIA\_版本属性包含的计算机安装的当前 WIA 版本 （作为字符串） 数。 WIA 服务创建并维护此属性。

## <span id="ddk_wia_dip_wia_version_si"></span><span id="DDK_WIA_DIP_WIA_VERSION_SI"></span>


属性类型：VT\_BSTR

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

应用程序读取 WIA\_DIP\_WIA\_版本以确定计算机安装的 WIA 的版本。

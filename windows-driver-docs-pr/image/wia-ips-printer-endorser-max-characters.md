# <a name="wiaipsprinterendorsermaxcharacters"></a>WIA\_IPS\_打印机\_印记签署器\_最大\_字符


**WIA\_IPS\_打印机\_印记签署器\_最大\_字符**属性描述最大数目的字符 （不包括 NULL 终止符） 的印刷器/印记签署器项可以打印或赞同将每个页面上。 此属性已初始化并维护 WIA 微型驱动程序，并且可与 Windows 8 和更高版本的 Windows。

属性类型：VT\_UI4

有效值：WIA\_PROP\_NONE

访问权限：只读

<a name="remarks"></a>备注
-------

**WIA\_IPS\_打印机\_印记签署器\_最大\_字符**印刷器/印记签署器的所有项必须都支持属性。 实现时，属性值**必须是**大于零 (0)。

# <a name="wiaipauploaditemsize"></a>WIA\_IPA\_上传\_项\_大小


WIA\_IPA\_上传\_项\_应用程序使用大小属性来指定要将某个项上传的字节数。 应用程序创建并维护此属性。

属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：读取/写入

# <a name="wiadpsopticalxres"></a>WIA\_DPS\_光学\_XRES


WIA\_DPS\_光学\_XRES 属性包含最高支持水平光学设备的分辨率，以每英寸点数 (dpi)。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dps_optical_xres_si"></span><span id="DDK_WIA_DPS_OPTICAL_XRES_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE。

访问权限：只读

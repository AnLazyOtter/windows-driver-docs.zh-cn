# <a name="wia-item-trees"></a>WIA 项树





WIA 项表示设备数据和设备属性。 图像处理应用程序显示 WIA 设备为层次结构树的项，但根项，它表示在设备本身，以及任何表示可编程数据源、 图像或包含图像的文件夹的子项目。 如果更改或配置而无需 （例如，当 TWAIN 驱动程序删除映像时） 使用 WIA 的设备数据或在项目树中的项所表示的设备属性，必须刷新 WIA 项树。 有关详细信息，请参阅[WIA TWAIN 兼容性](wia-twain-compatibility.md)。

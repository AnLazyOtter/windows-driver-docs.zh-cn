# <a name="wiaipcthumbnailheight"></a>WIA\_IPC\_缩略图\_高度


WIA\_IPC\_缩略图\_高度属性包含的当前高度，以像素为单位存储在设备的缩略图。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_ipc_thumbnail_height_si"></span><span id="DDK_WIA_IPC_THUMBNAIL_HEIGHT_SI"></span>


属性类型：VT\_I4

有效值：WIA\_PROP\_NONE

访问权限：只读

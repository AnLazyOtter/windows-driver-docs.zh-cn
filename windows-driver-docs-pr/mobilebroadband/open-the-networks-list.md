# <a name="open-the-networks-list"></a>打开网络列表


可以通过调用打开网络列表[ **ShowConnectionUI** ](https://msdn.microsoft.com/library/windows/apps/hh921594)此帐户当前的网络对象的方法。

例如：

``` syntax
account.currentNetwork.showConnectionUI()
```

# <a name="help-and-support"></a>帮助和支持


可以通过提供帮助减少客户支持呼叫并支持通过你的移动宽带应用。

在应用中，您可以访问除了辅助的支持渠道之外的自助式材料。 可以创建的用户可以在其中找到最常见的问题的答案的部分。 可以简化基于用户的设备或固件的帮助信息。

可以通过显示仅在用户的设备的帮助来简化用户体验。 可以通过使用获取设备型号和固件信息[订阅服务器和设备信息 API](subscriber-and-device-information-api.md)。

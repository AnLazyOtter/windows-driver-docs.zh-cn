# <a name="windowsinfo-xml-schema"></a>WindowsInfo XML 架构

[!include[MBAE deprecation warning](mbae-deprecation-warning.md)]


元数据包中包含一个 WindowsInfo.xml 文档。 其中包含操作系统使用定义的元数据的包中指定的服务的显示操作的信息。

文档的格式设置的 Windowsinfo.xml 中的数据基于 WindowsInfo XML 架构，这在本部分中所述。

**请注意**  必须通过使用 utf-8 编码保存的 XML 文档。

 

WindowsInfo XML 架构的完整定义，请参阅[WindowsInfo XML 架构定义](windowsinfo-xml-schema-definition.md)。

有关由 WindowsInfo XML 架构定义的元素的信息，请参阅[WindowsInfo XML 元素](windowsinfo-xml-elements.md)。

WindowsInfo XML 架构定义的格式中的 XML 数据的示例，请参阅[WindowsInfo XML 示例](windowsinfo-xml-example.md)。

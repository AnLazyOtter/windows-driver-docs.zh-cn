# <a name="hot-spot-authentication"></a>热点身份验证


你应卸载到 Wi-fi 热点，只要有可能的数据流量。 可用时，Windows 连接管理器可以自动连接到 Wi-fi 网络。 若要连接到 Wi-fi 网络，必须向 Windows 提供热点凭据。 可以通过在应用中实现你自己的身份验证提供热点凭据。

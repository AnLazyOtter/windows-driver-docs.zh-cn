# <a name="apn-xml-schema"></a>APN XML 架构


APN XML 架构指定 APN 数据库的内容。

**请注意**  必须通过使用 utf-8 编码保存的 XML 文档。

 

APN XML 架构的完整定义，请参阅[APN 架构定义](apn-schema-definition.md)。

有关由 APN XML 架构定义的元素的信息，请参阅[APN 元素](apn-elements.md)。

APN XML 架构定义的格式中的 XML 数据的示例，请参阅[APN 示例](apn-example.md)。

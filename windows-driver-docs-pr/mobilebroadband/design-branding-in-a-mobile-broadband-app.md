# <a name="design-branding-in-a-mobile-broadband-app"></a>设计移动宽带应用中的品牌


UWP 应用建议的品牌中通过以下方式为：

-   **自定义的颜色和字体**这包括能够自定义背景色和字体颜色。 例如，您可以在应用中使用外观或品牌颜色。

-   **使用品牌图像**Windows 连接管理器使用徽标以及元数据。 我们建议你将图像简单，若要保留的应用程序邀请的整体外观。 开始屏幕磁贴和初始屏幕上，可以使用品牌图像。

可以使用在以下位置中移动宽带应用品牌：

-   启动屏幕磁贴 （小）

    ![启动屏幕 1x1 磁贴](images/mb-fig1-start-screen-small-tile.png)

-   启动屏幕磁贴 （中）

    ![中等磁贴](images/mb-design-start-screen-medium-tile.png)

-   启动屏幕磁贴 （宽）

    ![启动屏幕 1 x 2 个磁贴](images/mb-fig2-start-screen-large-tile.png)

-   启动屏幕磁贴 （大）

    ![小磁贴](images/mb-fig2-start-screen-2x1-tile.png)

-   [与其他 Windows 组件集成，移动宽带应用](integrate-a-mobile-broadband-app-with-other-windows-components.md#app-settings)显示品牌名称

-   卸载上的用户界面 (UI)**启动**菜单会显示一个图标和名称

-   Toast 通知

    ![toast 通知](images/mb-fig3-toast.png)

-   使用品牌图标和名称的任务管理器

-   在计算机中的移动宽带信息设置 （显示运算符名称）

-   初始屏幕

    ![初始屏幕](images/mb-fig4-splash-screen.png)

-   Windows 连接管理器

    ![windows 连接管理器](images/mb-fig5-wcm.png)

有关 Windows 10 的品牌指导的详细信息，请参阅[UWP 应用营销准则](https://docs.microsoft.com/windows/uwp/publish/app-marketing-guidelines)。

## <a name="span-idquicksummaryspanspan-idquicksummaryspanspan-idquicksummaryspanquick-summary"></a><span id="Quick_summary"></span><span id="quick_summary"></span><span id="QUICK_SUMMARY"></span>快速摘要


适合品牌的运算符的设计：

-   使用主要和次要企业品牌颜色背景和字体应用程序中。

-   适当地使用品牌图标来维护简单起见，按照 UWP 应用程序设计指南。

运算符品牌的设计不完善：

-   不被填满图标和图像的空白区域。

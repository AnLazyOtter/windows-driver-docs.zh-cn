# <a name="sim-pin-api"></a>SIM 卡 PIN API


通过使用 SIM PIN API，你的移动宽带应用可以支持与 SIM PIN 和 Pin 解锁密钥 (PUK) 相关的操作。

有关 SIM PIN API 的详细信息，请参阅[移动宽带 API 接口](https://msdn.microsoft.com/library/windows/desktop/dd323268)。

# <a name="packageinfo-xml-schema-overview"></a>PackageInfo XML 架构概述

[!include[MBAE deprecation warning](mbae-deprecation-warning.md)]

元数据包中包含一个 PackageInfo.xml 文档，包含操作系统使用要安装元数据包和引用其内容的信息。

文档的格式设置的 PackageInfo.xml 中的数据基于 PackageInfo XML 架构，这在本部分中所述。

**注意**  该 XML 文档必须使用 UTF-8 编码进行保存。

 

PackageInfo XML 架构的完整定义，请参阅[PackageInfo XML 架构定义](packageinfo-xml-schema-definition.md)。

有关由 PackageInfo XML 架构定义的元素的信息，请参阅[PackageInfo XML 元素](packageinfo-xml-elements.md)。

有关由 PackageInfo XML 架构定义的数据类型的信息，请参阅[PackageInfo XML 数据类型](packageinfo-xml-data-types.md)。

PackageInfo XML 架构定义的格式中的 XML 数据的示例，请参阅[PackageInfo XML 示例](packageinfo-xml-example.md)。

# <a name="design-services-and-goods-pages-in-a-mobile-broadband-app"></a>设计移动宽带应用中的服务和产品页


移动宽带应用可用于宣传的其他相关的服务。 使用建议的布局来组织想要通过 Microsoft Store 提升的各种服务。 下面提供了按类别组织服务提升示例。

![服务和产品页](images/mb-fig1-services-and-goods-page.png)

## <a name="span-idquicksummaryspanspan-idquicksummaryspanspan-idquicksummaryspanquick-summary"></a><span id="Quick_summary"></span><span id="quick_summary"></span><span id="QUICK_SUMMARY"></span>快速摘要


适合的服务和产品页面的设计：

-   请按照服务页的网格布局。

-   使用图片和简短的标题来描述每个服务。

-   使用跨升级为 Microsoft Store。

-   使用自然的平移和缩放以及水平滚动。

服务页的设计不完善：

-   不链接到应用外部的外部站点。

# <a name="mobile-plans-integration"></a>移动计划集成

[!include[Mobile Plans Beta Prerelease](../mobile-plans-beta-prerelease.md)]

本主题详细介绍了集成的后端移动运营商和 Microsoft 系统组合起来以实现移动计划所需的工作。 完成此操作，集成将在准备就绪的生产检查点。

## <a name="mobile-operator-integration-into-microsoft-environment"></a>移动运营商集成到 Microsoft 的环境

Microsoft 将驱动器的集成，但它应移动运营商将在此过程中，包括访问 esim 卡配置文件，移动运营商帐户登录到 web 体验，支持 Microsoft 和工程资源来驱动 bug 修复。

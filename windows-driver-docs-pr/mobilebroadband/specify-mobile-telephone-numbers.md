# <a name="specify-mobile-telephone-numbers"></a>指定移动电话号码


必须以 ITU E.164 格式指定电话号码。 建议 ITU E.164 国际格式存储电话号码和一个库用于 ITU E.164 格式的电话号码转换为区域、 常见的书面格式的 SMS 客户端。

# <a name="cmprobtranslationfailed"></a>CM_PROB_TRANSLATION_FAILED

此函数保留供系统使用。

设备资源转换失败。

## <a name="error-code"></a>错误代码

33

### <a name="display-message"></a>显示消息

"Windows 无法确定哪些资源所需的此设备。 （代码 33）"

### <a name="recommended-resolution"></a>建议的解决方法

请尝试使用 BIOS 设置实用程序，或更新 BIOS。

配置、 修复或更换硬件。

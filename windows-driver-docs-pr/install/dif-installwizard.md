# <a name="difinstallwizard"></a>DIF_INSTALLWIZARD


此差异代码已过时，并在 Microsoft Windows 2000 和更高版本的 Windows 中不再受支持。

对于即插即用设备，Windows 使用 DIF_NEWDEVICEWIZARD_*XXX*请求，如[ **DIF_NEWDEVICEWIZARD_FINISHINSTALL**](dif-newdevicewizard-finishinstall.md)。

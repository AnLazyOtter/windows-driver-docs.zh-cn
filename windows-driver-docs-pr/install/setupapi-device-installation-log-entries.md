# <a name="setupapi-device-installation-log-entries"></a>SetupAPI 设备安装日志条目


在 Windows Vista 和更高版本的 Windows， [SetupAPI 日志](setupapi-logging--windows-vista-and-later-.md)中可以使用来验证安装设备并排查设备安装问题的纯文本日志文件的设备安装有关的信息。 如果存在签名问题，安装程序 Api 将日志文件中记录有关签名问题的信息。 此日志文件的名称是*SetupAPI.dev.log*，和它位于，默认情况下，在 Windows INF 文件目录 (*%systemroot%\\inf*)。

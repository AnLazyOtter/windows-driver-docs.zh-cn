# <a name="driver-packages"></a>驱动程序包


一个*驱动程序包*必须提供使你的设备以支持在 Windows 下的所有软件组件都组成。

安装的设备或驱动程序涉及到系统提供和供应商提供的组件。 系统提供的所有设备类的通用安装软件。 供应商必须提供的驱动程序包中的一个或多个特定于设备的组件。

本部分提供信息以帮助你确定要提供的驱动程序包的组件。

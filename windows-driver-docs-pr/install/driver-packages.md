# <a name="driver-packages"></a>驱动程序包


*驱动程序包*包含必须提供的所有软件组件，才能让设备在 Windows 下受支持。

安装设备或驱动程序涉及系统提供的组件和供应商提供的组件。 系统为所有设备类提供通用安装软件。 供应商必须在驱动程序包中提供一个或多个设备特定组件。

本部分提供的信息可帮助你确定要在驱动程序包中提供的组件。

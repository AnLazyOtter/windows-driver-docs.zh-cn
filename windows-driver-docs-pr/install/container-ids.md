# <a name="container-id"></a>容器 ID


容器 ID 是唯一组功能的设备与计算机中安装的单函数或多功能设备相关联的系统提供的设备标识字符串。

从 Windows 7 开始，插即用 (PnP) 管理器使用容器 ID 来一个或多个设备节点进行分组 (*devnodes*) 的源自和属于特定的物理设备的每个实例。 此实例称为*设备**容器*。

分组源自单个设备的实例的所有 devnodes 中可实现以下功能：

-   操作系统可以确定如何在子 devnodes 和其容器 devnode 相关功能。

-   用户或应用程序会显示而不是传统函数为中心的视图的设备的设备为中心的视图。

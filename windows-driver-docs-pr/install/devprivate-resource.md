# <a name="devprivateresource"></a>DEVPRIVATE_RESOURCE


**请注意**  此结构是仅供内部使用。

 

DEVPRIVATE_RESOURCE 结构用于指定资源的列表或了描述的设备实例专用的特定于设备的资源使用情况的资源要求列表。 有关资源的列表和资源要求列表的详细信息，请参阅[硬件资源](https://msdn.microsoft.com/library/windows/hardware/ff547012)。

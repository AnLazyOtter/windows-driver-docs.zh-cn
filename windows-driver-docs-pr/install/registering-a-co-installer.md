# <a name="registering-a-co-installer"></a>注册辅助安装程序





对于单一设备或特定的安装程序类中的所有设备，可以注册共同安装程序。 特定于设备的共同安装程序是动态注册通过 INF 文件时安装了一个其设备。 手动或通过自定义设备安装程序应用程序和 INF 注册类共同安装程序。

有关详细信息，请参阅

[注册的特定于设备的共同安装程序](registering-a-device-specific-co-installer.md)

[注册类共同安装程序](registering-a-class-co-installer.md)

若要更新辅助安装程序，每个新版本的 DLL 应具有新的文件名，因为该 DLL 通常在使用当用户单击设备属性页上的更新驱动程序按钮。

# <a name="setupapi-logging"></a>SetupAPI 日志记录


在 Windows Vista 和更高版本的 Windows， [SetupAPI](setupapi.md)包括以下日志记录组件：

-   插即用 (PnP) 管理器和 SetupAPI 设备安装文本日志中记录有关安装事件的信息 (*SetupAPI.dev.log*) 和应用程序安装文本日志 (*SetupAPI.app.log*). 设备安装文本日志包含有关设备和驱动程序安装的信息和应用程序安装文本日志包含有关与设备驱动程序安装相关联的应用程序软件安装的信息。 SetupAPI 文本日志的内容的详细信息，请参阅[SetupAPI 文本日志](setupapi-text-logs.md)，和有关如何启用日志记录的信息，请参阅[SetupAPI 日志记录注册表设置](setupapi-logging-registry-settings.md)。

-   [SetupAPI 日志记录功能](https://docs.microsoft.com/previous-versions/ff550878(v=vs.85))的即插即用设备安装应用程序、 类安装程序和共同安装程序可以使用 SetupAPI 文本日志中写入日志条目。 有关如何使用这些函数的信息，请参阅[使用 SetupAPI 日志记录函数](using-the-setupapi-logging-functions.md)。

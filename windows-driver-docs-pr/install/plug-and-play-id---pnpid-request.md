# <a name="plug-and-play-id---pnpid-request"></a>即插即用 ID - PNPID 请求


现在通过 UEFI.org 网站请求插 ID。 若要请求 ID，请参阅[即插即用 ID 和 ACPI ID 注册表](https://www.uefi.org/PNP_ACPI_Registry)。

# <a name="creating-a-catalog-file-for-a-test-signed-driver-package"></a>为测试签名驱动程序包创建编录文件


若要创建[编录文件](catalog-files.md)测试签名的 PnP[驱动程序包](driver-packages.md)，按照相同的步骤用于[创建编录文件的即插即用驱动程序包](creating-a-catalog-file-for-a-pnp-driver-package.md)。

若要创建编录文件的测试签名非 PnP 驱动程序包，请执行相同的过程用于[创建非 PnP 驱动程序包的目录文件](creating-a-catalog-file-for-a-non-pnp-driver-package.md)。

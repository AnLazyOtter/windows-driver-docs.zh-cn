# <a name="devprivaterange"></a>DEVPRIVATE_RANGE


**请注意**  此结构是仅供内部使用。

 

DEVPRIVATE_RANGE 结构指定了描述的设备实例专用的特定于设备的资源使用情况的资源要求列表。 有关资源要求列表的详细信息，请参阅[硬件资源](https://docs.microsoft.com/windows-hardware/drivers/kernel/hardware-resources)。

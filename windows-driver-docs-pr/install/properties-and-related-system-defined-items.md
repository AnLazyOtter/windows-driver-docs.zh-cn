# <a name="properties-and-related-system-defined-items"></a>属性和相关的系统定义项


在 Windows Vista 和更高版本的 Windows 中，统一的设备属性模型管理与安装相关的以下系统定义项和设备的管理之间的对应关系：

-   [系统定义的属性](system-defined-device-properties2.md)及其对应[属性键](property-keys.md)，[属性的数据类型](property-data-type-identifiers.md)，和属性值。

-   SPDRP_*Xxx*设备实例属性标识符和 SPCRP_*Xxx*中定义的设备安装程序类属性标识符*Setupapi.h。* CM_DRP_*Xxx*设备实例属性标识符和 CM_CRP_*Xxx*[设备安装程序类](device-setup-classes.md)中定义的标识符*Cfgmgr32.h*.

-   REGSTR_VAL_*Xxx*与设备安装和管理相关的注册表条目值标识符。 在中定义这些标识符*Regstr.h*。

-   注册表条目对应于设备属性的值。

-   修改设备属性的 INF 文件条目值。

与设备属性相关联的系统定义的项之间关系的信息，请参阅以下主题：

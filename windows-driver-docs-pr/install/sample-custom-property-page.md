# <a name="sample-custom-property-page"></a>示例自定义属性页


下面的对话框中包含一个示例自定义属性页：

![示例自定义属性页。](images/sampprop.png)

在上一对话框中，提供程序提供一个名为属性的属性页。 默认情况下，则系统将提供**常规**，**驱动程序**，并**详细信息**显示在对话框中的选项卡。 如果合适，系统还提供了**资源**并**Power**选项卡。

**请注意**  安装程序组件可以定义多个为其设备或设备类的属性页。 每个选项卡应具有用于简明地描述该页面的用途的名称。 系统不会检查以确保名称唯一。

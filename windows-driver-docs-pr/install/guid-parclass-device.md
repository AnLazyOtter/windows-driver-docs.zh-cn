# <a name="guidparclassdevice"></a>GUID_PARCLASS_DEVICE


GUID_PARCLASS_DEVICE 是附加到并行端口设备的设备接口类的已过时标识符。 对于此类的新实例，使用[ **GUID_DEVINTERFACE_PARCLASS** ](guid-devinterface-parclass.md)类标识符。

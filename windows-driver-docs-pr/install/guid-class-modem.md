# <a name="guidclassmodem"></a>GUID_CLASS_MODEM


GUID_CLASS_MODEM 是已过时标识符[设备接口类](https://docs.microsoft.com/windows-hardware/drivers/install/device-interface-classes)的调制解调器设备。 从 Microsoft Windows 2000 开始，使用[ **GUID_DEVINTERFACE_MODEM** ](guid-devinterface-modem.md)此类的新实例的类标识符。

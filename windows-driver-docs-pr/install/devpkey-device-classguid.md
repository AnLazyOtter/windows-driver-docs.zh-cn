<a name="remarks"></a>备注
-------

DEVPKEY_Device_ClassGuid 的值设置由提供的 INF ClassGUID 指令[ **INF 版本部分**](https://docs.microsoft.com/windows-hardware/drivers/install/inf-addservice-directive)安装设备的 INF 文件。

您可以调用[ **SetupDiGetDeviceProperty** ](https://docs.microsoft.com/windows/desktop/api/setupapi/nf-setupapi-setupdigetdevicepropertyw)检索 DEVPKEY_Device_ClassGuid 值。

Windows Server 2003、 Windows XP 和 Windows 2000 支持此属性，但不是支持 DEVPKEY_Device_ClassGuid 属性键。 相反，相应的 SPDRP_CLASSGUID 标识符可用于访问这些早期版本的 Windows 上的属性的值。 有关如何访问这些早期版本的 Windows 上此属性的值的信息，请参阅[访问设备实例 SPDRP_Xxx 属性](https://docs.microsoft.com/windows-hardware/drivers/install/accessing-device-instance-spdrp-xxx-properties)。

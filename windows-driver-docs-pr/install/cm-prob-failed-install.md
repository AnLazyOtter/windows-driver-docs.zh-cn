# <a name="cmprobfailedinstall"></a>CM_PROB_FAILED_INSTALL

此函数保留供系统使用。

未安装设备的驱动程序。

## <a name="error-code"></a>错误代码

28

### <a name="display-message"></a>显示消息

"未安装此设备的驱动程序。 （代码 28）"

### <a name="recommended-resolution"></a>建议的解决方法

请访问设备的制造商的网站，并查找此设备的最新驱动程序。

# <a name="using-the-usb-removable-capability-for-device-container-grouping"></a>将 USB 可移动功能用于设备容器分组


USB**可移动**功能允许操作系统创建旧设备的设备容器。 此机制存在不能提供 Microsoft 操作系统的设备的提供向后兼容性**ContainerID**描述符的设备与计算机不具有 USB 端口功能集成在一起或 (**_UPC**) 为相应的 USB 端口的值。

务必要识别 USB 集线器驱动程序，以报告更准确地使用可用其可移除性信息从物理 USB 硬件**可移动**到其内部的每个连接的设备的功能或面向外部的端口。 有关详细信息，请参阅[从可移动设备功能生成的容器 Id](container-ids-generated-from-the-removable-device-capability.md)。

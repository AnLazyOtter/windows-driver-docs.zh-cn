# <a name="accessing-device-setup-class-properties"></a>访问设备安装程序类属性


在 Windows Vista 和更高版本的 Windows，[统一的设备属性模型](unified-device-property-model--windows-vista-and-later-.md)包括以下：

-   [设备安装程序类属性](https://msdn.microsoft.com/library/windows/hardware/ff542239)对应于 SPCRP_*Xxx*中定义的标识符*Setupapi.h。*

-   属性不具有 SPCRP_*Xxx*标识符，但具有对应的系统定义的注册表项值。

有关如何在 Windows Vista 和更高版本的访问设备安装程序类属性的信息，请参阅[访问设备类属性 （Windows Vista 和更高版本）](accessing-device-class-properties--windows-vista-and-later-.md)。

# <a name="cmprobnotconfigured"></a>CM_PROB_NOT_CONFIGURED

此函数保留供系统使用。

没有设备存在的系统上没有**配置标志**注册表项。 这意味着安装任何驱动程序。 这通常意味着找不到 INF 文件。

## <a name="error-code"></a>错误代码

1

### <a name="display-message"></a>显示消息

"此设备配置不正确。 （代码 1）"

"若要更新此设备的驱动程序，请单击更新驱动程序。 如果这不起作用，请参阅硬件文档以了解详细信息。"

### <a name="recommended-resolution"></a>建议的解决方法

选择**更新驱动程序**，这将启动硬件更新向导。

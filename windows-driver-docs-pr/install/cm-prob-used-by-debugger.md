# <a name="cmprobusedbydebugger"></a>CM_PROB_USED_BY_DEBUGGER

此函数保留供系统使用。

## <a name="error"></a>错误

53

### <a name="display-message-windows-81-and-later-versions-of-windows"></a>显示消息 （Windows 8.1 和更高版本的 Windows）

"此设备具有已保留供 Windows 内核调试程序对此启动会话的持续时间。 （代码 53）"

### <a name="recommended-resolution-windows-81-and-later-versions-of-windows"></a>建议的解决方法 （Windows 8.1 和更高版本的 Windows）

禁用 Windows 内核调试以允许设备正常启动。

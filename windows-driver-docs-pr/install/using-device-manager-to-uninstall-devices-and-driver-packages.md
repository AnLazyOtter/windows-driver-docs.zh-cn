# <a name="using-device-manager-to-uninstall-devices-and-driver-packages"></a>使用设备管理器卸载设备和驱动程序包


[设备管理器](using-device-manager.md)是允许你管理设备和驱动程序在系统上的控制面板应用程序。 您可以使用设备管理器卸载设备和驱动程序包，有一些限制。 通常情况下，最终用户不使用设备管理器来卸载设备或驱动程序包，但可能需要使用设备管理器对其计算机可能会遇到的问题进行故障排除。

**请注意**  必须具有管理员特权才能卸载设备或[驱动程序包](driver-packages.md)使用设备管理器。

 

有关卸载驱动程序和驱动程序包的详细信息，请参阅[卸载如何设备和驱动程序包](how-devices-and-driver-packages-are-uninstalled.md)。

### <a href="" id="uninstalling-the-device"></a> 卸载设备

若要使用设备管理器卸载设备，请按照下列步骤：

1.  启动控制面板中，单击**硬件和声音**，然后单击**设备管理器**。

2.  展开节点，表示要卸载，请右键单击该设备条目，并单击设备的类型**卸载**。

3.  上**确认设备删除**对话框中，单击**确定**开始卸载过程。

4.  卸载过程完成后，删除该设备。

**请注意**  对于某些设备，你可能必须重新启动系统才能完成卸载过程。 设备可能会继续之前重新启动系统正常运行。

 

### <a href="" id="deleting-a-driver-package-from-the-driver-store"></a> 从驱动程序存储区中删除驱动程序包

上**确认设备删除**对话框中，单击**删除此设备的驱动程序软件**选项可以删除的设备使用的驱动程序包。

### <a href="" id="deleting-the-binary-files-of-the-installed-driver"></a> 删除已安装的驱动程序的二进制文件

设备管理器不能用于执行此操作。

# <a name="writeoncediskclassguid"></a>WriteOnceDiskClassGuid


WriteOnceDiskClassGuid 是已过时标识符[设备接口类](https://msdn.microsoft.com/library/windows/hardware/ff541339)以进行写入-一次磁盘设备。 从 Microsoft Windows 2000 开始，使用[ **GUID_DEVINTERFACE_WRITEONCEDISK** ](guid-devinterface-writeoncedisk.md)此类的新实例的类标识符。

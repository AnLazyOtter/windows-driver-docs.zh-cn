# <a name="cmwaitnopendinginstallevents"></a>CM_WaitNoPendingInstallEvents

此函数保留供系统使用。

**CM_WaitNoPendingInstallEvents**中定义*Cfgmgr32.h* ，如下所示：

```c
#define CM_WaitNoPendingInstallEvents CMP_WaitNoPendingInstallEvents
```

换而言之， **CM_WaitNoPendingInstallEvents**是另一种名称[ **CMP_WaitNoPendingInstallEvents**](https://docs.microsoft.com/windows/desktop/api/cfgmgr32/nf-cfgmgr32-cm_waitnopendinginstallevents)。 由于没有实际函数名为**CM_WaitNoPendingInstallEvents**， **GetProcAddress**将返回**NULL**。

# <a name="sample-device-installation-files"></a>示例设备安装文件





Windows Driver Kit (WDK) 包括各种示例安装文件，如 INF 文件、 设备安装应用程序、 类安装程序、 共同安装程序和属性页提供程序。

大多数示例安装文件位于*src\\安装程序*WDK 的子目录。 另请参阅*src\\常规\\toaster* WDK 的子目录。

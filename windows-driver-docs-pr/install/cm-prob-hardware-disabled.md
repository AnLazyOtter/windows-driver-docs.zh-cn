# <a name="cmprobhardwaredisabled"></a>CM_PROB_HARDWARE_DISABLED

此函数保留供系统使用。

设备已被禁用。

## <a name="error-code"></a>错误代码

29

### <a name="display-message"></a>显示消息

"因为设备的固件没有提供所需的资源，此设备已被禁用。 （代码 29）"

### <a name="recommended-resolution"></a>建议的解决方法

启用在 BIOS 中的该设备。

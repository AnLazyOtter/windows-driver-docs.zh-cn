# <a name="devpkeydeviceinterfacerestricted"></a>DEVPKEY_DeviceInterface_Restricted


DEVPKEY_DeviceInterface_Restricted 设备接口属性指示设备接口在其已存在并设置为 TRUE 时，应遵循设置的系统组件视为具有特权访问权限。

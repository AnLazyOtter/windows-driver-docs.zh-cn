# <a name="cmprobdisabled"></a>CM_PROB_DISABLED

此函数保留供系统使用。

设备已被禁用。

## <a name="error-code"></a>错误代码

22

### <a name="display-message"></a>显示消息

"禁用此设备。 （代码 22）"

### <a name="recommended-resolution"></a>建议的解决方法

设备已被禁用，因为它使用设备管理器禁用该用户。 选择**启用设备**，这将使该设备。

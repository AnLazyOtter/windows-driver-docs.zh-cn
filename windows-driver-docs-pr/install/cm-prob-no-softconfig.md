# <a name="cmprobnosoftconfig"></a>CM_PROB_NO_SOFTCONFIG

此函数保留供系统使用。

设备要求强制的配置。

### <a name="error-code"></a>错误代码

34

### <a name="display-message"></a>显示消息

"Windows 无法确定此设备的设置。 请参考这个设备随附的文档，并使用资源选项卡的配置设置。 （代码 34）"

### <a name="recommended-resolution"></a>建议的解决方法

更改硬件设置 （通过设置跨线或运行供应商提供的实用程序），然后使用设备管理器**资源**选项卡以设置强制的配置。

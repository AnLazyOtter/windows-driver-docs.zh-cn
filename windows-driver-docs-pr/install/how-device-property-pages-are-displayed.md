# <a name="how-device-property-pages-are-displayed"></a>设备属性页的显示方式


设备属性页将显示在以下方面：

-   设备管理器显示的属性页的设备的属性将显示任何时间。

-   从 Windows 7 开始**设备和打印机**显示属性页任何时间在用户右键单击设备。 若要显示在属性页**设备和打印机**，执行以下操作：

    -   右键单击中的图标**设备和打印机**表示连接到系统的设备。

    -   单击“属性” 。

    -   上**属性**窗口中，单击**硬件**选项卡，然后依次**属性**。

# <a name="installing-an-inbox-driver-or-a-preinstalled-driver"></a>安装收件箱驱动程序或预安装的驱动程序


如果存在收件箱驱动程序或[预安装的驱动程序](preinstalling-driver-packages.md)匹配设备时，Windows 会自动安装到设备的最佳匹配项的驱动程序。 Windows 不会提示用户插入分发介质。 此外，Windows 不会尝试访问 Windows 更新，以确定是否存在更好的匹配比计算机已安装的驱动程序的 Windows 更新驱动程序。

有关如何驱动程序包位于在详细信息[硬件第一个安装](hardware-first-installation.md)，请参阅[其中 Windows 搜索驱动程序](where-setup-searches-for-drivers.md)。

有关 Windows 如何选择具有最佳匹配到设备的驱动程序的信息，请参阅[Windows 中如何选择驱动程序](how-setup-selects-drivers.md)。

# <a name="cmprobfailedadd"></a>CM_PROB_FAILED_ADD

此函数保留供系统使用。

若要添加设备的驱动程序的尝试失败。

## <a name="error-code"></a>错误代码

31

### <a name="display-message"></a>显示消息

"此设备不正常工作由于 Windows 无法加载这个设备所需的驱动程序。 （代码 31）"

### <a name="recommended-resolution"></a>建议的解决方法

更新设备驱动程序。

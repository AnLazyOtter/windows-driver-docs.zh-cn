# <a name="cmprobneedrestart"></a>CM_PROB_NEED_RESTART

此函数保留供系统使用。

必须重新启动系统。

## <a name="error-code"></a>错误代码

14

### <a name="display-message"></a>显示消息

"此设备无法正常工作之前重新启动计算机。 （代码 14）

"立即重新启动计算机，单击重新启动计算机。"

### <a name="recommended-resolution"></a>建议的解决方法

选择**重新启动计算机**，这将重新启动您的计算机。

# <a name="cmprobsetpropertiesfailed"></a>CM_PROB_SETPROPERTIES_FAILED

此函数保留供系统使用。

无法设置设备属性。

## <a name="error-code"></a>错误代码

50

### <a name="display-message"></a>显示消息

"Windows 无法应用的所有此设备的属性。 设备属性可以包含描述设备的功能和设置 （如示例中的安全设置） 的信息。 （代码 50）"

"若要解决此问题，你可以尝试重新安装此设备。 但是，我们建议你为新的驱动程序与硬件制造商联系。"

### <a name="recommended-resolution"></a>建议的解决方法

请尝试重新安装该设备。 如果这不起作用，请从硬件供应商处获取新的驱动程序。

<a name="remarks"></a>备注
-------

DEVPKEY_DEVICE_CompatibleIds 的值将由*兼容 id*条目为中的设备提供的值[ **INF 模型部分**](https://docs.microsoft.com/windows-hardware/drivers/install/inf-models-section)的 INF 文件安装设备。

您可以调用[ **SetupDiGetDeviceProperty** ](https://docs.microsoft.com/windows/desktop/api/setupapi/nf-setupapi-setupdigetdevicepropertyw)检索 DEVPKEY_DEVICE_CompatibleIds 值。

Windows Server 2003、 Windows XP 和 Windows 2000 支持此属性，但不是支持 DEVPKEY_Device_CompatibleIds 属性键。 相反，相应的 SPDRP_COMPATIBLEIDS 标识符可用于访问这些早期版本的 Windows 上的属性的值。 有关如何访问这些早期版本的 Windows 上此属性的值的信息，请参阅[访问设备实例 SPDRP_Xxx 属性](https://docs.microsoft.com/windows-hardware/drivers/install/accessing-device-instance-spdrp-xxx-properties)。

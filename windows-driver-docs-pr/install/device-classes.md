# <a name="device-classes"></a>设备类





本部分提供有关用于将设备和驱动程序放入组的成员具有相似特征的两个设备类的信息：

<a href="" id="device-setup-classes"></a>[设备安装程序类](device-setup-classes.md)  
用于映射到组的设备组合在一起安装和配置类似的方式。

<a href="" id="device-interface-classes"></a>[设备接口类](device-interface-classes.md)  
用于对设备进行分组在一起，提供类似的功能。

由于在这两种类型的设备类的指定和使用的方式的相似之处，本部分提供了标题为主题[vs 安装程序类。接口类](setup-classes-versus-interface-classes.md)。

[SetupAPI](setupapi.md)函数使用[设备信息设置](device-information-sets.md)公开给用户模式代码类的成员。

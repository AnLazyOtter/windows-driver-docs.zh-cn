# <a name="softwareinfo-xml-document"></a>SoftwareInfo XML 文档


Windows 8 或更高版本，元数据的包可能包含一个 Softwareinfo.xml 文件，其中包含用于下载与设备相关联的应用和应用程序让特许访问权限的信息。

Softwareinfo.xml 文件中的数据的格式根据[ **SoftwareInfo XML 架构**](https://msdn.microsoft.com/library/windows/hardware/jj159075)。

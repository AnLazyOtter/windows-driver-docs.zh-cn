# <a name="saved-settings"></a>已保存的设置





当保存当前游戏杆设置，REGSTR\_VAL\_JOYNCONFIG 保存在 REGSTR 下\_密钥\_JOYCURR 密钥也写下 REGSTR\_密钥\_JOYSETTINGS 密钥在具有相同名称作为从中获取的 OEM 定义的设置的子项 （非 OEM 设置保存在一个子项"predef"以及类型数目）。 当游戏杆被替换时，保持已保存的设置，以便还原游戏杆，如果已保存的设置被放回到当前设置。 只能通过控制面板将使用这些注册表值。

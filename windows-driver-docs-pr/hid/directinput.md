# <a name="directinput"></a>DirectInput





Microsoft DirectInput 文档介绍如何编写 Microsoft DirectX 版本 1.0、 2.0、 3.0、 5.0、 6.0、 7.0、 7.0 a 和 8.0 的输入组件的驱动程序。 DirectX 版本 1.0 和 2.0 版中包含相同的游戏杆支持作为原始 Microsoft Windows 95 中，虽然 DirectX 3.0 添加鼠标和键盘访问的组件对象模型 (COM) 接口，并改进了某些游戏杆支持而无需实现更改的驱动程序模型。 5.0 及更高版本的 DirectX 游戏杆配置信息添加一个 COM 接口的访问游戏杆，强制反馈和简化的访问。

本文档包括以下主题：

[游戏杆支持](joystick-support.md)

[力反馈设备驱动程序接口](force-feedback-device-driver-interface.md)

[扩展 DirectInput 游戏控制器控制面板](extending-the-directinput-game-controller-control-panel.md)

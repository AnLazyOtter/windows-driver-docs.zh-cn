# <a name="the-windows-portable-devices-files"></a>Windows 便携设备文件


WpdHelloWorldDriver 项目包含以下 Windows 便携设备文件。

| Filename                | 描述                                                                                                                                                                        |
|-------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| WpdBaseDriver.cpp       | 包含 WpdBaseDriver 成员函数的实现。                                                                                                                  |
| WpdBaseDriver.h         | 包含 WpdBaseDriver 类的定义。 此类包含的成员函数可将消息调度、 初始化驱动程序对象和其他操作。                     |
| WpdCapabilities.cpp     | 包含 WpdCapabilities 成员函数的实现。                                                                                                                |
| WpdCapabilities.h       | 包含 WpdCapabilities 类的定义。 此类包含用于检索事件处理程序： 支持的命令、 命令选项，函数类别和等等。 |
| WpdObjectEnum.cpp       | 包含 WpdObjectEnumerator 类的实现。                                                                                                                       |
| WpdObjectEnum.h         | 包含 WpdObjectEnumerator 类的定义。 此类包含枚举支持的设备对象的成员函数。                                |
| WpdObjectProperties.cpp | 包含 WpdObjectProperties 类的实现。                                                                                                                       |
| WpdObjectProperties.h   | 包含 WpdObjectProperties 类的定义。 此类包含设置和检索支持的设备属性的成员函数。                      |
| WpdObjectResources.cpp  | 包含 WpdObjectResources 成员函数的实现。                                                                                                             |
| WpdObjectResources.h    | 包含 WpdObjectResources 类的定义。 此类包含驱动程序支持的资源中检索的成员函数。                              |

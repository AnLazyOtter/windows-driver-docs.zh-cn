# <a name="wpd-drivers"></a>WPD 驱动程序


Microsoft Windows 便携式设备 (WPD) 使计算机能够与连接的媒体和存储设备进行通信。 此系统取代了 Windows 媒体设备管理器 (WMDM) 和 Windows 图像采集 (WIA) 通过提供计算机的灵活、 可靠的方式与音乐播放机、 存储设备、 移动电话和许多其他类型的连接的设备进行通信。

Microsoft 提供了标准协议和设备，包括图片传输协议 (PTP)、 媒体传输协议 (MTP) 设备和大容量存储类 (MSC) 设备的多个驱动程序。 如果你的设备支持的唯一协议，可能需要开发您自己的驱动程序。 使用用户模式驱动程序框架 (UMDF) 编写此驱动程序。 有关此框架的详细信息，请参阅[入门 UMDF](https://docs.microsoft.com/windows-hardware/drivers/wdf/getting-started-with-umdf-version-2)。

有关为 Windows 便携设备编写的应用程序的详细信息，请参阅[WPD SDK 文档](https://go.microsoft.com/fwlink/p/?linkid=178695)。

有关 WPD 设备驱动程序开发和 WPD 应用程序开发的详细信息，请参阅[WPD 博客](https://go.microsoft.com/fwlink/p/?linkid=178687)，这是适用于 Windows 10 准确。

# <a name="wpd-programming-guide"></a>WPD 编程指南


本编程指南介绍了几个实现 WPD DDI 的 WPD 驱动程序示例。 第一个示例，WpdHelloWorldDriver，是一个简单的驱动程序，但功能受限演示基本概念。 这是类似于许多编程文本中显示和是建议的初学者示例学习如何编写 WPD 驱动程序的"Hello World"应用程序。

后续的驱动程序示例演示了附加功能，例如与硬件进行交互，实现设备服务和支持多个传输协议。

[高级任务](advanced-tasks.md)部分介绍了通过 WPD 设备驱动程序的高级的操作。 大部分信息和相应的代码示例摘自名为 WpdWudfSampleDriver 的示例驱动程序。

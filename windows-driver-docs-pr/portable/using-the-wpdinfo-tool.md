# <a name="using-the-wpdinfo-tool"></a>使用 WpdInfo 工具


*WpdInfo.exe*工具是基于 Windows 的应用程序，可用来驱动程序中测试的功能。 此工具可用于执行以下任务：

-   打开或关闭设备
-   创建或删除设备上的对象
-   创建、 保存或删除的对象资源
-   向设备发送命令
-   查看受支持的设备命令
-   查看受支持的设备的内容类型
-   查看支持的事件
-   查看支持的属性
-   查看给定资源的属性
-   查看某一给定属性的属性
-   执行大容量属性操作
-   打开或关闭设备服务
-   设备服务上调用方法

在启动工具，它会提示您选择一个可用的 WPD 设备。

![选择 wpdinfo 设备](images/wpdinfo1.png)

选择一台设备后，建立的连接并初始化 WpdInfo 应用程序窗口。 主窗口的左上方帧显示支持的对象的层次结构。 右上方框架显示与当前所选对象相关联的资源数据。 中部下方窗格中显示对象支持的属性。 在窗口的底部窗格显示在给定的操作期间由驱动程序返回任何错误信息。

![wpdinfo 默认屏幕](images/wpdinfo2.png)

您可以右键单击给定的对象、 资源或要显示的菜单选项适用于所选元素的属性。 例如，可以右键单击以将其删除，对其资源执行操作，查看关联的属性等的左上方窗格中显示的对象。

![wpdinfo 选项](images/wpdinfo3.png)

# <a name="wpd-and-earlier-technologies"></a>WPD 和早期技术


WPD 包含数字静态相机和可移植数字媒体播放器的许多增强功能。 WPD 是首选的方法来访问这些设备在 Windows XP 及更高版本。 现有 Windows 图像采集 (WIA) 和 Windows 媒体设备管理器 (WMDM) 中 Windows Api 将继续支持和兼容性层将提供与 WPD 的兼容性。 这些兼容性层确保 WPD 设备可以以透明方式向公开现有 WIA 和 WMDM 应用程序作为 WIA 的设备或 WMDM 服务提供商分别。 但是，这些应用程序可能具有相同范围的功能，就像与新的 WPD API 和新的 MTP 设备。

WPD MTP 驱动程序还支持图片传输协议 (PTP) 设备。 PTP WIA 中支持的设备将继续通过兼容性层 WIA 应用程序中的函数。 Microsoft 仍致力于确保应用程序和数字仍照相机中的兼容性的下一版本 Windows 和这些新技术旨在补充并增强在 Windows 中的数码照相机体验。

# <a name="the-wpdwudfsampledriver-sample"></a>WpdWudfSampleDriver 示例


WPD 驱动程序文档的此部分介绍了一个全面的示例驱动程序，WpdWudfSampleDriver，包括使用 Windows Driver Kit (WDK)。

全面 WPD 示例驱动程序 (WpdWudfSampleDriver) 演示了 Microsoft Windows 可移植 Devides (WPD) 设备驱动程序接口 (DDI) 的几乎所有方面。 此驱动程序旨在作为正常用户模式驱动程序框架 (UMDF) 驱动程序，还进程 WPD 设置的命令。 尽管此驱动程序不与实际硬件交互，它模拟与设备通信支持电话联系人、 图片、 音乐和视频。

演示概念的最简单方法是编写此驱动程序。 因此，示例驱动程序可能会执行操作或效率低下到生产环境的驱动程序的方式结构化。 此外，此示例不使用实际硬件。 相反，它通过使用在内存中数据结构来模拟的设备。 因此，该驱动程序可能并不现实的生产硬件的方式实现。

某些任务完成的 WpdWudfSampleDriver 专为高级 Windows 便携式设备 (WPD) 驱动程序开发人员。 在本部分后面的主题中描述了这些任务。

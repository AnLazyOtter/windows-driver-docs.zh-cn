# <a name="docking-versus-laptop-slate-conversion"></a>扩展连接与笔记本电脑平板电脑转换


本主题介绍为指示器便携式计算机的静态转换和停靠操作之间的区别。

## <a name="span-idlaptopslateconversionspanspan-idlaptopslateconversionspanspan-idlaptopslateconversionspanlaptop-slate-conversion"></a><span id="Laptop_slate_conversion"></span><span id="laptop_slate_conversion"></span><span id="LAPTOP_SLATE_CONVERSION"></span>便携式计算机盖板转换


便携式计算机盖板转换是用户将可移植 （不以物理方式连接） 的系统时更改键盘可用性的用户操作。 此类操作的示例如下所示：

-   附加可移植键盘附件 (与我们没有电池包)。
-   翻转、 旋转，或需要考虑的是永久连接的键盘。

## <a name="span-iddockingspanspan-iddockingspanspan-iddockingspandocking"></a><span id="Docking"></span><span id="docking"></span><span id="DOCKING"></span>停靠


停靠被视为可附件附加停靠系统可移植到固定且 （非可移植） 的用户执行任何操作。 此类操作的示例如下所示：

-   将附加到端口复制器。
-   将附加到已经集成了键盘保持静止停靠附件。 在这种情况下，停靠的操作和便携式计算机/盖板转换发生，因此必须实现两个指示器。

**请注意**   GPIO 停靠指示器实施的范围不在基于 USB 停靠附件。

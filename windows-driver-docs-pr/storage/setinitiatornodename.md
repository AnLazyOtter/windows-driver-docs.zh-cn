# <a name="setinitiatornodename"></a>SetInitiatorNodeName


**SetInitiatorNodeName**方法将通知发起程序名称已更改的发起程序。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**SetInitiatorNodeName**方法，请参阅成员的说明[ **SetInitiatorNodeName\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_setinitiatornodename_in)和[ **SetInitiatorNodeName\_OUT** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_setinitiatornodename_out)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类不支持所需**SetInitiatorNodeName**。

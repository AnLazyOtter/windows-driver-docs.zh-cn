# <a name="scsireportluns"></a>ScsiReportLuns


**ScsiReportLuns**方法发布一个 SCSI 报表 Lun 命令时到目标上的逻辑单元，然后返回结果。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**ScsiReportLuns**方法，请参阅成员的说明[ **ScsiReportLuns\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_scsireportluns_in)和[ **ScsiReportLuns\_出**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_scsireportluns_out)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类必须支持**ScsiReportLuns**。

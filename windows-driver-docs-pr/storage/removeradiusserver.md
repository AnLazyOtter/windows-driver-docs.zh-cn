# <a name="removeradiusserver"></a>RemoveRADIUSServer


**RemoveRADIUSServer**方法从监督发起方和目标之间的身份验证协商的 RADIUS 服务器的列表中移除的 RADIUS 服务器。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**RemoveRADIUSServer**方法，请参阅成员的说明[ **RemoveRADIUSServer\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_removeradiusserver_in)和[**RemoveRADIUSServer\_出**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_removeradiusserver_out)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类不支持所需**RemoveRADIUSServer**。

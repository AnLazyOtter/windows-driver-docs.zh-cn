# <a name="sendtargets"></a>SendTargets


**SendTargets**方法在现有发现会话上发出 SCSI 报表目标命令。

此 WMI 方法属于未发布的[MSiSCSI\_操作 WMI 类](msiscsi-operations-wmi-class.md)。 有关**SendTargets**方法的参数的说明，请参阅[**SendTargets\_IN**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_sendtargets_in)和[**SendTargets\_OUT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_sendtargets_out)结构的成员说明。

实现 MSiSCSI\_操作 WMI 类的微型端口驱动程序不需要支持**SendTargets**。

# <a name="sendtargets"></a>SendTargets


**SendTargets**方法上现有的发现会话发出一条 SCSI 报表目标命令。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**SendTargets**方法，请参阅成员的说明[ **SendTargets\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_sendtargets_in)和[ **SendTargets\_出**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_sendtargets_out)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类不支持所需**SendTargets**。

# <a name="sendtargets"></a>SendTargets


**SendTargets**方法上现有的发现会话发出一条 SCSI 报表目标命令。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**SendTargets**方法，请参阅成员的说明[ **SendTargets\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff565533)和[ **SendTargets\_出**](https://msdn.microsoft.com/library/windows/hardware/ff565540)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类不支持所需**SendTargets**。

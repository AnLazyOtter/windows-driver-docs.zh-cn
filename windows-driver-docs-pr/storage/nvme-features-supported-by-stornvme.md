# <a name="nvme-features-supported-by-stornvme"></a>StorNVMe 支持的 NVMe 功能

**StorNVMe**是系统提供的驱动程序，可提供对高速 NVMe 设备的访问。 它在 Windows Windows 8.1 （和 Windows Server 2012 R2）及更高版本中可用。 此页将提供**StorNVMe**提供的 NVMe 功能支持的相关信息。

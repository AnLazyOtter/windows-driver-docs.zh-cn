# <a name="removepersistentlogin"></a>RemovePersistentLogin


**RemovePersistentLogin**方法从永久登录的发起方的列表中删除目标。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**RemovePersistentLogin**方法，请参阅成员的说明[ **RemovePersistentLogin\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff564003)和[ **RemovePersistentLogin\_OUT** ](https://msdn.microsoft.com/library/windows/hardware/ff564008)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类必须支持**RemovePersistentLogin**。

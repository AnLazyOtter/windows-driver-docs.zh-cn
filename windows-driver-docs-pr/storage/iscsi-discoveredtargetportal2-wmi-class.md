# <a name="iscsidiscoveredtargetportal2-wmi-class"></a>ISCSI\_DiscoveredTargetPortal2 WMI 类


ISCSI\_DiscoveredTargetPortal2 WMI 类提供与已发现的目标门户相关联的信息。 除了目标门户的网络地址，此类包含有关目标门户的安全特征的信息。

ISCSI\_DiscoveredTargetPortal2 是类似于[ISCSI\_DiscoveredTargetPortal WMI 类](iscsi-discoveredtargetportal-wmi-class.md)，但 ISCSI\_DiscoveredTargetPortal2 包含目标门户信息的 ISCSI\_不具有 DiscoveredTargetPortal WMI 类，如门户组的标记号。

ISCSI\_DiscoveredTargetPortal2 WMI 类是由提供的信息的超集[ISCSI\_TargetPortal WMI 类](iscsi-targetportal-wmi-class.md)，用于定义仅为目标门户地址。

ISCSI\_DiscoveredTargetPortal2 WMI 类是一个未发布的类中定义的*Discover.mof*。

当 WMI 工具套件编译此类定义时，它会生成[ **ISCSI\_DiscoveredTargetPortal2** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsifnd/ns-iscsifnd-_iscsi_discoveredtargetportal2)数据结构

# <a name="logoutfromtarget"></a>LogoutFromTarget


**LogoutFromTarget**方法指示管理 iSCSI 发起程序目标中注销的 HBA，并从本地计算机的设备堆栈中删除目标的逻辑单元的微型端口驱动程序。

实现的微型端口驱动程序[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)不需要支持此方法。

MSiSCSI\_操作 WMI 类是未发布。 有关参数的说明**LogoutFromTarget**方法，请参阅成员的说明[ **LogoutFromTarget\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff562191)和[ **LogoutFromTarget\_出**](https://msdn.microsoft.com/library/windows/hardware/ff562194)结构。

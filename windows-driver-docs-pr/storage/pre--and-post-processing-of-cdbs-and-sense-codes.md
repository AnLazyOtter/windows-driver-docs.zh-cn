# <a name="pre--and-post-processing-of-cdbs-and-sense-codes"></a>预处理和后处理的 Cdb 和探测代码


Storport 不会验证、 拒绝或以任何方式通过它传递的 SCSI 命令描述符块 (Cdb) 转换。 同样，也还没有筛选或转换的探测代码或返回的响应命令失败，并检查条件状态中的设备接收其他检测数据。

# <a name="iscsidiscoveredtarget2-wmi-class"></a>ISCSI\_DiscoveredTarget2 WMI 类


ISCSI\_DiscoveredTarget2 WMI 类提供与搜索到的目标相关的信息。

此类是类似于[ISCSI\_DiscoveredTarget WMI 类](iscsi-discoveredtarget-wmi-class.md)，但 ISCSI\_DiscoveredTarget2 包含目标门户信息的 ISCSI\_DiscoveredTarget WMI 类不有，如门户组的标记号。

ISCSI\_DiscoveredTarget2 类中定义，则不发布*Discover.mof*。

当 WMI 工具套件编译此类定义时，它会生成[ **ISCSI\_DiscoveredTarget2** ](https://msdn.microsoft.com/library/windows/hardware/ff561503)数据结构。

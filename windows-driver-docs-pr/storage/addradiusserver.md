# <a name="addradiusserver"></a>AddRADIUSServer


**AddRADIUSServer**方法将 RADIUS 服务器添加到发起方使用建立会话包含目标 RADIUS 服务器的列表。 HBA 发起程序应在非易失性内存中存储的 RADIUS 服务器的列表。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**AddRADIUSServer**方法，请参阅成员的说明[ **AddRADIUSServer\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_addradiusserver_in)和[ **AddRADIUSServer\_出**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_addradiusserver_out)结构。

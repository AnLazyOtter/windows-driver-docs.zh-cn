# <a name="msiscsibootinformation-wmi-class"></a>MSiSCSI\_BootInformation WMI 类


MSiSCSI\_BootInformationWMI 类公开有关包含的启动设备的节点的信息。

此类是未发布。 有关信息的说明，MSiSCSI\_BootInformation WMI 类报表，请参阅成员的说明[ **MSiSCSI\_BootInformation** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_msiscsi_bootinformation)结构。

微型端口驱动程序实现 MSiSCSI\_BootInformation WMI 类必须注册使用的物理设备对象 (PDO) 实例名称的类。

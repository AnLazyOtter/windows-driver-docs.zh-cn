# <a name="msiscsi_bootinformation-wmi-class"></a>MSiSCSI\_BootInformation WMI 类


MSiSCSI\_BootInformationWMI 类公开有关包含启动设备的节点的信息。

此类未发布。 有关 MSiSCSI\_BootInformation WMI 类报告的信息的说明，请参阅[**MSiSCSI\_BootInformation**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_msiscsi_bootinformation)结构的成员说明。

实现 MSiSCSI\_BootInformation WMI 类的微型端口驱动程序必须使用物理设备对象（PDO）实例名称注册该类。

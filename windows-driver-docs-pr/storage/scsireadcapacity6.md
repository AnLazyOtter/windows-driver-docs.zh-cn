# <a name="scsireadcapacity"></a>ScsiReadCapacity


**ScsiReadCapacity**方法指示的微型端口驱动程序管理 iSCSI 发起程序进行登录到目标并颁发读取容量命令目标上的逻辑单元到 SCSI 的 HBA，然后将结果返回。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**ScsiReadCapacity**方法，请参阅成员的说明[ **ScsiReadCapacity\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff564897)和[ **ScsiReadCapacity\_出**](https://msdn.microsoft.com/library/windows/hardware/ff564906)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类必须支持**ScsiReadCapacity**。

# <a name="clearallmpiodiskhealthcounters-function"></a>ClearAllMpioDiskHealthCounters 函数


ClearAllMpioDiskHealthCounters 方法用于清除所有收集到的 MPIO 多路径磁盘统计信息。

<a name="syntax"></a>语法
------

```ManagedCPlusPlus
unknown ClearAllMpioDiskHealthCounters(void);
```

<a name="parameters"></a>参数
----------

此函数没有任何参数。

<a name="return-value"></a>返回值
------------

无

<a name="remarks"></a>备注
-------

此 WMI 方法属于 MPIO\_WMI\_方法 WMI 类。

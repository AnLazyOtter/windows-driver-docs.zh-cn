# <a name="setradiussharedsecret"></a>SetRADIUSSharedSecret


**SetRADIUSSharedSecret**方法设置发起方将使用 RADIUS 服务器进行自身身份验证的共享的机密。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**SetRADIUSSharedSecret**方法，请参阅成员的说明[ **SetRADIUSSharedSecret\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_setradiussharedsecret_in)和[ **SetRADIUSSharedSecret\_OUT** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_setradiussharedsecret_out)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类必须支持**SetRADIUSSharedSecret**。

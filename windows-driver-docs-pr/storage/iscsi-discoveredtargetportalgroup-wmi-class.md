# <a name="iscsi_discoveredtargetportalgroup-wmi-class"></a>ISCSI\_DiscoveredTargetPortalGroup WMI 类


## <span id="ddk_iscsi_discoveredtargetportalgroup_wmi_class_kr"></span><span id="DDK_ISCSI_DISCOVEREDTARGETPORTALGROUP_WMI_CLASS_KR"></span>


ISCSI\_DiscoveredTargetPortalGroup WMI 类包含有关门户组中所有门户的信息。

此类已取消发布，并在*发现*中定义。

当 WMI 工具套件编译此类定义时，它将生成[**ISCSI\_DiscoveredTargetPortalGroup**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_iscsi_discoveredtargetportalgroup)数据结构。

# <a name="iscsipersistentlogin-wmi-class"></a>ISCSI\_持久\_登录 WMI 类


## <span id="ddk_iscsi_persistent_login_wmi_class_kr"></span><span id="DDK_ISCSI_PERSISTENT_LOGIN_WMI_CLASS_KR"></span>


ISCSI\_的永久\_登录 WMI 类定义一个持久的登录名。 自动管理 iSCSI 发起程序的微型端口驱动程序建立持久的登录连接，就立即加载到存储驱动程序堆栈。 此时间保证的目标对发起方为其保留永久性登录将可供尽可能在启动过程中尽早系统。

ISCSI\_的永久\_Login 类中定义，则不发布*Operations.mof*。

当 WMI 工具套件编译此类定义时，它会生成[ **ISCSI\_的永久\_登录名**](https://msdn.microsoft.com/library/windows/hardware/ff561553)数据结构。

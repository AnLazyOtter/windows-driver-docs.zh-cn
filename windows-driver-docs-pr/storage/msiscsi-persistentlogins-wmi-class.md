# <a name="msiscsi_persistentlogins-wmi-class"></a>MSiSCSI\_PersistentLogins WMI 类


## <span id="ddk_msiscsi_persistentlogins_wmi_class_kr"></span><span id="DDK_MSISCSI_PERSISTENTLOGINS_WMI_CLASS_KR"></span>


MSiSCSI\_PersistentLogins WMI 类报告指定发起方实例的持久目标登录会话的列表。 与每个特定持久性登录会话关联的信息由[ISCSI\_持久\_登录 WMI 类](iscsi-persistent-login-wmi-class.md)定义。

管理 HBA 发起程序的微型端口驱动程序需要实现 MSiSCSI\_PersistentLogins WMI 类。

MSiSCSI\_PersistentLogins 类未发布，并在*操作*中定义。

当 WMI 工具套件编译此类定义时，它会生成[**MSiSCSI\_PersistentLogins**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_msiscsi_persistentlogins)数据结构。

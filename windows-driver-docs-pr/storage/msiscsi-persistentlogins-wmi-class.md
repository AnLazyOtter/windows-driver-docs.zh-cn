# <a name="msiscsipersistentlogins-wmi-class"></a>MSiSCSI\_PersistentLogins WMI 类


## <span id="ddk_msiscsi_persistentlogins_wmi_class_kr"></span><span id="DDK_MSISCSI_PERSISTENTLOGINS_WMI_CLASS_KR"></span>


MSiSCSI\_PersistentLogins WMI 类报告指示发起方实例的持久的目标登录会话的列表。 定义与每个特定的永久性登录会话关联的信息[ISCSI\_的永久\_登录名 WMI 类](iscsi-persistent-login-wmi-class.md)。

管理 HBA 发起程序的微型端口驱动程序所需实现 MSiSCSI\_PersistentLogins WMI 类。

MSiSCSI\_PersistentLogins 类中定义，则不发布*Operations.mof*。

当 WMI 工具套件编译此类定义时，它会生成[ **MSiSCSI\_PersistentLogins** ](https://msdn.microsoft.com/library/windows/hardware/ff563093)数据结构。

# <a name="clearcache"></a>ClearCache


**ClearCache**方法指示发起方 HBA 清除 iSCSI 身份验证和预共享密钥缓存。 此方法*不*应清除任何其他持久性信息。 具体而言，它不应删除永久登录。

**ClearCache**属于未发布的[MSISCSI\_SecurityConfigOperations WMI 类](msiscsi-securityconfigoperations-wmi-class.md)。 有关**ClearCache**方法的参数的说明，请参阅[**ClearCache\_OUT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_clearcache_out)结构的成员说明。

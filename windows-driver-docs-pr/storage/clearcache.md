# <a name="clearcache"></a>ClearCache


**ClearCache**方法指示发起方 HBA 清除 iSCSI 身份验证和预共享密钥的缓存。 此方法应*不*清除任何其他持久性信息。 具体而言，它不应删除永久登录。

**ClearCache**属于未发布[MSiSCSI\_SecurityConfigOperations WMI 类](msiscsi-securityconfigoperations-wmi-class.md)。 有关参数的说明**ClearCache**方法，请参阅成员的说明[ **ClearCache\_OUT** ](https://msdn.microsoft.com/library/windows/hardware/ff552447)结构。

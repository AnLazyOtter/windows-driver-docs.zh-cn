# <a name="problem-description"></a>问题说明


在 Windows 计算机系统、 故障转储和休眠状态中具有特殊的存储驱动程序、 端口和微型端口支持崩溃转储驱动程序提供支持。 当系统崩溃时，转储数据写入磁盘而不典型的存储堆栈中使用这些特殊的驱动程序集。 这些驱动程序而无需使用文件系统卷或卷管理器驱动程序写入数据，并且它们不以前支持崩溃转储堆栈中添加的任何驱动程序。

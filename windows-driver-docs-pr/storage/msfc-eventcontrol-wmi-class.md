# <a name="msfceventcontrol-wmi-class"></a>MSFC\_将 EventControl WMI 类


## <span id="ddk_msfc_eventcontrol_wmi_class_kr"></span><span id="DDK_MSFC_EVENTCONTROL_WMI_CLASS_KR"></span>


MSFC\_将 EventControl 类定义允许 WMI 客户端控件的链接、 端口和目标的事件传递的 WMI 方法。

此 WMI 类都有无数据块，并因此 WMI 工具套件生成保存属于类的方法的参数数据结构的声明但不会生成对应于类本身在结构声明。

每个方法都属于此类的 MOF 语法所述的方法的参考页。 以下参考页描述了这些方法以及其随附的结构：

[**AddLink**](addlink.md)

[**AddPort**](addport.md)

[**AddTarget**](addtarget.md)

[**RemoveLink**](removelink.md)

[**RemovePort**](removeport.md)

[**RemoveTarget**](removetarget.md)

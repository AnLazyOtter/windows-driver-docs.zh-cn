# <a name="scsiinquiry"></a>ScsiInquiry


**ScsiInquiry**方法指示的微型端口驱动程序，用于管理 iSCSI 发起程序目标上的逻辑单元向发出 SCSI 查询命令并返回结果的 HBA。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**ScsiInquiry**方法，请参阅成员的说明[ **ScsiInquiry\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_scsiinquiry_in)和[ **ScsiInquiry\_出**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_scsiinquiry_out)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类必须支持**ScsiInquiry**。

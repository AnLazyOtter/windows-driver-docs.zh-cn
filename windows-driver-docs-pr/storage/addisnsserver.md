# <a name="addisnsserver"></a>AddiSNSServer


**AddiSNSServer**方法将 iSNS 服务器添加到的目标的发起程序查询 iSNS 服务器的列表。 HBA 发起程序应在非易失性内存中存储的 iSNS 服务器的列表。

此 WMI 方法属于未发布[MSiSCSI\_Operations WMI 类](msiscsi-operations-wmi-class.md)。 有关参数的说明**AddiSNSServer**方法，请参阅成员的说明[ **AddiSNSServer\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_addisnsserver_in)和[**AddiSNSServer\_出**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_addisnsserver_out)结构。

微型端口驱动程序实现 MSiSCSI\_操作 WMI 类不需要支持此方法。

# <a name="reportdiscoveredtargets2"></a>ReportDiscoveredTargets2


**ReportDiscoveredTargets2** WMI 方法将报告所有发现的目标。

此 WMI 方法属于未发布[MSiSCSI\_DiscoveryOperations WMI 类](msiscsi-discoveryoperations-wmi-class.md)中定义*Discover.mof*。 有关参数的说明**ReportDiscoveredTargets2**方法，请参阅成员的说明[ **ReportDiscoveredTargets2\_出**](https://msdn.microsoft.com/library/windows/hardware/ff564054)结构。

**ReportDiscoveredTargets2**类似于[ReportDiscoveredTargets](reportdiscoveredtargets.md)方法，但**ReportDiscoveredTargets2**报告目标门户信息的**ReportDiscoveredTargets**方法不会报告，如门户组的标记号。

微型端口驱动程序实现 MSiSCSI\_DiscoveryOperations WMI 类必须支持**ReportDiscoveredTargets2**。

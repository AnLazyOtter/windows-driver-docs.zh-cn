# <a name="storage-management-and-wmi-classes"></a>存储管理和 WMI 类


## <span id="ddk_storage_wmi_classes_kr"></span><span id="DDK_STORAGE_WMI_CLASSES_KR"></span>


本部分介绍的存储管理计划规范 (SMI-S) 要求和系统提供的 Windows Management Instrumentation (WMI) 类，此 Windows 存储体系结构中使用。 有关如何存储微型端口驱动程序处理 WMI 类的讨论，请参阅[在存储微型端口驱动程序中处理 WMI Srb](https://msdn.microsoft.com/library/windows/hardware/ff556028)。

存储管理提供程序、 SMI-S 要求和系统提供的 WMI 类是在以下几类：

-   [存储 Managemement 提供程序](https://msdn.microsoft.com/library/windows/hardware/dn342891)

-   [SMI-S 要求](https://msdn.microsoft.com/library/windows/hardware/dn265461)

-   [HBA WMI 类](hba-wmi-classes.md)

-   [iSCSI WMI 类](iscsi-wmi-classes.md)

-   [MPIO WMI 类](mpio-wmi-classes.md)

-   [NPIV WMI 类](npiv-wmi-classes.md)

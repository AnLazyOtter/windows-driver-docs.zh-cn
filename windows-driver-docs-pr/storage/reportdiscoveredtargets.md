# <a name="reportdiscoveredtargets"></a>ReportDiscoveredTargets


**ReportDiscoveredTargets** WMI 方法将报告所有发现的目标。

此 WMI 方法属于未发布[MSiSCSI\_DiscoveryOperations WMI 类](msiscsi-discoveryoperations-wmi-class.md)中定义*Discover.mof*。 有关参数的说明**ReportDiscoveredTargets**方法，请参阅成员的说明[ **ReportDiscoveredTargets\_OUT** ](https://msdn.microsoft.com/library/windows/hardware/ff564058)结构。

微型端口驱动程序实现 MSiSCSI\_DiscoveryOperations WMI 类必须支持**ReportDiscoveredTargets**。

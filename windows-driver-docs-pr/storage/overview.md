# <a name="overview"></a>概述


本主题介绍的基础的设计和体系结构存储接收器驱动程序系统组件，使以下事件序列：

1.  用户连接到他们的系统 IEEE 1667 合规的存储设备。

2.  识别设备。 安装和配置合适的驱动程序、 系统组件和第三方组件。

3.  Windows shell 和任何已安装的第三方应用程序可以发现、 访问、 管理和与根据 IEEE 1667 规范设备进行交互。

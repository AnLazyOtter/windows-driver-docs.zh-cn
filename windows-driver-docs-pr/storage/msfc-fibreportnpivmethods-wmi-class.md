# <a name="msfcfibreportnpivmethods-wmi-class"></a>MSFC\_FibrePortNPIVMethods WMI 类


NPIV WMI 提供程序使用 MSFC\_FibrePortNPIVMethods 类，以允许管理客户端来创建虚拟端口。 这是特别有用的迁移与虚拟机从一个主机到另一个关联的文件。

此 WMI 类只包含方法，并有无数据块。 因此，WMI 工具套件生成包含属于此类的方法的参数数据的结构，但它不会生成对应于类本身的结构。

每个方法都属于此类的 MOF 文件语法所述的方法的参考页。 以下主题介绍了这些方法和其随附的结构：

[**CreateVirtualPort**](createvirtualport.md)

[**RemoveVirtualPort**](removevirtualport.md)

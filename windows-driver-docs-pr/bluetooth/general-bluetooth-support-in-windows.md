**请注意**  蓝牙在 Windows 10 之前的支持信息，请参阅[蓝牙在早期 Windows 版本中支持](bluetooth-support-in-previous-windows-versions.md)。

 

## <a name="span-idwhichbluetoothversionsdoeswindows10supportspanspan-idwhichbluetoothversionsdoeswindows10supportspanspan-idwhichbluetoothversionsdoeswindows10supportspanwhich-bluetooth-versions-does-windows10-support"></a><span id="Which_Bluetooth_versions_does_Windows_10_support_"></span><span id="which_bluetooth_versions_does_windows_10_support_"></span><span id="WHICH_BLUETOOTH_VERSIONS_DOES_WINDOWS_10_SUPPORT_"></span>Windows 10 支持蓝牙版本？


（主页、 专业版、 企业版和教育版） 的桌面版本的 Windows 10 和 Windows 10 移动版支持以下蓝牙版本：

-   版本 1.1
-   版本 2.0
-   2\.0 版本带有 edr 规范
-   版本 2.1
-   版本 2.1 使用 edr 规范
-   版本 4.0
-   版本 4.1
-   5\.0 版

Windows Server 2016 不支持蓝牙。

## <a name="span-idwhichbluetoothprofileshavein-boxsupportinwindows10spanspan-idwhichbluetoothprofileshavein-boxsupportinwindows10spanspan-idwhichbluetoothprofileshavein-boxsupportinwindows10spanwhich-bluetooth-profiles-have-in-box-support-in-windows10"></a><span id="Which_Bluetooth_profiles_have_in-box_support_in_Windows_10_"></span><span id="which_bluetooth_profiles_have_in-box_support_in_windows_10_"></span><span id="WHICH_BLUETOOTH_PROFILES_HAVE_IN-BOX_SUPPORT_IN_WINDOWS_10_"></span>哪些蓝牙配置文件在 Windows 10 中有内置支持？

### <a name="windows-10-for-desktop-editions-in-box-bluetooth-profiles"></a>Windows 10 桌面版现成蓝牙配置文件：

Windows 10 (版本 1803) 支持蓝牙版本 5.0 和以下蓝牙用户配置文件：
- 高级音频分发配置文件 (A2DP 1.2)
- 音频/视频控制传输协议目标 (AVCTP 1.4)
- 音频/视频分发传输协议 (AVDTP 1.2)
- 音频/视频远程控制配置文件 (AVRCP 1.6.1)
- 电池服务通过 GATT 配置文件 (1.0) 
- 蓝牙 LE 泛型属性 (GATT) 客户端
- 蓝牙 LE 泛型属性 (GATT) 服务器
- 蓝牙网络封装协议 (BNEP 1.0)
- 设备 ID 配置文件 (DID 1.3)
- 设备信息服务通过 GATT 配置文件 (DIS 1.1)
- 拨号网络配置文件 (DUN 1.1)
- 通用访问配置文件 (GAP)
- 通用音频/视频分发配置文件 (GAVDP 1.2)
- 无配置文件 (HFP 1.6) 
- 打印电缆替换配置文件 (HCRP 1.2)
- HID over GATT 配置文件 (HOGP 1.0) 
- 人机接口设备 (HID 1.1)
- 人机接口设备服务 （hid 设备）
- 互操作性 (IOP)
- 逻辑链接控件和自适应协议 (L2CAP)
- 对象推送配置文件 (OPP 1.1)
- 个人区域网络用户配置文件 (PANU 1.0)
- RFCOMM (与 TS 07.10 1.1)
- 扫描参数配置文件的客户端一遍 GATT 配置文件 (ScPP 2.1)
- 安全管理器协议 (SMP)
- 串行端口配置文件 (SPP 1.2)
- 服务发现协议 (SDP)

### <a name="windows-10-mobile-in-box-bluetooth-profiles"></a>Windows 10 移动版中框蓝牙配置文件：

- 高级音频分发配置文件 (A2DP 1.2)
- 音频/视频远程控制配置文件 (AVRCP 1.3)
- 蓝牙 LE 泛型属性 (GATT) 客户端
- 设备 ID 配置文件 (DID 1.3)
- 无配置文件 (HFP 1.6) 
- HID over GATT 配置文件 (HOGP 1.0) 
- 人机接口设备 (HID 1.1)
- 消息访问配置文件 (1.1)
- 对象推送配置文件 (OPP 1.1)
- 个人区域网络接入点 (平移 NAP 1.0)
- 电话薄访问配置文件 (PBAP 1.1)
- 串行端口配置文件 (SPP 1.2)

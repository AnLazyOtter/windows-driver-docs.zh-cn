# <a name="bluetooth-certification"></a>蓝牙认证


## <a name="span-idwherearethewindowshardwarecertificationprogramrequirementsforbluetoothwirelesstechnologyspanspan-idwherearethewindowshardwarecertificationprogramrequirementsforbluetoothwirelesstechnologyspanspan-idwherearethewindowshardwarecertificationprogramrequirementsforbluetoothwirelesstechnologyspanwhere-are-the-windows-hardware-certification-program-requirements-for-bluetooth-wireless-technology"></a><span id="Where_are_the_Windows_Hardware_Certification_Program_requirements_for_Bluetooth_wireless_technology_"></span><span id="where_are_the_windows_hardware_certification_program_requirements_for_bluetooth_wireless_technology_"></span><span id="WHERE_ARE_THE_WINDOWS_HARDWARE_CERTIFICATION_PROGRAM_REQUIREMENTS_FOR_BLUETOOTH_WIRELESS_TECHNOLOGY_"></span>Bluetooth 无线技术的 Windows 硬件认证计划要求在哪里？


Windows 硬件认证计划指定的硬件和软件的要求，若要以最佳方式使用 Windows。 有关 Bluetooth 无线电收发器和设备的 Windows 硬件认证计划要求的特定详细信息，请参阅[Windows 硬件认证要求和策略](https://go.microsoft.com/fwlink/p/?linkid=317937)。

有关 Windows 硬件认证计划的常规信息，请参阅[Windows 硬件认证工具包 (HCK)](https://go.microsoft.com/fwlink/p/?LinkId=733613)文档。

## <a name="span-iddowindows81andwindows8havenewbluetoothrequirementsspanspan-iddowindows81andwindows8havenewbluetoothrequirementsspando-windows-81-and-windows-8-have-new-bluetooth-requirements"></a><span id="do_windows_8.1_and_windows_8_have_new_bluetooth_requirements_"></span><span id="DO_WINDOWS_8.1_AND_WINDOWS_8_HAVE_NEW_BLUETOOTH_REQUIREMENTS_"></span>Windows 8.1 和 Windows 8 是否有新的蓝牙要求？


对于 Windows 8.1 和 Windows 8，Windows 硬件认证计划要求 Bluetooth 无线电收发器，以支持蓝牙版本 4.0 才能符合 Windows 徽标。 更新的测试包含在 HCK 的最新版本。 有关在何处下载 HCK 的详细信息，请参阅[Windows 硬件认证要求和策略](https://go.microsoft.com/fwlink/p/?linkid=317937)。

# <a name="connecting-to-an-ieee-12843-data-link-device"></a>连接到 IEEE 1284.3 数据链路设备





并行端口的系统提供的函数驱动程序不提供完全支持 IEEE 1284.3 数据链接层服务;但是，它提供高级别的驱动程序可用来连接、 断开连接、 重置，并发出信号的 IEEE 1284.3 数据链接设备内部设备控制请求。 这些服务是处于开发阶段，适用于支持不是已正式发布的新设备。 有关 IEEE 1284.3 数据链接服务支持的详细信息，请参阅示例代码 *\\src\\内核\\parport*中 Microsoft Windows 驱动程序开发工具包 (DDK)。 (在 DDK 前面带有 Windows Driver Kit \[WDK\]。)

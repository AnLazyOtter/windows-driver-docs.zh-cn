# <a name="obtaining-information-about-a-parallel-device"></a>获取有关并行设备的信息





除了[连接到并行设备](connecting-to-a-parallel-device.md)，客户端可以使用以下设备控制请求以获得有关并行的设备的其他信息：

[**IOCTL\_IEEE1284\_获取\_模式**](https://msdn.microsoft.com/library/windows/hardware/ff543975)

[**IOCTL\_PAR\_获取\_默认\_模式**](https://msdn.microsoft.com/library/windows/hardware/ff544061)

[**IOCTL\_PAR\_GET\_DEVICE\_CAPS**](https://msdn.microsoft.com/library/windows/hardware/ff544067)

[**IOCTL\_PAR\_IS\_端口\_免费**](https://msdn.microsoft.com/library/windows/hardware/ff544073)

[**IOCTL\_PAR\_查询\_设备\_ID\_大小**](https://msdn.microsoft.com/library/windows/hardware/ff544080)

[**IOCTL\_PAR\_查询\_设备\_ID**](https://msdn.microsoft.com/library/windows/hardware/ff544076)

[**IOCTL\_PAR\_查询\_信息**](https://msdn.microsoft.com/library/windows/hardware/ff544086)

[**IOCTL\_PAR\_查询\_位置**](https://msdn.microsoft.com/library/windows/hardware/ff544091)

[**IOCTL\_PAR\_查询\_RAW\_设备\_ID**](https://msdn.microsoft.com/library/windows/hardware/ff544100)

[**IOCTL\_串行\_获取\_超时**](https://msdn.microsoft.com/library/windows/hardware/ff544120)

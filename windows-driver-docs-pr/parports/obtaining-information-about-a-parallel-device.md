# <a name="obtaining-information-about-a-parallel-device"></a>获取有关并行设备的信息





除了[连接到并行设备](connecting-to-a-parallel-device.md)，客户端可以使用以下设备控制请求以获得有关并行的设备的其他信息：

[**IOCTL\_IEEE1284\_获取\_模式**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_ieee1284_get_mode)

[**IOCTL\_PAR\_获取\_默认\_模式**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_get_default_modes)

[**IOCTL\_PAR\_GET\_DEVICE\_CAPS**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_get_device_caps)

[**IOCTL\_PAR\_IS\_端口\_免费**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_is_port_free)

[**IOCTL\_PAR\_查询\_设备\_ID\_大小**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_query_device_id_size)

[**IOCTL\_PAR\_查询\_设备\_ID**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_query_device_id)

[**IOCTL\_PAR\_查询\_信息**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_query_information)

[**IOCTL\_PAR\_查询\_位置**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_query_location)

[**IOCTL\_PAR\_查询\_RAW\_设备\_ID**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddpar/ni-ntddpar-ioctl_par_query_raw_device_id)

[**IOCTL\_串行\_获取\_超时**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddser/ni-ntddser-ioctl_serial_get_timeouts)

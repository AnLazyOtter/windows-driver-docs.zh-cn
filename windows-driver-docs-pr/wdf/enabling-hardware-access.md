# <a name="enabling-hardware-access"></a>启用硬件访问

[!include[UMDF 1 Deprecation](../umdf-1-deprecation.md)]

若要启用硬件访问，UMDF 驱动程序必须设置**UmdfDirectHardwareAccess** INF 指令**AllowDirectHardwareAccess**。

在 UMDF INF 指令的详细信息，请参阅[INF 文件中指定 WDF 指令](specifying-wdf-directives-in-inf-files.md)。

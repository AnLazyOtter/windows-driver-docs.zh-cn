# <a name="developing-drivers-with-the-windows-driver-foundation-reference-book"></a>使用 Windows Driver Foundation 开发驱动程序：参考手册


*使用 Windows Driver Foundation 开发驱动程序*著作也是可用于帮助你了解的概念和基础知识的 Windows 驱动程序框架 (WDF)。 本书介绍了 Windows 驱动程序和基本的内核模式编程，然后介绍 WDF 体系结构和编程模型。 它提供了使用框架开发 Windows 驱动程序的实用的、 面向示例的指南。

# <a name="video-accessing-driver-ifr-logs-without-a-debugger"></a>视频：访问而无需调试程序的驱动程序 IFR 日志

本主题包含视频的本教程演示如何将驱动程序可以访问 IFR 日志，而无需附加调试器。

若要了解如何配置您的驱动程序使用即时记录器 (IFR)，请参阅[日志中记录跟踪的即时跟踪记录器](https://msdn.microsoft.com/library/windows/hardware/dn914610)。

您可以找到 GetIfr.ps1 脚本中的视频中所述[WDF GitHub 存储库](https://github.com/Microsoft/Windows-Driver-Frameworks/tree/master/Tools)。

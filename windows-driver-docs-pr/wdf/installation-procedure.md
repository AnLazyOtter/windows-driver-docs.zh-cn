# <a name="installation-procedure"></a>安装过程


内核模式驱动程序框架 (KMDF) 和用户模式驱动程序框架 (UMDF) 驱动程序安装时使用 INF 文件，就像 WDM 驱动程序。 但是，基于框架的驱动程序必须使用适当的共同安装程序，并且必须添加有关 KMDF 或 UMDF 需要其他服务的信息。

有关实现的详细信息，请参阅[构建、 安装和测试](building--installing--and-testing-a-wdf-driver.md)。

# <a name="kspropertyaudiosignalprocessing-enumeration"></a>KSPROPERTY\_AUDIOSIGNALPROCESSING 枚举


KSPROPERTY\_AUDIOSIGNALPROCESSING 枚举定义一个常量，它由与此插针上的音频处理模式相关的音频驱动程序。

<a name="syntax"></a>语法
------

```ManagedCPlusPlus
typedef enum _KSPROPERTY_AUDIOSIGNALPROCESSING { 
  KSPROPERTY_AUDIOSIGNALPROCESSING_MODES
} KSPROPERTY_AUDIOSIGNALPROCESSING;
```

<a name="constants"></a>常量
---------

<span id="KSPROPERTY_AUDIOSIGNALPROCESSING_MODES"></span><span id="ksproperty_audiosignalprocessing_modes"></span>**KSPROPERTY\_AUDIOSIGNALPROCESSING\_MODES**  
指定的 ID [ **KSPROPERTY\_AUDIOSIGNALPROCESSING\_模式**](ksproperty-audiosignalprocessing-modes.md)属性。

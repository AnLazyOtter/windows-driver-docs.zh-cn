# <a name="speaker-phantoming"></a>扬声器构成幻路


通常情况下，多渠道的系统，包括中心和附属 loudspeakers 中的所有 loudspeakers 始终都是存在的。 但是，用户可能不具有所有预期的 loudspeakers 或他们可以选择一个或多个将其打开。 一个常见示例是一个多渠道的系统，缺少 center 扬声器。 演讲者 phantoming 重现中缺少扬声器，声音通常通过相邻 loudspeakers 之间拆分。 但是，phantoming 还可以使用如 rear 左和 rear 右或端左和端右扬声器其他组合。

# <a name="kspropsetidacousticechocancel"></a>KSPROPSETID\_声学\_Echo\_取消


## <span id="ddk_kspropsetid_acoustic_echo_cancel_ks"></span><span id="DDK_KSPROPSETID_ACOUSTIC_ECHO_CANCEL_KS"></span>


`KSPROPSETID_Acoustic_Echo_Cancel`属性设置用于控制和监视 AEC 节点 ([**KSNODETYPE\_声学\_ECHO\_取消**](ksnodetype-acoustic-echo-cancel.md))。 在 Microsoft Windows XP 及更高版本支持此属性集。 在此集中的属性是 AEC 节点的可选属性。

在此集中的属性指定为 KSPROPERTY\_AEC 枚举值。

`KSPROPSETID_Acoustic_Echo_Cancel`属性集包含以下属性：

[**KSPROPERTY\_AEC\_模式**](ksproperty-aec-mode.md)

[**KSPROPERTY\_AEC\_NOISE\_FILL\_ENABLE**](ksproperty-aec-noise-fill-enable.md)

[**KSPROPERTY\_AEC\_状态**](ksproperty-aec-status.md)

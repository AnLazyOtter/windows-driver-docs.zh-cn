# <a name="ksnodetypesum"></a>KSNODETYPE\_总和


## <span id="ddk_ksnodetype_sum_ks"></span><span id="DDK_KSNODETYPE_SUM_KS"></span>


KSNODETYPE\_总和节点表示简单求和 (SUM) 函数。 SUM 节点都有多个输入的流和一个输出流，所有使用相同的数据格式。 同样混合的输入的流，以生成输出流。

KSNODETYPE\_总和节点不需要支持的任何属性。

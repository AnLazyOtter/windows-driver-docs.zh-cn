# <a name="windows-default-apos"></a>Windows 默认 a p o s


使用 Windows Vista 音频驱动程序不提供数字音频系统效果。 由称为系统效果音频处理对象 (Apo) 的实时进程内 COM 对象提供用于创建系统效果处理的所有数字信号。

Windows 提供了**增强功能**选项卡上配置这些系统效果控件面板的声音小程序。 随 Windows 提供默认情况下不显示在下面的列表。

# <a name="ksnodetypedelay"></a>KSNODETYPE\_延迟


## <span id="ddk_ksnodetype_delay_ks"></span><span id="DDK_KSNODETYPE_DELAY_KS"></span>


KSNODETYPE\_延迟节点表示延迟控件。 延迟控件具有一个输入的流和一个输出流，并且这两个流共享相同的数据格式。 在延迟的控件导致要按某些指定的时间内滞后于输入流的输出流。

KSNODETYPE\_延迟节点应支持以下必需的属性：

[**KSPROPERTY\_AUDIO\_DELAY**](ksproperty-audio-delay.md)

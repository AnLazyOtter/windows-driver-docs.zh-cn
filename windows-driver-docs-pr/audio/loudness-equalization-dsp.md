# <a name="loudness-equalization-dsp"></a>响度均衡 DSP


响度均衡 DSP 可确保跨不同源的音频信号的音量级别保持不变。 电视节目，例如，可能是在只是适当的音量级别而在程序中的商业插播可能差别卷中。 这要求用户调整相应地设置音量。 今天的昂贵支持 HD 的电视的一些可以均衡卷，以便在某种程度上常量级别保持声音。 这有效如果依赖于您的声音播放的电视但大多数家庭影院和家庭音频爱好者电视音频直接连接到其立体声音响系统。 此外，今天的响度均衡解决方案通常不是为不同的音频内容和源有效的。

使用此音频效果的 Windows 版本可以维护更一致的感知的响度跨不同的数字音频文件或比早期的 Windows 技术的源。 这意味着该响度始终保持在指定范围内，即使对于不同的数字信号。

# <a name="ksnodetypeloudness"></a>KSNODETYPE\_响度


## <span id="ddk_ksnodetype_loudness_ks"></span><span id="DDK_KSNODETYPE_LOUDNESS_KS"></span>


KSNODETYPE\_响度节点表示调整输出流的动态范围的音量控件。 响度控件具有一个输入的流和一个输出流，并且两个流的每个具有相同的数据格式。

KSNODETYPE\_响度节点应支持至少一个以下属性：

[**KSPROPERTY\_音频\_响度**](ksproperty-audio-loudness.md)

[**KSPROPERTY\_AUDIO\_DYNAMIC\_RANGE**](ksproperty-audio-dynamic-range.md)

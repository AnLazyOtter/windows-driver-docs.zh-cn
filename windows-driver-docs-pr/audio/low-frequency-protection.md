# <a name="low-frequency-protection"></a>低频率保护


低频率保护是一种[转发低音管理](bass-management.md)。 使用没有任何低音并且所有发言人很小，缺少低音功能。 低频率保护 APO 应用时，结果将是一部分的低于指定的 APO 的算法的截止频率的音频信号删除。

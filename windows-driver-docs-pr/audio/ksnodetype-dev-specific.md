# <a name="ksnodetypedevspecific"></a>KSNODETYPE\_DEV\_SPECIFIC


## <span id="ddk_ksnodetype_dev_specific_ks"></span><span id="DDK_KSNODETYPE_DEV_SPECIFIC_KS"></span>


KSNODETYPE\_开发人员\_特定节点表示的特定于设备的控件。 特定于设备的控件可以有多个输入的流和多个输出流。 它执行一组特定于特定的音频设备的函数。

KSNODETYPE\_开发人员\_特定节点应支持以下必需的属性：

[**KSPROPERTY\_AUDIO\_DEV\_SPECIFIC**](ksproperty-audio-dev-specific.md)

# <a name="customizing-control-panel"></a>自定义控制面板


在 Windows XP 和更高版本的 Windows 中，你可以自定义控制面板中的声音应用程序。 若要执行此操作，添加自定义位图或图标。 或者，如果您要开发通过 SPDIF 流式传输的第三方数字音频格式，你可以使此数字音频格式可在控制面板中。

有关如何添加位图或图标或如何使用 INF 文件将第三方的数字音频格式添加到控制面板的详细信息，请参阅以下主题：

[品牌打造控件面板中的位图或图标](branding-control-panel-with-bitmaps-or-icons.md)

[添加新的数字格式都移到控件面板](adding-new-digital-formats-to-control-panel.md)

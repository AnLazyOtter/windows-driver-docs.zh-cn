# <a name="hardware-resource-management"></a>硬件资源管理


通过 HD 音频 DDI，功能用于音频和调制解调器的编解码器的驱动程序可以分配和释放 HD Audio 控制器设备中的硬件资源。 这些资源主要有：

-   HD Audio 控制器中的 DMA 引擎。

-   HD 音频链接上的总线带宽。

本部分包括：

分配 DMA 引擎

分配链路带宽

条带化

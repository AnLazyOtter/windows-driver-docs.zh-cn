# <a name="ksnodetypemute"></a>KSNODETYPE\_静音


## <span id="ddk_ksnodetype_mute_ks"></span><span id="DDK_KSNODETYPE_MUTE_KS"></span>


KSNODETYPE\_静音节点表示的静音控件。 此节点有一个输入的流和一个输出流，并在两个流的每个具有相同的数据格式。 打开或关闭，它将切换输入。

KSNODETYPE\_静音节点应支持以下必需的属性：

[**KSPROPERTY\_AUDIO\_MUTE**](ksproperty-audio-mute.md)

# <a name="kspropsetidjack"></a>KSPROPSETID\_插孔


`KSPROPSETID_Jack`属性集用于实现物理的音频插孔的表示形式。 属性集公开的物理特征和使用情况的音频插孔。

`KSPROPSETID_Jack`属性集包含[ **KSPROPERTY\_JACK** ](ksproperty-jack.md)枚举以及以下属性：

[**KSPROPERTY\_JACK\_说明**](ksproperty-jack-description.md)

[**KSPROPERTY\_JACK\_DESCRIPTION2**](ksproperty-jack-description2.md)

[**KSPROPERTY\_JACK\_SINK\_INFO**](ksproperty-jack-sink-info.md)

[**KSPROPERTY\_JACK\_CONTAINERID**](ksproperty-jack-containerid.md)

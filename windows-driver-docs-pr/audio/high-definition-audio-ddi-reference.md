# <a name="high-definition-audio-ddi-reference"></a>高清音频 DDI 参考


本部分是一项技术参考，描述了[**HDAUDIO\_总线\_接口**](https://docs.microsoft.com/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_bus_interface)的三个版本的高清晰度（HD）音频设备驱动程序接口（DDI）中的例程， [**HDAUDIO\_总线\_接口\_V2**](https://docs.microsoft.com/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_bus_interface_v2)和[**HDAUDIO\_BUS\_接口\_BDL**](https://docs.microsoft.com/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_bus_interface_bdl)结构定义。 它还描述了这些例程用作参数的结构类型。

有关 DDI 的详细信息，请参阅[高清晰音频 DDI](https://docs.microsoft.com/windows-hardware/drivers/audio/high-definition-audio-ddi)。

# <a name="acpi-power-meter-interface"></a>ACPI 功率表接口


WDM 驱动程序，提供 Power 计量接口 (PMI) 从底层硬件平台访问的能源计量和预算 (PMB) 信息。 该驱动程序通过使用 ACPI PMI 对象执行此操作。 该平台通过使用 ACPI 4.0 能源计量对象公开此信息。

本部分提供 ACPI PMI 的高级概述。

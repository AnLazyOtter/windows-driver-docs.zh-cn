# <a name="windows-driver-kit-documentation"></a>Windows 驱动程序工具包文档

Windows 驱动程序工具包 (WDK) 10 集成了 Microsoft Visual Studio 2017 和 [Windows 调试工具](/windows-hardware/drivers/debugger/index)。此集成环境提供了开发、生成、打包、部署、测试和调试驱动程序所需的工具。 可以在此集成环境中运行许多基本认证测试。 Windows 驱动程序工具包 (WDK) 针对多种技术和驱动程序模型都提供了模板，这包括 [Windows 驱动程序框架](https://docs.microsoft.com/windows-hardware/drivers/wdf/)、通用串行总线 (USB)、打印、网络和文件系统筛选器。

<p><b>WDK 10 的运行时要求</b></p>

WDK 需要 Visual Studio。 有关详细信息，请参阅 [Visual Studio 2017 系统要求](https://docs.microsoft.com/visualstudio/productinfo/vs2017-system-requirements-vs)。

EWDK 还另外需要 .NET 4.6.1。 有关 .NET 要求的信息，请参阅 [.NET Framework 系统要求](https://docs.microsoft.com/dotnet/framework/get-started/system-requirements)。 

可以使用 WDK Insider Preview 和 EWDK Insider Preview 开发在以下操作系统上运行的驱动程序：

|  客户端 OS  |         服务器 OS          |
|-------------|----------------------------|
| Windows 10  |    Windows Server 2016     |
| Windows 8.1 |   Windows Server 2012 R2   |
|  Windows 8  |    Windows Server 2012     |
|  Windows 7  | Windows Server 2008 R2 SP1 |

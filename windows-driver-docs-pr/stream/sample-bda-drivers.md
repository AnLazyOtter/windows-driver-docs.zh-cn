# <a name="sample-bda-drivers"></a>示例 BDA 驱动程序



Windows Driver Kit (WDK) 随附的版本 7600 与 Windows 7 包括示例，用于演示的原则和广播驱动程序体系结构 (BDA) 和 BDA 微型驱动程序如何使用 AVStream 类驱动程序的功能。 这些示例使用实现可用作一个起始点，您可以编写专有 BDA 微型驱动程序的源代码。

在安装 WDK 版本 7600 时，可以安装这些示例。 每个 BDA 驱动程序示例安装在\\src\\Swtuner\\ WDK 安装的主目录下的子目录。 有关这些示例的详细信息，请参阅 WDK 中每个 BDA 驱动程序示例的源代码。

这些示例中不可用[Windows 驱动程序示例](https://github.com/Microsoft/Windows-driver-samples)GitHub 上。

# <a name="avstream-headers"></a>AVStream 标头





所有内核流式处理和 AVStream 都引用材料包括结构和在 Ks.h 和 Ksmedia.h 中声明的函数。 这些是必须包含微型驱动程序，以便访问 Microsoft 提供 KS 和 AVStream 类驱动程序支持的标头。

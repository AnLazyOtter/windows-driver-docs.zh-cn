# <a name="broadcast-network-type-guids"></a>广播网络类型 Guid


AVStream 微型驱动程序使用的网络类型 Guid 来指定它支持使用执行扫描操作的广播的网络类型。 *Bdamedia.h*标头文件定义了这些 Guid。 有关获取支持的网络类型的扫描功能的详细信息，请参阅[ **KSPROPERTY\_调谐器\_扫描\_CAPS** ](ksproperty-tuner-scan-caps.md)和[**KSPROPERTY\_调谐器\_NETWORKTYPE\_扫描\_CAPS** ](ksproperty-tuner-networktype-scan-caps.md)属性。

BDA 中提供了以下网络类型 Guid:

<span id="ANALOG_AUXIN_NETWORK_TYPE"></span><span id="analog_auxin_network_type"></span>ANALOG\_AUXIN\_NETWORK\_TYPE  
将模拟辅助信号广播的网络类型。

<span id="ANALOG_FM_NETWORK_TYPE"></span><span id="analog_fm_network_type"></span>ANALOG\_FM\_NETWORK\_TYPE  
向发出信号广播频率调制 (FM) 广播的网络类型。

<span id="ANALOG_TV_NETWORK_TYPE"></span><span id="analog_tv_network_type"></span>ANALOG\_TV\_NETWORK\_TYPE  
模拟电视信号广播的网络类型。

<span id="DIGITAL_CABLE_NETWORK_TYPE"></span><span id="digital_cable_network_type"></span>数字\_电缆\_网络\_类型  
将数字信号广播通过电缆的网络类型。

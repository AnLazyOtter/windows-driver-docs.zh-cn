# <a name="kseventsetidvpnotify"></a>KSEVENTSETID\_VPNotify


## <span id="ddk_kseventsetid_vpnotify_ks"></span><span id="DDK_KSEVENTSETID_VPNOTIFY_KS"></span>


**KSEVENTSETID\_VPNotify**事件集用于视频端口事件发生，例如中的视频端口的像素格式的更改时通知客户端。

在此集中的事件项指定在 KSEVENT\_VPNOTIFY 枚举。

在此集中的唯一事件是[ **KSEVENT\_VPNOTIFY\_格式**](ksevent-vpnotify-formatchange.md)。

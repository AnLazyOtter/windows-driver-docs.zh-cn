# <a name="kseventsetidvolumelimit"></a>KSEVENTSETID\_VolumeLimit


KSEVENTSETID\_VolumeLimit 事件集音频音量限制更改事件发生时通知客户端。

在此集中的事件项中指定**KSEVENT\_VOLUMELIMIT**枚举。

在此集中的唯一事件是[ **KSEVENT\_VOLUMELIMIT\_CHANGED**](ksevent-volumelimit-changed.md)。

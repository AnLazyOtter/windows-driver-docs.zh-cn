# <a name="kseventstreamallocatorinternalfreeframe"></a>KSEVENT\_STREAMALLOCATOR\_INTERNAL\_FREEFRAME


**KSEVENT\_STREAMALLOCATOR\_内部\_FREEFRAME**事件是中的默认分配器实现只在内部使用和其他实现。 到服务等待者具有挂起的分配 Irp，不应启用客户端使用此事件。

# <a name="eventsetidvideodecoder"></a>EVENTSETID\_VIDEODECODER


## <span id="ddk_eventsetid_videodecoder_ks"></span><span id="DDK_EVENTSETID_VIDEODECODER_KS"></span>


**EVENTSETID\_VIDEODECODER**事件集发生视频解码器更改事件，如选择新的物理输入的连接器时通知客户端。

在此集中的事件项指定在 KSEVENT\_VIDEODECODER 枚举。

在此集中的唯一事件是[ **KSEVENT\_VIDEODECODER\_CHANGED**](ksevent-videodecoder-changed.md)。

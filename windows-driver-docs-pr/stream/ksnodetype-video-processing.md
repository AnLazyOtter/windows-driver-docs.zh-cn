# <a name="ksnodetypevideoprocessing"></a>KSNODETYPE\_视频\_处理


此节点类型中定义*ksmedia.h*。 它可能返回到调用应用程序**获取\_NodeType**方法**IKsTopologyInfo**接口。 **IKsTopologyInfo** DirectX 9.2 SDK 文档的 DirectShow 部分中所述。 **获取\_NodeType**参考页介绍了此节点类型。

**IKsTopologyInfo**接口可能使用的应用程序与 Microsoft 提供通信[USB 视频类驱动程序](https://docs.microsoft.com/windows-hardware/drivers/stream/usb-video-class-driver)(UVC) 类驱动程序。 驱动程序编写人员不要使用此节点类型。

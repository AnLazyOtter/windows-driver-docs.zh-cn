# <a name="kspropsetiddvdsubpic"></a>KSPROPSETID\_DvdSubPic


## <span id="ddk_kspropsetid_dvdsubpic_ks"></span><span id="DDK_KSPROPSETID_DVDSUBPIC_KS"></span>


KSPROPSETID\_DvdSubPic 属性集定义属性来控制颜色、 对比度和子画面显示输出。

KSPROPERTY\_中的 DVDSUBPIC 枚举*Ksmedia.h*指定此集的属性。

DVD 解码器微型驱动程序应实现支持以下属性：

[**KSPROPERTY\_DVDSUBPIC\_PALETTE**](ksproperty-dvdsubpic-palette.md)

[**KSPROPERTY\_DVDSUBPIC\_HLI**](ksproperty-dvdsubpic-hli.md)

[**KSPROPERTY\_DVDSUBPIC\_复合\_ON**](ksproperty-dvdsubpic-composit-on.md)

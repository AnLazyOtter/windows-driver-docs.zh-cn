# <a name="kseventsetidcameraasynccontrol"></a>KSEVENTSETID\_CameraAsyncControl


KSEVENTSETID\_CameraAsyncControl 事件集指定照相机异步控件事件，还可以启用对筛选器。 一组包含以下事件：

[**KSEVENT\_CAMERACONTROL\_焦点**](ksevent-cameracontrol-focus.md)

[**KSEVENT\_CAMERACONTROL\_缩放**](ksevent-cameracontrol-zoom.md)

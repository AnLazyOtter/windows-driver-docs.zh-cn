# <a name="kseventstreamallocatorfreeframe"></a>KSEVENT\_STREAMALLOCATOR\_FREEFRAME


**KSEVENT\_STREAMALLOCATOR\_FREEFRAME**事件可用于免费的框架是可用于分配时接收通知。 可以通过使用直接调用接口，而不 IRP 接口，它通常排队待处理的请求无法完成客户端使用此事件。

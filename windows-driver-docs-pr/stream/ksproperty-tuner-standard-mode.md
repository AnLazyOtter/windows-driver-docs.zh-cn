# <a name="ksproperty_tuner_standard_mode"></a>KSPROPERTY\_调谐器\_标准\_模式


KSPROPERTY\_调谐器\_标准\_模式属性检索一个布尔值，该值指示驱动程序是否可以将调谐设备设置为自动检测信号本身的调谐器标准。 此属性可以选择实现。

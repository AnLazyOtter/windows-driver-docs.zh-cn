# <a name="kspropsetidaudiodecoderout"></a>KSPROPSETID\_AudioDecoderOut


## <span id="ddk_kspropsetid_audiodecoderout_ks"></span><span id="DDK_KSPROPSETID_AUDIODECODEROUT_KS"></span>


KSPROPSETID\_AudioDecoderOut 属性集定义属性以指定可从 DVD 解码器的音频输出的窗体。

KSPROPERTY\_中的 AUDDECOUT 枚举*Ksmedia.h*指定此集的属性。

DVD 解码器微型驱动程序应实现对以下音频流属性的支持：

[**KSPROPERTY\_AUDDECOUT\_模式**](ksproperty-auddecout-modes.md)

[**KSPROPERTY\_AUDDECOUT\_CUR\_模式**](ksproperty-auddecout-cur-mode.md)

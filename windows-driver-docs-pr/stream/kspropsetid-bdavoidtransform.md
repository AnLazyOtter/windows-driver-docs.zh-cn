# <a name="kspropsetidbdavoidtransform"></a>KSPROPSETID\_BdaVoidTransform


## <span id="ddk_kspropsetid_bdavoidtransform_ks"></span><span id="DDK_KSPROPSETID_BDAVOIDTRANSFORM_KS"></span>


KSPROPSETID\_BdaVoidTransform 是 BDA void 转换属性集。 它用于控制节点启动和停止时。 网络提供程序筛选器使用此属性设置来控制节点的操作。

可用属性如下：

<span id="KSPROPERTY_BDA_VOID_TRANSFORM_START"></span><span id="ksproperty_bda_void_transform_start"></span>[**KSPROPERTY\_BDA\_VOID\_转换\_开始**](ksproperty-bda-void-transform-start.md)  
启动节点的操作。

<span id="KSPROPERTY_BDA_VOID_TRANSFORM_STOP"></span><span id="ksproperty_bda_void_transform_stop"></span>[**KSPROPERTY\_BDA\_VOID\_转换\_停止**](ksproperty-bda-void-transform-stop.md)  
停止节点的操作。

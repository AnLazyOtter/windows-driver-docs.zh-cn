# <a name="kspropsetidwave"></a>KSPROPSETID\_批


## <span id="ddk_kspropsetid_wave_ks"></span><span id="DDK_KSPROPSETID_WAVE_KS"></span>


KSPROPSETID\_批属性集定义属性来控制输出量，从硬件 DVD 解码器或具有环回电缆连接到声卡的模拟电视调谐器卡。

KSPROPERTY\_中的批枚举*Ksmedia.h*指定此集的属性。

定义以下属性：

[**KSPROPERTY\_批\_兼容\_功能**](ksproperty-wave-compatible-capabilities.md)

[**KSPROPERTY\_批\_输入\_功能**](ksproperty-wave-input-capabilities.md)

[**KSPROPERTY\_批\_输出\_功能**](ksproperty-wave-output-capabilities.md)

[**KSPROPERTY\_批\_缓冲区**](ksproperty-wave-buffer.md)

[**KSPROPERTY\_批\_频率**](ksproperty-wave-frequency.md)

[**KSPROPERTY\_批\_卷**](ksproperty-wave-volume.md)

[**KSPROPERTY\_批\_平移**](ksproperty-wave-pan.md)

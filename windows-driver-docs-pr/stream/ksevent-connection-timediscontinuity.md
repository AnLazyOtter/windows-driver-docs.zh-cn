# <a name="kseventconnectiontimediscontinuity"></a>KSEVENT\_连接\_TIMEDISCONTINUITY


**KSEVENT\_连接\_TIMEDISCONTINUITY**事件允许客户端接收通知的时间同步。 通过流接口的插针发送时的不连续性标志时，将发生此事件。 没有为此事件参数。

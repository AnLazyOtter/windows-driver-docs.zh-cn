# <a name="ksproperty_bda_null_transform_start"></a>KSPROPERTY\_BDA\_NULL\_转换\_开始


客户端使用 KSPROPERTY\_BDA\_NULL\_转换\_开始禁用该节点之前应用到信号的任何转换，并允许信号无变化地通过该节点。

# <a name="kspropsetidclock"></a>KSPROPSETID\_时钟


## <span id="ddk_kspropsetid_clock_ks"></span><span id="DDK_KSPROPSETID_CLOCK_KS"></span>


流式处理时钟的内核对象支持 KSPROPSETID\_时钟属性集。 客户端可以使用以下属性来检查时钟状态：

[**KSPROPERTY\_时钟\_CORRELATEDPHYSICALTIME**](ksproperty-clock-correlatedphysicaltime.md)

[**KSPROPERTY\_时钟\_CORRELATEDTIME**](ksproperty-clock-correlatedtime.md)

[**KSPROPERTY\_时钟\_FUNCTIONTABLE**](ksproperty-clock-functiontable.md)

[**KSPROPERTY\_CLOCK\_PHYSICALTIME**](ksproperty-clock-physicaltime.md)

[**KSPROPERTY\_时钟\_解析**](ksproperty-clock-resolution.md)

[**KSPROPERTY\_CLOCK\_STATE**](ksproperty-clock-state.md)

[**KSPROPERTY\_CLOCK\_TIME**](ksproperty-clock-time.md)

# <a name="ksmethodbdacreatetopology"></a>KSMETHOD\_BDA\_创建\_拓扑


客户端使用 KSMETHOD\_BDA\_创建\_拓扑，以反映筛选器中的已知的连接的 3 环中创建的拓扑结构。

# <a name="device-transform-manager-events"></a>设备转换管理器事件


本部分包含有关设备转换管理器事件的信息。

| Event                                                                        | 描述                                                                                                               |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [METransformFlushInputStream](metransformflushinputstream.md)               | 通知设备 Transform Manager 刷新已连接到的输入设备 MFT devproxy 的输出流。 |
| [METransformHaveOutput](metransformhaveoutput.md)                           | 指示设备转换具有一个其输出流上准备一个示例。                                        |
| [METransformInputStreamStateChanged](metransforminputstreamstatechanged.md) | 指示输入流状态，或者必须更改媒体类型。                                                      |
| [METransformNeedInput](metransformneedinput.md)                             | 指示设备转换需要一个输入。                                                                         |

# <a name="ksproperty_tuner_networktype_scan_caps"></a>KSPROPERTY\_调谐器\_NETWORKTYPE\_扫描\_CAP


KSPROPERTY\_调谐器\_NETWORKTYPE\_扫描\_CAP 属性介绍了优化设备支持的特定广播网络类型的扫描功能。 此属性可以选择实现。

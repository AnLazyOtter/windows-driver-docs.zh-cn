# <a name="kspropertytunernetworktypescancaps"></a>KSPROPERTY\_TUNER\_NETWORKTYPE\_SCAN\_CAPS


KSPROPERTY\_调谐器\_NETWORKTYPE\_扫描\_CAPS 属性描述优化设备支持的特定广播的网络类型的扫描功能。 可以根据需要实现此属性。

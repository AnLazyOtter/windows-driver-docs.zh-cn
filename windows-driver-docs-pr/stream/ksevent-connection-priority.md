# <a name="kseventconnectionpriority"></a>KSEVENT\_连接\_优先级


**KSEVENT\_连接\_优先级**事件允许客户端接收通知的优先级失败。 由于优先级别而丢失资源时，将发生此事件。 没有为此事件参数。

# <a name="dvd-data-streams"></a>DVD 数据流





DVD 解码器微型驱动程序支持这两个输入和输出数据流。 流由三个 Guid （主要类型、 次要类型和说明符） 和格式设置块唯一定义。 Stream 格式也称为媒体类型。 有关格式的块和媒体类型的详细信息，请参阅 DirectShow 中的 Microsoft Windows SDK 文档

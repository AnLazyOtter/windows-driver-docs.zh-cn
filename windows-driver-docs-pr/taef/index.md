# <a name="test-authoring-and-execution-framework-taef"></a>测试创作和执行框架 (TAEF)


测试创作和执行框架 (TAEF) 为开发人员和测试人员提供了一致的机制来用于编写和运行自动化。 该框架提供了单一的基础结构，允许他们跨专业和团队共享测试。

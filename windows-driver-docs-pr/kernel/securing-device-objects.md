# <a name="securing-device-objects"></a>保护设备对象





本部分介绍如何保护驱动程序的设备对象从未经授权的访问。 它包含以下各节：

[控制设备访问权限](controlling-device-access.md)

[控制设备 Namespace 访问权限](controlling-device-namespace-access.md)

[设备对象的 SDDL](sddl-for-device-objects.md)

有两个方面的必须安全的设备对象：

-   对设备本身的访问。 这中所述[控制的设备访问](controlling-device-access.md)。

-   对设备的命名空间的访问。 这中所述[控制设备 Namespace 访问](controlling-device-namespace-access.md)。

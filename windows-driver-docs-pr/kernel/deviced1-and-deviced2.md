# <a name="deviced1-and-deviced2"></a>DeviceD1 和 DeviceD2





**DeviceD1**并**DeviceD2**的成员[**设备\_功能**](https://msdn.microsoft.com/library/windows/hardware/ff543095)指示是否在设备硬件支持这些设备的电源状态。 每个是如果设备支持状态，并且如果设备不支持状态明确设置的单个位。 操作系统将认为所有设备都支持 D0 和 D3[设备的电源状态](device-power-states.md)。

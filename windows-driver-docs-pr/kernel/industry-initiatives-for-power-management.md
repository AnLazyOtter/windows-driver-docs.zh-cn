# <a name="industry-initiatives-for-power-management"></a>电源管理的行业举措





OnNow 计划定义所需的电源管理硬件和软件的支持。

*高级配置和电源接口规范*、 部分的 OnNow 计划中，定义一个硬件级别界面，让操作系统来实现电源管理中的一致性独立于平台的方式。

一个*设备类电源管理引用规范*适用于每个常见的设备类，例如音频或通信设备。 每个这些规范定义一类设备的电源管理的要求。 驱动程序编写人员应参考这些规范，可通过[ACPI / 电源管理](https://go.microsoft.com/fwlink/p/?linkid=57185)网站，了解特定于设备的详细信息。

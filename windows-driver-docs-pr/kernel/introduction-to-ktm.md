# <a name="introduction-to-ktm"></a>KTM 简介


内核事务管理器 (KTM) 是一种事务管理服务，可用于在用户模式或内核模式 （或两者） 中创建事务处理系统 (TPS)。

KTM 是 Microsoft Windows 操作系统的内核模式服务。 KTM 是 Windows Vista 和更高版本的 Windows 上可用。

KTM 提供用户模式和内核模式下的接口。 本文档描述 KTM 的内核模式的接口。 有关 Microsoft Win32 用户模式接口的信息，请参阅 Microsoft Windows SDK。

有关事务从理论上讲的详细信息，请参阅标题为本书*事务处理：概念和技术*Jim Gray 和 Andreas Reuter。

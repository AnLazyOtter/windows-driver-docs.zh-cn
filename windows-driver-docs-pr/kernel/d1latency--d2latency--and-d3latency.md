# <a name="d1latency-d2latency-and-d3latency"></a>D1Latency、D2Latency 和 D3Latency





**D1Latency**， **D2Latency**，并**D3Latency**的成员[**设备\_功能**](https://msdn.microsoft.com/library/windows/hardware/ff543095)包含的大概时间，以 100 微秒为单位，设备所需的用于从睡眠状态的每个返回到 D0 的状态。 驱动程序应指定不支持任何设备电源状态为零的滞后时间。

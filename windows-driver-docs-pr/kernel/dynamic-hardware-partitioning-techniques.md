# <a name="dynamic-hardware-partitioning-techniques"></a>动态硬件分区技术


更改服务器的硬件配置时，服务器正在运行被称为*动态硬件分区*。 如果你想要在支持动态硬件分区的服务器上运行设备驱动程序，您的驱动程序必须支持对服务器的硬件配置动态更改。

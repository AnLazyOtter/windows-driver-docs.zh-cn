# <a name="removing-a-device-in-a-filter-driver"></a>在筛选器驱动程序中删除设备





筛选器驱动程序时删除设备，必须撤消任何操作它执行添加和启动设备。 删除设备时，筛选器驱动程序应遵循与功能驱动程序实质上是相同的步骤。

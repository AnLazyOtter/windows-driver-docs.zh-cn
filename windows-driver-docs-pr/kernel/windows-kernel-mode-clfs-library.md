# <a name="windows-kernel-mode-clfs-library"></a>Windows 内核模式 CLFS 库


Windows 系统文件提供了事务日志记录系统。 此系统称为公用日志文件系统 (CLFS)。 CLFS 的详细信息，请参阅[公用日志文件系统](using-common-log-file-system.md)。

例程，以字母为前缀 CLFS 提供直接接口"**Clfs**"; 有关 CLFS 库例程的列表，请参阅[公用日志文件系统 (CLFS) 库例程](https://msdn.microsoft.com/library/windows/hardware/ff541804)。 CLFS 还提供了一系列例程，可实现来管理 CLFS;CLFS 管理的详细信息，请参阅[CLFS 管理库例程](https://msdn.microsoft.com/library/windows/hardware/ff541830)。

CLFS 是一种技术，与事务处理的文件系统; 相关有关事务的详细信息，请参阅[Windows 内核模式下事务管理器](windows-kernel-mode-kernel-transaction-manager.md)。

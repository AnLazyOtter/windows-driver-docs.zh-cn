# <a name="user-defined-ecps"></a>用户定义的 ECP


若要定义自己 ECP，定义用于标识你 ECP 和描述你 ECP 的上下文结构的 GUID。 有关如何定义 Guid 并使用这些驱动程序代码中的信息，请参阅[驱动程序中使用 Guid](https://msdn.microsoft.com/library/windows/hardware/ff565392)。

# <a name="disabling-nfp"></a>禁用 NFP


订阅、 发布和状态，可以暂时禁用客户端。

暂时禁用针对订阅、 发布和状态显示通过发送[ **IOCTL\_NFP\_禁用**](https://msdn.microsoft.com/library/windows/hardware/jj853315)句柄。 当客户端想要禁用的邻近性功能，但保留分配快速重新启用它们时所需的资源时，这很有用。

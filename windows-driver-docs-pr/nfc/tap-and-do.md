# <a name="tap-and-do"></a>点击和执行


*点击，然后执行*是用于触发执行这样的操作一起它们持有的设备之间的邻近的人物之间自然交互的笔势。

从 Windows 8 开始，新手势引入现实世界交互称作*点击，然后执行*。 *点击，然后执行*在窄物理卷 （大约厘米） 中工作，因此它是刻意。 编程模型映射到物理环境内的设备之间执行操作的触发此阐明。 基础系统的附近字段邻近 (NFP) 主要用于电磁波字段，例如 NFC，技术上建模，但平台灵活，满足 NFP 要求其他创新型系统受支持。 Windows 提供了基于 NFP 很容易理解，轻型且直观的用户模型。 Windows 包括了大量的内置利用 NFP 的体验。 该 API 是适用于第三方开发的。

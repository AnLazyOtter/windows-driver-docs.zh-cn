# <a name="srcsrv"></a>SrcSrv


SrcSrv 工具 (Srcsrv.dll)，客户端检索用于生成应用程序的源文件的确切版本。 由于模块的源代码可以更改版本之间和在过去的年，请务必存在相关模块的版本生成时查看的源代码。

SrcSrv 从源代码管理中检索相应的文件。 若要使用 SrcSrv，应用程序必须已编制源索引。

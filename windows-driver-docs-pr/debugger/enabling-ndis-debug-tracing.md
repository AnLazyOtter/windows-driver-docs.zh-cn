# <a name="enabling-ndis-debug-tracing"></a>启用 NDIS 调试跟踪


NDIS 调试跟踪，用于调试的 NDIS 驱动程序的主要方法。 如果设置了跟踪的 NDIS 调试时，实际上要启用使用 NDIS DbgPrint 语句的一个或多个级别。 生成的信息足以调试大多数网络驱动程序问题。

可以通过设置相应的注册表值来启用调试跟踪。 有关详细信息，请参阅[启用 NDIS 调试跟踪通过设置注册表值](enabling-ndis-debug-tracing-by-setting-registry-values.md)。

设置注册表值，以启用调试跟踪的工作原理，即使主机计算机没有安装 Ndis.sys 符号的经检查的版本。

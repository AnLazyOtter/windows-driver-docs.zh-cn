# <a name="windbg-preview---breakpoints"></a>WinDbg Preview - Breakpoints

本部分介绍如何处理使用 WinDbg 预览调试器的断点。

## <a name="breakpoints-menu"></a>断点菜单

使用断点菜单创建新的和删除现有断点以及切换初始断点 （初始断点目前仅内核）。

![在调试器中的断点菜单](images/windbgx-breakpoints-menu.png)

## <a name="breakpoints-window"></a>断点窗口

使用断点窗口，通过视图菜单中，打开要查看、 设置和清除断点。 此外可以双击断点，以打开其源文件。

![在调试器中的断点菜单](images/windbgx-breakpoints-window.png)

断点窗口保持每次命中断点时的运行总计。


使用断点的一般过程是与以前版本的 WinDbg 类似。 有关设置断点的详细信息，请参阅[在 WinDbg 中设置断点](setting-breakpoints-in-windbg.md)。

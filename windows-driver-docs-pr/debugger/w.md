# <a name="w"></a>W


<span id="wdbgexts_api"></span><span id="WDBGEXTS_API"></span>**WdbgExts API**  
由扩展的调试器引擎公开的接口。 它包含的调试器引擎 API 功能的子集，并且只能由扩展。

<span id="wdbgexts_extension"></span><span id="WDBGEXTS_EXTENSION"></span>**WdbgExts 扩展**  
扩展基于 wdbgexts.h 标头文件中的原型。 这些扩展使用 WdbgExts API。

<span id="wow64"></span><span id="WOW64"></span>**WOW64**  
在可以运行 32 位 Windows 应用程序的 64 位 Windows 仿真层。 调试 64 位 Windows 上运行的 32 位应用程序时，必须区分应用程序本身和 WOW64 子系统。

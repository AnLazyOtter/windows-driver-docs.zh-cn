# <a name="source-control-systems"></a>源代码管理系统


SrcSrv 包括 Visual SourceSafe (vss.pm)、 Perforce (p4.pm)、 Team Foundation Server (tfn.pm) 和 Subversion (svn.pm) 提供程序的模块。 此外，还有一个并发版本系统 (CVS) 模块，用于与 SrcSrv，一起使用，但未广泛测试。 还有可能要生成你自己的模块以支持其他版本控制系统。

本部分包括：

[使用 Visual SourceSafe 进行调试](debugging-with-visual-sourcesafe.md)

[使用 CVS](using-cvs.md)

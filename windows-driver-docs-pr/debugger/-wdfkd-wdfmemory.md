# <a name="wdfkdwdfmemory"></a>!wdfkd.wdfmemory


**！ Wdfkd.wdfmemory**扩展显示的地址和与 framework 内存对象相关联的缓冲区的大小。

```dbgcmd
!wdfkd.wdfmemory Handle
```

# <a name="undocking-a-window"></a>取消停靠窗口


## <span id="ddk_undocking_a_window_dbg"></span><span id="DDK_UNDOCKING_A_WINDOW_DBG"></span>


若要取消停靠窗口，并使其浮动窗口，请执行以下操作：

-   双击窗口的标题栏。

-   通过右键单击窗口的标题栏中，右键单击窗口的选项卡，如果它是一个选项卡式的集合的一部分或单击窗口的右上角的图标打开快捷菜单，然后单击**取消停靠**。

-   在 WinDbg 窗口中，在**窗口**菜单上，单击**取消停靠所有**。 此命令将变为所有停靠窗口浮动窗口。

当按上述方法之一移除一个窗口时，窗口返回到其以前的停靠位置。

此外可以通过单击其标题栏将停靠的窗口。 此操作，可将窗口移动到其他停靠位置，或将其取消停靠。 （将停靠的窗口拖动到新位置工作方式完全相同将浮动窗口拖动到新位置。 有关将浮动窗口拖动到新位置的详细信息，请参阅[停靠窗口](docking-a-window.md)。)

当尝试取消停靠或拖动选项卡式的窗口的任何一种方法时，会移动只有选项卡式的集合中的活动窗口。

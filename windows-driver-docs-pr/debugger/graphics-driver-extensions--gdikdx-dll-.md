# <a name="graphics-driver-extensions-gdikdxdll"></a>图形驱动程序扩展 (Gdikdx.dll)


## <span id="ddk_graphics_driver_extensions_gdikdx_dll__dbg"></span><span id="DDK_GRAPHICS_DRIVER_EXTENSIONS_GDIKDX_DLL__DBG"></span>


可以 Gdikdx.dll 中找到可用于调试图形驱动程序接口 (GDI) 的扩展命令。

此扩展的 Windows 2000 版本 DLL 将出现在 w2kfre 和 w2kchk 目录中。 没有任何版本的此 DLL 适用于 Windows XP 或更高版本的 Windows。

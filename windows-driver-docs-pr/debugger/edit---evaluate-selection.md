# <a name="edit--evaluate-selection"></a>编辑 | 评估所选内容


## <span id="ddk_edit_evaluate_selection_dbg"></span><span id="DDK_EDIT_EVALUATE_SELECTION_DBG"></span>


单击**评估所选内容**上**编辑**菜单来评估中的当前选定[源窗口](source-window.md)并显示在结果[调试器命令窗口](debugger-command-window.md)。

此命令相当于按下 CTRL + SHIFT + V，单击**评估所选内容**在源窗口的快捷菜单上，或使用[ **??(评估C++表达式)** ](----evaluate-c---expression-.md)命令及作为其参数的所选文本。

如果所选的文本中包含多个行，会产生语法错误。 如果在源窗口中不选择任何文本，则不能使用此命令。

源窗口的其他功能的详细信息，请参阅[源窗口](source-window.md)。

# <a name="view--registers"></a>视图 | 寄存器


## <span id="ddk_view_registers_dbg"></span><span id="DDK_VIEW_REGISTERS_DBG"></span>


单击**注册**上**视图**菜单打开[寄存器窗口](registers-window.md)。 如果此窗口已打开，变为活动状态。

此命令相当于按下 ALT + 4 或单击**寄存器 (Alt + 4)** 按钮 (![寄存器按钮的屏幕截图](images/tbreg.png)) 工具栏上。

有关此窗口及其用法的详细信息，请参阅[寄存器窗口](registers-window.md)。

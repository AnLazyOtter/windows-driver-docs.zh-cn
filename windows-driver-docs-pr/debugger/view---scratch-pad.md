## <span id="ddk_view_scratch_pad_dbg"></span><span id="DDK_VIEW_SCRATCH_PAD_DBG"></span>


单击**便笺**上**视图**菜单打开便笺。 如果此窗口已打开，变为活动状态。

此命令相当于按下 ALT + 8 或单击**草稿板 (Alt + 8)** 按钮 (![的草稿板按钮的屏幕截图](images/tbspad.png)) 工具栏上。

# <a name="usbkdusbhcdhccontext"></a>!usbkd.usbhcdhccontext


**！ Usbkd.usbhcdhccontext**命令将显示**USB2LIB\_HC\_上下文**USB 主控制器。

```dbgcmd
!usbkd.usbhcdhccontext DeviceExtension
```

# <a name="bug-check-0x32-phase1initializationfailed"></a>Bug 检查 0x32：阶段 1\_初始化\_失败


阶段 1\_初始化\_失败错误检查的值为 0x00000032。 这表示该系统初始化失败。

> [!IMPORTANT]
> 本主题面向程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://www.windows.com/stopcode)。

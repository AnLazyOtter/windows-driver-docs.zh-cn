# <a name="v"></a>V


<span id="virtual_process"></span><span id="VIRTUAL_PROCESS"></span>**虚拟过程**  
在内核模式下调试调试器引擎创建了单个*虚拟过程*来表示目标的内核。

<span id="virtual_thread"></span><span id="VIRTUAL_THREAD"></span>**虚拟线程**  
创建在内核模式下调试调试器引擎*虚拟线程*为目标计算机中的每个处理器。

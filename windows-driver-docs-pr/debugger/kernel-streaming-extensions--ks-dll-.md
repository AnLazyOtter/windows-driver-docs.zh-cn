# <a name="kernel-streaming-extensions-ksdll"></a>流式处理扩展插件 (Ks.dll) 的内核


可以 Ks.dll 中找到可用于调试内核流式处理驱动程序和 AVStream 驱动程序的扩展命令。

这些引用页面中的示例输出的大多数生成的调试 Avssamp.sys 以筛选器为中心的示例。 此示例包含在 Windows 驱动程序工具包。

如果你想要使用 Windows 2000 中 Ks.dll 扩展，必须将 Ks.dll 从 kdexts 目录复制到 w2kfre 目录。

您需要使用此扩展的特殊符号。 有关详细信息，请参阅[有关 Windows 调试工具](https://go.microsoft.com/fwlink/p/?linkid=8708)Web 站点。

只需通过输入到调试器中不带任何参数的命令，可以在本部分中获取的许多扩展命令的其他信息。

有关详细信息，请参阅[流式处理的内核调试](kernel-streaming-debugging.md)。

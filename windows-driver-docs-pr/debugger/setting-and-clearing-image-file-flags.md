# <a name="setting-and-clearing-image-file-flags"></a>设置和清除映像文件标志


## <span id="ddk_setting_and_clearing_image_file_flags_dtools"></span><span id="DDK_SETTING_AND_CLEARING_IMAGE_FILE_FLAGS_DTOOLS"></span>


图像文件设置会影响在该命令完成后启动指定的程序的实例。 它们保存在注册表中，并保持有效，直到更改它们。

**若要设置和清除的图像文件注册表标志**

1.  单击**映像文件**选项卡。

    下面的屏幕截图所示**映像文件**Windows Vista 中的选项卡。

    ![windows vista 中的图像文件选项卡的屏幕截图 ](images/gflags-image.png)

2.  在中**图像**框中，键入可执行文件或 DLL，其中包括文件扩展名的名称，然后按 TAB 键。

    这将激活上对应的复选框**映像文件**选项卡。

3.  设置或清除标志，通过选中或清除关联的标志的复选框。

4.  已选择或清除所有所需的标志，请单击**应用**。

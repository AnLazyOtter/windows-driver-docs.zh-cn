# <a name="resizing-and-moving-a-window"></a>调整窗口大小和移动窗口


## <span id="ddk_resizing_and_moving_windows_dbg"></span><span id="DDK_RESIZING_AND_MOVING_WINDOWS_DBG"></span>


浮动窗口始终是与 WinDbg 窗口相关联。 如果最大程度减少 WinDbg 所有浮动窗口最小化。 如果要还原的 WinDbg，还原所有浮动窗口中。 WinDbg 窗口后面，可以永远不会将浮动窗口。

每个浮动窗口独立地相互和从移动 WinDbg 窗口中，除非你已选择**移动与帧**窗口的快捷菜单上。

停靠的窗口所占的 WinDbg 帧内的固定的位置。 如果在调整大小的 WinDbg，所有停靠窗口自动缩放的新大小。 这种情况适用于 windows 的已停靠在单独的停靠。

如果您将对边框的鼠标指针移动两个停靠窗口之间时，鼠标指针将变为箭头。 通过拖动此箭头，可以调整两个相邻的窗口，并将它们保留在停靠状态。

WinDbg 窗口始终填充停靠窗口。 永远不会任意空白区域中没有窗口除非没有停靠窗口。 这种情况适用于独立停靠。

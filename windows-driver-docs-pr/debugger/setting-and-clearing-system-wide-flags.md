# <a name="setting-and-clearing-system-wide-flags"></a>设置和清除系统范围的标志


## <span id="ddk_setting_and_clearing_system_wide_flags_dtools"></span><span id="DDK_SETTING_AND_CLEARING_SYSTEM_WIDE_FLAGS_DTOOLS"></span>


整个系统的注册表设置会影响运行在 Windows 上的所有进程。 它们保存在注册表中，并保持有效，直到更改它们。

**若要设置和清除整个系统注册表标志**

1.  单击**系统注册表**选项卡。

    下面的屏幕截图显示了 Windows Vista 中的系统注册表选项卡。![windows vista 中系统注册表选项卡的屏幕截图](images/gflags-registry.png)

2.  设置或清除标志，通过选中或清除关联的标志的复选框。

3.  已选择或清除所有所需的标志，请单击**应用**。

4.  重新启动 Windows 以使更改生效。

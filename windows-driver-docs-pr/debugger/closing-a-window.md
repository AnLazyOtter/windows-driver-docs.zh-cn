# <a name="closing-a-window"></a>关闭窗口


## <span id="ddk_closing_a_window_dbg"></span><span id="DDK_CLOSING_A_WINDOW_DBG"></span>


若要关闭调试的信息窗口，请单击**关闭**在窗口的右上角的按钮。

如果你想要关闭的活动窗口，还可以单击**关闭当前窗口**上**文件**菜单或按 CTRL + F4。

可以关闭一个调试信息窗口中按 ALT + SHIFT +*数量*，其中，ALT +*数*是打开此窗口键盘快捷方式。 (有关可能的快捷键的列表，请参阅[打开一个窗口](opening-a-window.md)。)

若要关闭所有[源 windows](source-window.md)在同一时间，单击[关闭所有源 Windows](window---close-all-source-windows.md)上**窗口**菜单。 此命令不会关闭源窗口中，已被指定为选项卡形式停靠目标。 此设置的详细信息，请参阅源窗口主题。

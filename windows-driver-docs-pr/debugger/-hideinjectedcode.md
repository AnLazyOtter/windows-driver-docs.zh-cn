# <a name="hideinjectedcode-hide-injected-code"></a>.hideinjectedcode（隐藏注入的代码）


**.Hideinjectedcode**命令上，打开或关闭的插入的代码隐藏。

```dbgcmd
.hideinjectedcode { on | off } 
.hideinjectedcode help 
.hideinjectedcode
```

## <a name="span-idddkmetaholdandcomparememorydbgspanspan-idddkmetaholdandcomparememorydbgspanparameters"></a><span id="ddk_meta_hold_and_compare_memory_dbg"></span><span id="DDK_META_HOLD_AND_COMPARE_MEMORY_DBG"></span>参数


<span id="_______on_____________"></span><span id="_______ON_____________"></span> **on**   
打开的插入的代码隐藏。

<span id="_______off_"></span><span id="_______OFF_"></span> **off** ****   
关闭的插入的代码隐藏

<span id="_______help______"></span><span id="_______HELP______"></span> **help**   
显示此命令的帮助。

<a name="remarks"></a>备注
-------

如果输入此命令不使用任何参数，它显示有关命令的帮助。

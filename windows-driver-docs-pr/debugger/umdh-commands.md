# <a name="umdh-commands"></a>UMDH 命令


## <span id="ddk_umdh_commands_dtools"></span><span id="DDK_UMDH_COMMANDS_DTOOLS"></span>


UMDH 有两种不同模式，每个都有其自己的命令语法和参数：

-   [**分析运行中的进程**](analyze-a-running-process.md) -显示进程中的所有堆分配。

-   [**分析 UMDH 日志**](analyze-umdh-logs.md) -在相同的过程的两个不同时间进行分配列表进行比较。 分析差异可帮助确定内存泄漏。

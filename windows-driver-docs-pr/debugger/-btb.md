# <a name="btb"></a>!btb


**！ Btb**扩展插件都会显示在基于 Itanium 的处理器、 分支跟踪缓冲区 (BTB) 配置和跟踪注册为当前的处理器。

```dbgcmd
!btb
```

**重要**  此命令已被 Windows 调试器版本 10.0.14257 中不推荐使用和更高版本，并不再可用。

# <a name="agestore"></a>AgeStore


AgeStore 工具 (agestore.exe) 中删除一个目录或目录树中，基于其上次访问日期中的文件。 此程序是用于从符号服务器，以便节省磁盘空间使用的下游存储区中删除旧文件特别有用。

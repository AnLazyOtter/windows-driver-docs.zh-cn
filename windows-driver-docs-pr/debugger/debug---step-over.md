# <a name="debug--step-over"></a>调试 | 逐步运行


## <span id="ddk_debug_step_over_dbg"></span><span id="DDK_DEBUG_STEP_OVER_DBG"></span>


单击**单步跳过**上**调试**菜单以在目标上执行一条指令。 如果指令是函数调用，则执行整个函数。

此命令相当于按 F10 或单击**逐过程执行 (F10)** 按钮 (![的屏幕截图步骤按钮上方](images/tbover.png)) 工具栏上。

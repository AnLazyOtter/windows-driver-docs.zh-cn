# <a name="adplus"></a>ADPlus


## <span id="ddk_adplus_dtoolq"></span><span id="DDK_ADPLUS_DTOOLQ"></span>


ADPlus 7.0 版是 ADPlus 总重写。 ADPlus V7.0 用托管代码中，使我们可以轻松地添加新功能。 ADPlus.exe 保留 ADPlus.vbs 的基本功能，并添加了一些其他功能。 此外，不会调用 ADPlusManager，将 ADPlus 扩展到类似 HPC 计算机群集的分布式环境的新助手。

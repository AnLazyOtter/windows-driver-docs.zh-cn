# <a name="list-of-open-windows"></a>打开的窗口列表


## <span id="ddk_list_of_open_windows_dbg"></span><span id="DDK_LIST_OF_OPEN_WINDOWS_DBG"></span>


在底部**窗口**菜单中，所有当前打开的调试信息窗口的列表。 单击此列表以打开窗口中的任何窗口。

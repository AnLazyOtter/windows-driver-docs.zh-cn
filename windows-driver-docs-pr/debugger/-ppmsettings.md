# <a name="ppmsettings"></a>!ppmsettings


**！ Ppmsettings**命令显示当前处于活动状态的 ppm 处理器设置。

```dbgcmd
!ppmsettings
```

无

在以下版本的 Windows 上支持此命令：

- Windows 10 版本 1511
- Windows 10 版本 1607
- Windows 10 版本 1703

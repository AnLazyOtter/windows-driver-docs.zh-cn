# <a name="ttd-module-objects"></a>TTD 模块对象
## <a name="description"></a>描述
*TTD 模块*对象用于提供已加载和卸载期间跟踪会话的模块有关的信息。

## <a name="properties"></a>属性

| 属性 | 描述 |
| --- | --- |
| 名称 | 名称和模块的路径。 |
| 地址 | 已加载的模块地址。 |
| 大小 | 以字节为单位的模块的大小。 |
| 校验和 | 模块的校验和。 |
| 时间戳 | 模块的时间戳。 |

*挂起的信息*

# <a name="ttd-module-objects"></a>TTD 模块对象

## <a name="description"></a>描述
*TTD 模块*对象用于在跟踪会话期间，为加载和卸载的模块生成相关信息。

## <a name="properties"></a>“属性”

| 属性 | 描述 |
| --- | --- |
| 名称 | 模块的名称和路径。 |
| 地址 | 加载模块的地址。 |
| Size | 模块的大小（以字节为单位）。 |
| 校验和 | 模块的校验和。 |
| 时间戳 | 模块的时间戳。 |

# <a name="sieextpubdll"></a>SieExtPub.dll


## <span id="ddk_oem_support_extensions_kdex2x86_dll__dbg"></span><span id="DDK_OEM_SUPPORT_EXTENSIONS_KDEX2X86_DLL__DBG"></span>


SieExtPub.dll 扩展模块不属于 Microsoft 调试工具的 Windows，并且 SieExtPub.dll 不再包括在调试诊断工具。 许多 SieExtPub.dll 中的扩展命令现在处于 ext.dll。 若要查看在 ext.dll 扩展命令的列表，请在调试器中输入以下命令。

**.extmatch /e ext \\***

**请注意**   **！ critlist**扩展命令不再可用。

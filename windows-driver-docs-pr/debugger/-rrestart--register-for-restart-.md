# <a name="rrestart-register-for-restart"></a>.rrestart （注册于重启）


**.Rrestart**命令注册在重新启动或应用程序故障情况下重新启动调试会话。

```dbgcmd
.rrestart
```

<a name="remarks"></a>备注
-------

此命令的提升的调试器会话不起作用。

# <a name="view--watch"></a>视图 | 监视


## <span id="ddk_view_watch_dbg"></span><span id="DDK_VIEW_WATCH_DBG"></span>


单击**Watch**上**视图**菜单以打开监视窗口。 如果此窗口已打开，变为活动状态。

此命令相当于按下 ALT + 2 或单击**Watch (Alt + 2)** 按钮 (![的监视按钮屏幕截图](images/tbwatch.png)) 工具栏上。

# <a name="windbg-preview---home-menu"></a>WinDbg 预览-主页菜单

![带有位模式的 windbg 预览的小徽标](images/windbgx-preview-logo.png)

本主题介绍如何使用主菜单。

![调试器中的 "视图" 菜单](images/windbgx-home-menu.png)

## <a name="flow-control"></a>流控制

使用*流控制*按钮可以中断到连接的调试目标、恢复目标上的代码执行以及单步执行和跳出代码。

## <a name="reverse-flow-control"></a>反向流控制

使用 "*反向流控制*" 按钮及时回复。 有关详细信息，请参阅[时间行程调试 - 概述](time-travel-debugging-overview.md)。

## <a name="end"></a>结束

使用*结束*按钮重新启动、分离和停止调试。

## <a name="preferences"></a>“首选项”

使用*首选项*按钮在源代码和程序集视图之间切换，并访问 "**设置**" 菜单。

## <a name="help-support"></a>帮助（支持）

使用 "*帮助*" 按钮可以执行以下操作：
- 查看本地帮助-脱机
- 联机帮助-最新

### <a name="feedback-hub"></a>反馈中心

- 发送反馈（有关发送反馈以改善 WinDbg 的详细信息，请参阅[提供反馈](debugging-using-windbg-preview.md#providing-feedback)。）

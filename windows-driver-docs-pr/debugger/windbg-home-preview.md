# <a name="windbg-preview---home-menu"></a>WinDbg 预览的主菜单 

本主题介绍如何使用主菜单。

![在调试器中的视图菜单](images/windbgx-home-menu.png)


## <a name="flow-control"></a>流控制

使用*流控制*按钮中断连接的调试目标，恢复目标上的代码执行和逐，为执行和跳出代码。

## <a name="end"></a>结束

使用*最终*按钮以重新启动，分离和停止调试。

## <a name="source-mode"></a>源模式

使用*源模式*按钮之间进行切换源代码和程序集的视图。

## <a name="help-support"></a>帮助 （支持）

使用*帮助*按钮以执行以下操作：
- 查看帮助
- 发送反馈 (发送反馈，以改进 WinDbg 的详细信息，请参阅[提供反馈](debugging-using-windbg-preview.md#providing-feedback)。)

# <a name="bug-check-0x22-filesystem"></a>Bug 检查 0x22：文件\_系统


该文件\_检查系统错误的值为 0x00000022。

检查此错误极少出现。

> [!IMPORTANT]
> 本主题面向程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://windows.microsoft.com/windows-10/troubleshoot-blue-screen-errors)。

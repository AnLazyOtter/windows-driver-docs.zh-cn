# <a name="debug--step-out"></a>调试 | 跳出


## <span id="ddk_debug_step_out_dbg"></span><span id="DDK_DEBUG_STEP_OUT_DBG"></span>


单击**单步跳出**上**调试**菜单以继续在目标系统上运行。 此命令执行当前函数的其余部分，并在完成返回的函数时中断。

此命令相当于按 SHIFT + F11 或单击**跳出 (Shift + F11)** 按钮 (![单步跳出按钮的屏幕截图](images/tbout.png)) 工具栏上。

# <a name="register-objects"></a>注册对象 
## <a name="summary"></a>总结
通过注册对象描述的寄存器中机器指令的操作数使用。 注册对象的字符串转换是注册的名称。
## <a name="object-properties"></a>对象属性
|名称|描述|
|--- |--- |
|ID|唯一标识符 （在特定的体系结构的域） 的注册。|

## <a name="remarks"></a>备注
目前，可以通过字符串转换仅获取注册的名称。

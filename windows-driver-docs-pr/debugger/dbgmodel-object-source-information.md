# <a name="source-information-objects"></a>源信息对象 
## <a name="summary"></a>总结
通过源信息对象的源代码与机器指令。
## <a name="object-properties"></a>对象属性
|名称|描述|
|--- |--- |
|FunctionName|要描述函数的名称。|
|FunctionAddress|要描述函数的基址。|
|FunctionOffset|中的函数所描述的位置的偏移量。|

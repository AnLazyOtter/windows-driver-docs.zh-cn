# <a name="launching-a-program-with-flags"></a>启动带标志的程序


## <span id="ddk_launching_a_program_with_flags_dtools"></span><span id="DDK_LAUNCHING_A_PROGRAM_WITH_FLAGS_DTOOLS"></span>


此功能与指定的标志一次运行的程序。 这些设置会影响仅启动程序的实例。 不保存在注册表中。

**若要启动的程序标志**

1.  单击**映像文件**选项卡。

2.  在中**图像**框中，键入可执行文件或 DLL，包括文件扩展名，以及该程序，任何命令的名称，然后按 TAB 键。

    这将激活**启动**按钮和复选框**图像文件**选项卡。

3.  设置或清除标志，通过选中或清除关联的标志的复选框。

4.  单击**启动**按钮。

    下面的屏幕截图所示**启动**按钮**图像文件**Windows Vista 中的选项卡。

    ![windows vista 中的图像文件选项卡的屏幕截图 ](images/gflags-launch.png)

**请注意**  标志在注册表中设置不会影响启动程序的实例。
即使它们不是图像文件标志，在对话框中设置的标志用于启动的实例。

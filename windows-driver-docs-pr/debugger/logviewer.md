# <a name="logviewer"></a>LogViewer


## <span id="ddk_logviewer_dtoolq"></span><span id="DDK_LOGVIEWER_DTOOLQ"></span>


日志查看器实用程序可以操作.lgv 文件，即记录器工具生成的已压缩的日志文件。 若要加载文件，只需启动 Logviewer.exe 通过 Windows 资源管理器或从命令提示符窗口。 将需要一段时间来分析清单文件。 完成此操作，可以调用**文件 |打开**并选择所需的.lgv 文件。

日志查看器打开后.lgv 文件，它将以已记录的顺序显示所有函数的列表。 日志查看器支持功能强大的命令来筛选出不感兴趣的函数。 它还可以将筛选的日志文件的版本保存为文本文件。

[筛选日志查看器函数列表](filtering-the-logviewer-function-list.md)

# <a name="oem-support-extensions-kdex2x86dll"></a>OEM 支持扩展 (kdex2x86.dll)


## <span id="ddk_oem_support_extensions_kdex2x86_dll__dbg"></span><span id="DDK_OEM_SUPPORT_EXTENSIONS_KDEX2X86_DLL__DBG"></span>


OEM 支持扩展插件可在 w2kfre\\kdex2x86.dll 的有关 Windows 调试工具安装目录的子目录。

您可以只适用于 Windows 2000 目标使用这些扩展。

本文档中未介绍这些扩展。 有关这些扩展的说明，请参阅 OEM 支持工具文档。 若要获取此文档，以及整个 OEM 支持工具程序包，请转到[Microsoft 支持文章 253066](https://go.microsoft.com/fwlink/p/?LinkId=241339)并按照该页上的说明。

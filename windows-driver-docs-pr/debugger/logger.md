# <a name="logger"></a>记录器


## <span id="ddk_logger_dtoolq"></span><span id="DDK_LOGGER_DTOOLQ"></span>


可以通过两个不同的车辆激活记录器工具。 一种方法是使用独立的 Logger.exe 程序。 另一个按钮用于启动 CDB 或的 WinDbg 中，并使用 Logexts.dll 调试器扩展。 这两种方法将生成相同类型的日志输出。 但是，若要在任何基于 NT 的操作系统上使用的最佳工具是 CDB 或 WinDbg 与 Logexts.dll 扩展命令。

记录器车辆也适用，但使用调试器为您提供记录器功能强大的调试器的完整功能。

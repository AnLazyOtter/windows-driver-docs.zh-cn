# <a name="view--verbose-output"></a>视图 | 详细输出


## <span id="ddk_view_verbose_output_dbg"></span><span id="DDK_VIEW_VERBOSE_OUTPUT_DBG"></span>


单击**详细输出**上**视图**菜单启用详细模式下打开和关闭。

此命令相当于按下 CTRL + ALT + V。 （和 KD 中按 CTRL + V）。

启用详细模式后，某些显示的命令 （例如注册转储） 生成更详细的输出。 显示发送到调试器的每个模块加载操作。 并且每次将驱动程序或 DLL 加载的操作系统，调试器会通知。

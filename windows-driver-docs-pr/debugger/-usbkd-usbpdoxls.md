# <a name="usbkdusbpdoxls"></a>!usbkd.usbpdoxls


**！ Usbkd.usbpdoxls**命令显示有关所有物理设备对象 (PDOs) 创建的 USB 集线器驱动程序的信息。 对每个 PDO，此命令显示的逗号分隔值的行。

```dbgcmd
!usbkd.usbpdoxls
```

# <a name="rcdrkdrcdrtmffile"></a>!rcdrkd.rcdrtmffile


**！ Rcdrkd.rcdrtmffile**扩展设置或清除跟踪消息格式 (TMF) 文件的名称。

```dbgcmd
!rcdrkd.rcdrtmffile [Filename] 
```

## <a name="span-idddkdevobjdbgspanspan-idddkdevobjdbgspanparameters"></a><span id="ddk__devobj_dbg"></span><span id="DDK__DEVOBJ_DBG"></span>参数


<span id="_______Filename______"></span><span id="_______filename______"></span><span id="_______FILENAME______"></span> *文件名*   
TMF 文件的名称。 如果未指定此参数，将清除 filename。

# <a name="ndiskdpkt"></a>!ndiskd.pkt


**警告**  此扩展是旧的 NDIS 5.x 驱动程序。 [NDIS\_数据包](https://msdn.microsoft.com/library/windows/hardware/ff557086)结构和其关联的体系结构已被弃用。

 

**！ Ndiskd.pkt**扩展显示有关的信息[NDIS\_数据包](https://msdn.microsoft.com/library/windows/hardware/ff557086)结构。

```console
!ndiskd.pkt [-packet] [-verbosity] 
```

# <a name="urestart-unregister-for-restart"></a>.urestart（取消注册以重启）


**.Urestart**命令取消注册在重新启动或应用程序故障情况下重新启动调试会话。

```dbgcmd
.urestart
```

<a name="remarks"></a>备注
-------

此命令的提升的调试器会话不起作用。

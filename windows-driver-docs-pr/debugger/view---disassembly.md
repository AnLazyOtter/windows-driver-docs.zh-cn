# <a name="view--disassembly"></a>视图 | 反汇编


## <span id="ddk_view_disassembly_dbg"></span><span id="DDK_VIEW_DISASSEMBLY_DBG"></span>


单击**反汇编**上**视图**菜单打开[反汇编窗口](disassembly-window.md)。 如果此窗口已打开，变为活动状态。

此命令相当于按下 ALT + 7 或单击**反汇编 (Alt + 7)** 按钮 (![反汇编按钮的屏幕截图](images/tbdisasm2.png)) 工具栏上。

有关此窗口及其用法的详细信息，请参阅[反汇编窗口](disassembly-window.md)。

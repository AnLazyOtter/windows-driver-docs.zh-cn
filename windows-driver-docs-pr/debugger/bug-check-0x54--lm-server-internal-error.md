# <a name="bug-check-0x54-lmserverinternalerror"></a>Bug 检查 0x54：LM\_SERVER\_INTERNAL\_ERROR

检查此错误极少出现。

> 本主题面向程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://www.windows.com/stopcode)。

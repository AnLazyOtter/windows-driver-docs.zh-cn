# <a name="windbg-preview---notes-command-memory-and-source-menus"></a>WinDbg 预览-注释、命令、内存和源菜单

本部分介绍如何使用 WinDbg Preview 中的注释、命令、内存和源菜单。

## <a name="notes"></a>注释

使用菜单可以：

- 打开注释文件
- 保存注释文件

## <a name="command"></a>命令

使用命令菜单可以：

- 优先使用 DML
- 突出显示并取消选中当前选定文本（CTRL + ALT + H）
- 清除命令窗口文本
- 将窗口文本保存到 dml 文件

## <a name="memory"></a>内存

使用 "内存" 菜单可以：

- 设置数据模型内存查询
- 设置内存大小，例如，到字节或长
- 设置显示格式，例如 hex 或带符号
- 设置文本显示格式，例如 ASCII

## <a name="source"></a>来源

使用 "源" 菜单可以：

- 打开源文件
- 设置指令指针
- 运行到光标处
- 关闭所有源窗口

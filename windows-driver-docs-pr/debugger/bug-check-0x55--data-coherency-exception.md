# <a name="bug-check-0x55-data_coherency_exception"></a>Bug 检查0x55：数据\_一致性\_异常


数据\_一致性\_异常 bug 检查的值为0x00000055。

此 bug 检查很少出现。

> [!IMPORTANT]
> 本主题面向程序员。 如果你是在使用计算机时收到蓝屏错误代码的客户，请参阅[排查蓝屏错误](https://www.windows.com/stopcode)。


## <a name="resolution"></a>分辨率 
[ **！分析**](https://docs.microsoft.com/windows-hardware/drivers/debugger/-analyze)调试扩展显示有关 bug 检查的信息，可帮助确定根本原因。

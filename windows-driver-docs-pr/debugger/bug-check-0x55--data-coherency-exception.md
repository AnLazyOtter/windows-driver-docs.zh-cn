# <a name="bug-check-0x55-datacoherencyexception"></a>Bug 检查 0x55：数据\_协调性\_异常


数据\_协调性\_异常错误检查的值为 0x00000055。

检查此错误极少出现。

> [!IMPORTANT]
> 本主题面向程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://www.windows.com/stopcode)。

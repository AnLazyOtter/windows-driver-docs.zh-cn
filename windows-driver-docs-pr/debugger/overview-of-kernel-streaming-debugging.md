# <a name="overview-of-kernel-streaming-debugging"></a>内核调试流式处理的概述


可在扩展模块 Ks.dll 内核调试的流式处理扩展插件。

可以使用 Ks.dll 以帮助调试 KS、 AVStream，端口类和流式传输类驱动程序。

Ks.dll 中的扩展命令的完整列表，请参阅[内核流式处理扩展插件 (Ks.dll)](kernel-streaming-extensions--ks-dll-.md)。

如果你想要使用 Windows 2000 中这些扩展，必须将 Ks.dll 从 kdexts 目录复制到 w2kfre 目录。

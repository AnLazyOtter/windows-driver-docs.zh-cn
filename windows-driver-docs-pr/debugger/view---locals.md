# <a name="view--locals"></a>视图 | 局部变量


## <span id="ddk_view_locals_dbg"></span><span id="DDK_VIEW_LOCALS_DBG"></span>


单击**局部变量**上**视图**菜单以打开局部变量窗口。 如果此窗口已打开，变为活动状态。

此命令相当于按下 ALT + 3 或单击**局部变量 (Alt + 3)** 按钮 (![局部变量按钮的屏幕截图](images/tblocal.png)) 工具栏上。

有关此窗口及其用法的详细信息，请参阅[局部变量窗口](locals-window.md)。

# <a name="bug-check-0x63-security1initializationfailed"></a>Bug 检查 0x63:SECURITY1\_初始化\_失败


SECURITY1\_初始化\_失败错误检查的值为 0x00000063。

检查此错误极少出现。

**重要**本主题适用于程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://windows.microsoft.com/windows-10/troubleshoot-blue-screen-errors)。

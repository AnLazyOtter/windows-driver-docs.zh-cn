# <a name="debugging-windows-apps-using-the-windows-debugger"></a>使用 Windows 调试器调试 Windows 应用


可以使用 Windows 调试器 （WinDbg、 CDB 和 NTSD） 调试 Windows 应用。 使用[ **PLMDebug** ](plmdebug.md)工具能够挂起、 继续和终止一个 Windows 应用程序进行调试时的控制。

若要调试托管代码的 Windows 应用程序，加载[SOS 调试扩展 (sos.dll)](https://go.microsoft.com/fwlink/p/?linkid=223345)和数据访问组件 (mscordacwks.dll)。 有关详细信息，请参阅[使用 Windows 调试器调试托管代码](debugging-managed-code.md)。

Windows 调试器是独立于 Visual Studio 调试器。 有关 windows 调试器和 Visual Studio 调试器之间的区别的信息，请参阅[Windows 调试](index.md)。

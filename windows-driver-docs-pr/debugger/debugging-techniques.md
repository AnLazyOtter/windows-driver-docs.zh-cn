# <a name="debugging-techniques"></a>调试方法


本部分介绍两种类型的调试技术： 标准和专用。 标准的技术适用于大多数调试方案中，且示例包括设置断点、 检查调用堆栈和查找内存泄漏。 专业的技术应用于特定的技术或类型的代码和示例： 插调试、 调试、 内核模式驱动程序框架和 RPC 调试。

# <a name="bug-check-0x5d-unsupportedprocessor"></a>Bug 检查 0x5D：不支持\_处理器


不支持\_处理器错误检查的值为 0x0000005D。 这表示在计算机尝试在不受支持的处理器上运行 Windows。

> [!IMPORTANT]
> 本主题面向程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://support.microsoft.com/help/14238/windows-10-troubleshoot-blue-screen-errors)。


## <a name="unsupportedprocessor-parameters"></a>不支持\_处理器参数


无

<a name="cause"></a>原因
-----

Windows 需要比使用 higher-grade 处理器。

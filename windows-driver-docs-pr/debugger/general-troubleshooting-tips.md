# <a name="general-tips-for-blue-screens"></a>有关蓝屏的一般提示


如果您的计算机停止工作，并且显示蓝屏，计算机已关闭突然保护自己免受数据丢失。 硬件设备、 其驱动程序或相关的软件可能会导致此错误。 如果计算机附带的 Windows 副本，您可能想要与您的计算机制造商联系。

如果你有经验的计算机，并想要尝试从此错误中恢复，请按照中所述的步骤[疑难解答蓝屏错误](https://go.microsoft.com/fwlink/p/?linkid=183646)。

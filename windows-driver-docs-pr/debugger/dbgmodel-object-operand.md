# <a name="operand-objects"></a>操作数的对象 
## <a name="summary"></a>摘要
由操作数的对象描述了各个计算机指令的操作数。 对应字符串将是操作数的整个指令的字符串转换的一部分。 
## <a name="object-properties"></a>对象属性
|名称|描述|
|--- |--- |
|属性|[操作数属性](dbgmodel-object-operand-attributes.md)描述操作数的某些方面对象。|
|注册|一个[集合](dbgmodel-namespace-collections.md)的[注册](dbgmodel-object-register.md)描述操作数使用的所有注册的对象。|

# <a name="bug-check-0x120-bitlockerfatalerror"></a>Bug 检查 0x120：BITLOCKER\_致命错误\_错误


BITLOCKER\_致命错误\_错误 bug 检查的值为 0x00000120。 这表示 BitLocker 驱动器加密时遇到问题并且无法恢复。

> [!IMPORTANT]
> 本主题面向程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://support.microsoft.com/help/14238/windows-10-troubleshoot-blue-screen-errors)。


## <a name="bitlockerfatalerror-parameters"></a>BITLOCKER\_致命错误\_错误参数


| 参数 | 描述     |
|-----------|-----------------|
| 1         | 类型的问题 |
| 2         | 保留        |
| 3         | 保留        |
| 4         | 保留        |

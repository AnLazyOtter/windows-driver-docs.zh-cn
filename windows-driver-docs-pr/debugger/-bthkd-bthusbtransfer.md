# <a name="bthkdbthusbtransfer"></a>!bthkd.bthusbtransfer


**！ Bthkd.bthusbtransfer**命令显示蓝牙 usb 传输上下文包括 Irp、 Bip 和传输缓冲区信息。

```dbgsyntax
!bthkd.bthusbtransfer addr 
```

<span id="_______addr______"></span><span id="_______ADDR______"></span> *addr*   
蓝牙 USB 传输上下文的地址。

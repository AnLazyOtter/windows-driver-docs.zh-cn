# <a name="windbg-preview---installation"></a>WinDbg 预览版 - 安装


## <a name="installation"></a>安装

本部分介绍如何安装 WinDbg 预览调试器。

Microsoft Store 中提供了 WinDbg 预览调试器。 它需要安装 Windows 10 周年更新。 若要安装它，请打开 Microsoft Store 并搜索 "WinDbg Preview"，或单击[此处](
https://www.microsoft.com/store/apps/9pgjgd53tn86)。

在应用程序处于找到后，单击它以下载并安装。


## <a name="checking-for-updates"></a>检查更新

1. 打开应用商店应用，并单击 "搜索" 框旁边的帐户图片。 

2. 单击 "**下载和更新**" 以检查更新。

3. 在 "下载和更新" 页上，选择 "**获取更新**"。


## <a name="debugger-coexistence"></a>调试器共存  

通过在同一台计算机上使用经典的 WinDbg 调试器共存，可以同时使用这两个版本。

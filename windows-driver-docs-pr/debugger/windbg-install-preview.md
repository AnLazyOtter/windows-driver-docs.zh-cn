# <a name="windbg-preview---installation"></a>WinDbg 预览版 - 安装


## <a name="installation"></a>安装

本部分介绍如何安装 WinDbg 预览调试程序。

WinDbg 预览调试器目前在 Microsoft Store。 它需要 Windows 10 周年更新安装。 若要安装它，请打开 Microsoft Store 并搜索"WinDbg 预览"，或单击[此处](
https://www.microsoft.com/store/apps/9pgjgd53tn86)。

位于应用程序后，单击它以下载并安装。


## <a name="checking-for-updates"></a>检查更新

1. 打开应用商店应用程序，然后单击搜索框旁边将帐户图片。 

2. 单击**下载和更新**检查更新。

3. 在下载和更新页中，选择**获取更新**。


## <a name="debugger-coexistence"></a>调试器共存  

WinDbg 预览与在同一台计算机上的经典 WinDbg 调试器共存，因此可以使用这两个版本在同一时间。

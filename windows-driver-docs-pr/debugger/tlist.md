# <a name="tlist"></a>TList


## <span id="ddk_tlist_dtools"></span><span id="DDK_TLIST_DTOOLS"></span>


TList （任务列表查看器中），Tlist.exe，是一个命令行工具，将显示有关每个进程的有用信息以及在本地计算机上运行的进程。

TList 显示：

-   在该计算机，以及其进程 Id (Pid) 上运行的所有进程。

-   显示哪些进程创建的每个进程树。

-   进程，包括其虚拟内存的使用和启动该进程的命令的详细信息。

-   每个进程 （包括其线程 Id、 入口点、 上次报告的错误和线程状态） 中运行的线程。

-   在每个进程，包括版本号、 属性和虚拟地址的模块中运行的模块。

搜索按名称或 PID，进程或查找已加载指定的模块的所有进程，可以使用 TList。

在 Windows XP 和更高版本的 Windows，TList 已替换为 TaskList (Tasklist.exe) 帮助和支持中所述的那些系统。 TList 包含在调试工具的 Windows 支持用户不具有到任务列表的访问权限。

# <a name="pars"></a>!pars


**！ 分析**扩展显示指定的处理器应用程序注册文件。

```dbgcmd
!pars Address
```

**重要**  此命令已被 Windows 调试器版本 10.0.14257 中不推荐使用和更高版本，并不再可用。

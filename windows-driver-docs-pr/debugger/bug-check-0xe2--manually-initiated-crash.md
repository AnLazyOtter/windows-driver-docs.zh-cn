# <a name="bug-check-0xe2-manuallyinitiatedcrash"></a>Bug 检查 0xE2：手动\_启动\_崩溃


手动\_启动\_崩溃 bug 检查的值为 0x000000E2。 这表示用户谨慎地启动内核调试程序或键盘的故障转储。

**重要**本主题适用于程序员。 如果你已使用计算机时收到一个蓝色的屏幕，错误代码的客户，请参阅[疑难解答蓝屏错误](https://windows.microsoft.com/windows-10/troubleshoot-blue-screen-errors)。

## <a name="manuallyinitiatedcrash-parameters"></a>手动\_启动\_崩溃参数


无

<a name="remarks"></a>备注
-------

有关手动启动故障转储的详细信息，请参阅强制系统崩溃。 .

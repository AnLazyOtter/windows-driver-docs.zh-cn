# <a name="showreadfailures"></a>.show\_read\_failures


**.Show\_读取\_故障**命令启用或禁用读取失败的显示。

```dbgcmd
.show_read_failures /v 
.show_sym_failures /V
```

<span id="________v______"></span><span id="________V______"></span> **/v**   
可用读取失败的显示。

<span id="________V______"></span><span id="________v______"></span> **/V**   
禁止显示读取失败。

# <a name="debugging-tools-for-windows-new-for-windows-8"></a>Windows 调试工具：Windows 8 的新增功能


从 Windows 8 开始，可以开发、 生成和调试 Microsoft Visual Studio 中的所有内核模式和用户模式组件。 新增对通过网络连接或 USB 3.0 连接和改进了对调试优化的代码和内联函数支持调试的支持。 有关详细信息，请参阅以下主题：

-   [使用 Visual Studio 调试](debugging-using-visual-studio.md)
-   [手动设置的网络连接](setting-up-a-network-debugging-connection.md)
-   [USB 3.0 连接手动设置](setting-up-a-usb-3-0-debug-cable-connection.md)
-   [调试优化的代码或内联函数](debugging-optimized-code-and-inline-functions-external.md)

两个新的调试器扩展命令集将包含：

-   [USB 3.0 扩展](usb-3-extensions.md)
-   [RCDRKD 扩展](rcdrkd-extensions.md)

除了 Visual Studio 中，你可以 sse Windows 调试器来调试 Windows 应用。 有关 Windows 调试工具包包括， [ **PLMDebug.exe**](plmdebug.md)，这样您就可以手动控制的挂起、 继续、 调试和终止 Windows 应用。

Sos.dll 是用于调试托管的代码的组件。 Windows 8 调试工具的 Windows 包不包括 sos.dll。 若要了解如何获取 sos.dll，请参阅[获取 SOS 调试扩展 (sos.dll)](debugging-managed-code.md#getting-the-sos-debugging-extension)。

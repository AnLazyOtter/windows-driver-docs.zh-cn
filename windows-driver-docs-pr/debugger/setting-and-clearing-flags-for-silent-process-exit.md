# <a name="configuring-silent-process-exit-monitoring"></a>配置静默进程退出监视


从 Windows 7 开始，你可以使用**进程退出，无提示**选项卡来启用和配置监视的无提示退出进程。

## <span id="ddk_setting_and_clearing_image_file_flags_dtools"></span><span id="DDK_SETTING_AND_CLEARING_IMAGE_FILE_FLAGS_DTOOLS"></span>


在中指定的设置**无提示的进程退出**选项卡保存在注册表和保持有效，直到更改它们。

**若要启用和配置无提示的进程退出监视**

1.  单击**无提示的进程退出**选项卡。

    下面的屏幕截图所示**无提示的进程退出**Windows 8 中的选项卡。

    ![windows vista 中的图像文件选项卡的屏幕截图 ](images/gflagssilentprocessexit01.png)

2.  在中**图像**框，键入的可执行文件，其中包括文件扩展名，名称，然后按 TAB 键。

    这将激活上对应的复选框**无提示的进程退出**选项卡。

3.  指定您的首选项，通过选中或清除复选框和文本框中输入值。

4.  指定所有您的首选项，请单击**应用**。

# <a name="file--exit"></a>文件 | 退出


## <span id="ddk_file_exit_dbg"></span><span id="DDK_FILE_EXIT_DBG"></span>


单击**退出**上**文件**菜单以结束调试会话并退出 WinDbg。

此命令相当于按 alt+f4。

### <a name="span-idadditionalinformationspanspan-idadditionalinformationspanadditional-information"></a><span id="additional_information"></span><span id="ADDITIONAL_INFORMATION"></span>其他信息

有关退出 WinDbg 或结束调试会话的详细信息，请参阅[结束调试会话在 WinDbg 中](ending-a-debugging-session-in-windbg.md)。

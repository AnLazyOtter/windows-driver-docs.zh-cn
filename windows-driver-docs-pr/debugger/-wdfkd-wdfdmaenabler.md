# <a name="wdfkdwdfdmaenabler"></a>!wdfkd.wdfdmaenabler


**！ Wdfkd.wdfdmaenabler**扩展显示有关对 WDF 直接内存访问 (DMA) 对象，其事务和公共缓冲区对象的信息。

```dbgcmd
!wdfkd.wdfdmaenabler Handle
```

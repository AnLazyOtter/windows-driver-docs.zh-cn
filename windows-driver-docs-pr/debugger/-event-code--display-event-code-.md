# <a name="eventcode-display-event-code"></a>后面\_代码 （显示事件的代码）


**后面\_代码**命令显示当前事件说明进行操作。

```dbgcmd
.event_code 
```

<a name="remarks"></a>备注
-------

**后面\_代码**命令将显示在当前事件的指令指针的十六进制的说明进行操作。 如果有，显示包括最多 64 个字节的说明。

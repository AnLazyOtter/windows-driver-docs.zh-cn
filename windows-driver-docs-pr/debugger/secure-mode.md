# <a name="secure-mode"></a>安全模式


## <span id="ddk_secure_mode_dbg"></span><span id="DDK_SECURE_MODE_DBG"></span>


当您在执行内核模式调试时，可以在运行调试器*安全模式下*。 这使调试器无法影响主机计算机，但不会显著降低其自由地调试目标计算机。

如果想要允许远程客户端加入你的调试会话，建议使用安全模式。

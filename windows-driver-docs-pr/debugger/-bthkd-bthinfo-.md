# <a name="bthkdbthinfo"></a>!bthkd.bthinfo


**！ Bthkd.bthinfo**命令显示有关 BTHPORT FDO 的详细信息。 此命令是蓝牙调查的良好起点，因为它显示可用于访问的许多其他蓝牙调试扩展命令的地址信息。

```dbgsyntax
!bthkd.bthinfo 
```

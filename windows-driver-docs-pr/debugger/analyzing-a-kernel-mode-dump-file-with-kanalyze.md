# <a name="analyzing-a-kernel-mode-dump-file-with-kanalyze"></a>使用 KAnalyze 分析内核模式转储文件


## <span id="ddk_analyzing_a_kernel_mode_dump_file_with_kanalyze_dbg"></span><span id="DDK_ANALYZING_A_KERNEL_MODE_DUMP_FILE_WITH_KANALYZE_DBG"></span>


内核内存空间分析器 （KAnalyze，Kanalyze.exe） 是另一个工具，可以检查内核模式转储文件。

KAnalyze 和及其说明文档是 OEM 支持工具程序包的一部分。

若要下载这些工具，请转到[Microsoft 支持文章 253066](https://go.microsoft.com/fwlink/p/?linkid=241339)并按照该页上的说明。

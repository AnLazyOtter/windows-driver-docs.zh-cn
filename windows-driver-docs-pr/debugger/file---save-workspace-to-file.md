# <a name="file--save-workspace-to-file"></a>文件 |将工作区保存到文件


## <span id="ddk_file_save_workspace_to_file_dbg"></span><span id="DDK_FILE_SAVE_WORKSPACE_TO_FILE_DBG"></span>


单击**保存到文件的工作区**上**文件**菜单保存到文件的当前工作区。

### <a name="span-iddialogboxspanspan-iddialogboxspandialog-box"></a><span id="dialog_box"></span><span id="DIALOG_BOX"></span>对话框

当您单击**保存到文件的工作区**，则**保存到文件的工作区**对话框随即出现。 在此对话框中，输入你想要为工作区另存文件的名称。 然后，使用**将保存在**列表以导航到要保存该文件的目录或选择你想要覆盖特定文件。 （默认的文件扩展名是.wew。）

单击**确定**以保存该文件，或单击**取消**退出。

### <a name="span-idadditionalinformationspanspan-idadditionalinformationspanadditional-information"></a><span id="additional_information"></span><span id="ADDITIONAL_INFORMATION"></span>其他信息

有关不同级别的工作区和如何使用它们的详细信息，请参阅[使用工作区](using-workspaces.md)。

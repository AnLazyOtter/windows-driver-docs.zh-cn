# <a name="debugging-tools-for-windows-new-for-windows-81"></a>Windows 调试工具：Windows 8.1 的新增功能


对于 Windows 8.1 的 Windows 调试工具包括这些新功能。

-   [GPIO 扩展](gpio-extensions.md)
-   [HID 的扩展](hid-extensions.md)
-   大部分[内核模式驱动程序框架扩展命令](kernel-mode-driver-framework-extensions--wdfkd-dll-.md)现在使用与 KMDF 或 UMDF 2。 某些命令 (例如[ **！ wdfkd.wdfumdevstacks**](-wdfkd-wdfumdevstacks.md)) 专门支持 UMDF 2 已添加到此集。
-   现在可以在 CAB 文件和内存转储文件中包含分页文件。 请参阅[CAB 文件，其中包含分页文件以及内存转储](cab-files-that-contain-paging-files-along-with-a-memory-dump.md)

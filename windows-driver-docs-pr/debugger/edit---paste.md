# <a name="edit--paste"></a>编辑 |粘贴


## <span id="ddk_edit_paste_dbg"></span><span id="DDK_EDIT_PASTE_DBG"></span>


单击**粘贴**上**编辑**菜单粘贴剪贴板中当前光标位置到的文本。

此命令相当于按 CTRL + V 或 SHIFT + INSERT 或单击**粘贴 (Ctrl + V)** 按钮 (![粘贴按钮的屏幕截图](images/tbpaste.png)) 工具栏上。

可以使用**粘贴**命令仅使用停靠或选项卡式窗口。 但您可以与支持此功能的任何窗口使用键盘快捷方式和工具栏按钮。

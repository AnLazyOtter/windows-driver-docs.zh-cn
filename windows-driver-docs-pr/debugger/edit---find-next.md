# <a name="edit--find-next"></a>编辑 |查找下一个


## <span id="ddk_edit_find_next_dbg"></span><span id="DDK_EDIT_FIND_NEXT_DBG"></span>


单击**查找下一步**上**编辑**菜单可重复执行上一个搜索和查找下一个匹配项。

此命令相当于按 f3 键。

若要重复执行向后的搜索，请按 SHIFT + F3。

如果以前未执行任何搜索，使用**编辑 |查找下一步**命令，请按 f3 键，或按 SHIFT + F3 以打开**查找**对话框中 (类似于[编辑 |查找](edit---find.md)命令)。

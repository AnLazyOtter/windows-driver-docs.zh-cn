# <a name="ending-a-debugging-session-in-visual-studio"></a>在 Visual Studio 中的调试会话结束

> [!IMPORTANT]
> 此功能不在 Windows 10，版本 1507年和更高版本的 WDK 中可用。
>

若要结束调试会话在 Microsoft Visual Studio 中，从**调试**菜单中，选择**停止调试**。

# <a name="file--recent-files"></a>文件 | 最近使用的文件


## <span id="ddk_file_recent_files_dbg"></span><span id="DDK_FILE_RECENT_FILES_DBG"></span>


指向**最近使用的文件**上**文件**菜单可显示在 WinDbg 中最近打开的四个源文件的列表。

若要打开这些文件之一，请单击其名称从菜单。

# <a name="h"></a>H


<span id="host"></span><span id="HOST"></span>**host**  
查看主机的计算机。

<span id="handling-status"></span><span id="HANDLING_STATUS"></span>**处理状态**  
处理状态指定为已处理或不调试器引擎是否应该标记异常事件。 处理状态是异常筛选器的一部分。

另请参阅中断状态。 有关详细信息，请参阅[控制异常和事件](controlling-exceptions-and-events.md)和[事件筛选器](event-filters.md)

<span id="host_computer"></span><span id="HOST_COMPUTER"></span>**主计算机**  
在主计算机是运行调试会话的计算机。 所有调试器操作，例如，执行命令并扩展和符号加载-会在主计算机上执行。

在典型的两个系统内核，调试会话，调试器在主计算机连接到正在调试的目标计算机上运行。

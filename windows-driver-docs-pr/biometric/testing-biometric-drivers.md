# <a name="testing-biometric-drivers"></a>测试生物识别驱动程序


Microsoft 建议所有驱动程序 WDK 中运行的静态分析工具。 有关详细信息，请参阅[通过使用代码分析工具来分析驱动程序质量](https://docs.microsoft.com/windows-hardware/drivers)。

此外使用运行时工具，例如[WDF 验证程序控件应用程序](https://docs.microsoft.com/windows-hardware/drivers/devtest/wdf-verifier-control-application)并[应用程序验证工具](https://docs.microsoft.com/windows-hardware/drivers/devtest/application-verifier)。

下载[Windows 硬件认证工具包 (HCK)](https://go.microsoft.com/fwlink/p/?LinkId=733613)工具来测试 Windows 生物识别框架使用的驱动程序包。

# <a name="verification-engine"></a>验证引擎


SDV*验证引擎*是一个功能强大的证明引擎，为 SDV 执行规则验证。

验证引擎采用以下输入：

-   驱动程序代码

-   [驱动程序验证程序的静态规则](static-driver-verifier-rule.md)

-   Static Driver Verifier[操作系统模型](operating-system-model.md)

# <a name="powerdownfail-rule-wdm"></a>PowerDownFail 规则 (wdm)


**PowerDownFail**规则指定 FDO 或 FIDO 驱动程序不会失败 IRP\_MN\_设置\_POWER 请求时关闭设备。 此规则仅适用于 FDO 和 FIDO 驱动程序。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

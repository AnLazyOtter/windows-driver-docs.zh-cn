# <a name="computerhardwareids"></a>ComputerHardwareIds


ComputerHardwareIds (ComputerHardwareIds.exe) 是一个命令行工具，可用于生成特定模型的供应商的计算机系统的硬件 Id。 供应商可以使用这些硬件 Id 来匹配到该特定计算机型号的设备元数据包。

运行 Windows 7 或更高版本的 Windows 操作系统的系统上支持 ComputerHardwareIds 工具。 此工具包含在工具中 Windows Driver Kit (WDK)\\IOther 目录。 有用于 32 位 Windows 平台的单独版本 (工具\\其他\\x86) 和 64 位 Windows 平台 (工具\\其他\\ia64 和工具\\其他\\amd64)。

本部分包含以下主题：

[**ComputerHardwareIds 概述**](computerhardwareids-overview.md)

[ComputerHardwareIds 输出](computerhardwareids-output.md)

有关设备元数据包的详细信息，请参阅[设备元数据包](https://docs.microsoft.com/windows-hardware/drivers/install/device-metadata-packages)。

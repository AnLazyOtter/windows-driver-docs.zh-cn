# <a name="adding-a-rule"></a>添加规则


若要将规则添加到筛选器：

1.  在中[跟踪会话列表](trace-session-list.md)，在会话中，右键单击任一单元，然后单击**管理筛选器**。

2.  在中**管理筛选器**对话框底部的框中，单击**新规则**。

3.  在中**到修改规则或添加**区域中，选择[的规则元素](filter-rule-elements.md)。

4.  单击**添加规则**。

如果使用的[工作区](using-traceview-workspaces.md)，你必须[重新保存工作区](saving-or-resaving-a-workspace.md)后添加规则。 否则，该规则不添加到工作区中。

# <a name="stopping-and-removing-a-workspace-session"></a>停止和删除工作区会话

在打开工作区，启动新的跟踪会话或新的日志显示。 停止会话和显示中删除它们，根据需要对任何跟踪会话使用相同的过程。

### <a name="span-idtostopaworkspacesessionspanspan-idtostopaworkspacesessionspanto-stop-a-workspace-session"></a><span id="to_stop_a_workspace_session"></span><span id="TO_STOP_A_WORKSPACE_SESSION"></span>若要停止某一工作区会话

1.  在中[跟踪会话列表](trace-session-list.md)、 右键单击任意单元格的行的表示跟踪会话或跟踪日志显示。

2.  单击**停止跟踪**。

### <a name="span-idtoremoveaworkspacesessionspanspan-idtoremoveaworkspacesessionspanto-remove-a-workspace-session"></a><span id="to_remove_a_workspace_session"></span><span id="TO_REMOVE_A_WORKSPACE_SESSION"></span>若要删除工作区会话

1.  停止工作区会话。

2.  在中[跟踪会话列表](trace-session-list.md)、 右键单击任意单元格的行的表示跟踪会话或跟踪已停止显示日志。

3.  单击**删除日志会话**。

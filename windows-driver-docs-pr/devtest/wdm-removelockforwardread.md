# <a name="removelockforwardread-rule-wdm"></a>RemoveLockForwardRead 规则 (wdm)


**RemoveLockForwardRead**规则验证的调用[ **IoAcquireRemoveLock** ](https://msdn.microsoft.com/library/windows/hardware/ff548204)并[ **IoReleaseRemoveLock**](https://msdn.microsoft.com/library/windows/hardware/ff549560)中正确使用。

|--------------|-----|

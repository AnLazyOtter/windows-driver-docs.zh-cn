# <a name="removelock-rule-wdm"></a>RemoveLock 规则 (wdm)


**RemoveLock**规则指定的调用[ **IoAcquireRemoveLock** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-ioacquireremovelock)并[ **IoReleaseRemoveLock** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-ioreleaseremovelock)正确使用。 此外，在末尾 IRP\_MJ\_PNP 或 IRP\_MJ\_POWER 例程，该驱动程序不应阻止**RemoveLock**。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

# <a name="devicecreatefail-rule-kmdf"></a>DeviceCreateFail 规则 (kmdf)


**DeviceCreateFail**规则指定的 EVT\_WDF\_驱动程序\_设备\_添加返回的错误状态时调用[ **WdfDeviceCreate** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdfdevice/nf-wdfdevice-wdfdevicecreate)失败。

对于服务设备的驱动程序，必须已成功创建设备对象。

|              |      |
|--------------|------|
| 驱动程序模型 | KMDF |

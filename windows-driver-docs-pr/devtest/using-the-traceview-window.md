本部分介绍 TraceView 窗口，并说明如何使用它。

TraceView 窗口可以包含一个**跟踪会话列表**和多个**跟踪消息列出了**。 列表会出现在独立的窗格，您可以移动和排列方式抓取上每个窗格的左边缘双竖线并将它移动到新位置。

# <a name="traceview-overview"></a>TraceView 概述

TraceView 旨在替换所有的函数执行的一种工具的许多传统、 单函数命令行跟踪工具[跟踪控制器](trace-controller.md)和一个[跟踪使用者](trace-consumer.md)中方便的图形用户界面。

使用 TraceView，可以启动跟踪会话，并从实时跟踪会话和跟踪日志的跟踪消息会立即显示，而无需单独的格式设置步骤。 它可以提取所有所需的提供程序，包括控制 Guid 有关的信息并跟踪的消息格式数据，从提供程序的 PDB 符号文件，消除了几个步骤。

TraceView 窗口是易于使用，并释放你正在查找或记住的命令行参数。

TraceView 显示中的跟踪消息[跟踪消息列表](trace-message-lists.md)窗口中，以便可以查看、 筛选和排序轻松地跟踪消息。 并且，可以直接从窗口中复制的跟踪消息并将其粘贴到文本文件、 电子表格文件或数据库文件进行保存或以后处理。

有关 TraceView 功能的详细信息，请参阅[使用 TraceView](using-traceview.md)。

# <a name="irqlinterruptfunction-rule-ndis"></a>Irql\_中断\_函数规则 (ndis)


Irql\_中断\_函数规则指定必须在正确的 IRQL 级别上名为中断的 NDIS 函数。

此规则验证以下 NDIS 函数：

**NdisMDeregisterInterruptEx**
**NdisMRegisterInterruptEx**

|              |      |
|--------------|------|
| 驱动程序模型 | NDIS |

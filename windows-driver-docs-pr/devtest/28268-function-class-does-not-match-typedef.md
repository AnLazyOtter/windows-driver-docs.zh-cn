# <a name="c28268"></a>C28268


警告 C28268:针对函数的函数类与此处使用的 typedef 上的函数类不匹配

此警告意味着函数类批注与函数类型，指定的类型声明不匹配。 此警告意味着批注中而不所分析的代码存在错误。

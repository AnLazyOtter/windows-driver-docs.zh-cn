# <a name="restarting-a-trace-session"></a>重启跟踪会话

当创建跟踪会话时，TraceView 将自动启用的跟踪提供程序，并启动跟踪会话。 可以使用**启动跟踪**选项可以重新启动跟踪会话，您[停止](stopping-a-trace-session.md)。 若要重新启动已停止的跟踪会话，请执行以下操作：

1.  在中[跟踪会话列表](trace-session-list.md)，用鼠标右键单击行的任何单元格，为跟踪会话。

2.  单击**启动跟踪**。

短暂暂停的值之后**状态**列会更改从**已停止**到**运行**。

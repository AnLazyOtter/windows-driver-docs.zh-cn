# <a name="ksirqlddis-rule-"></a>KsIrqlDDIs 规则 （)


KsIrqlDDIs 规则指定，内核流式处理 (KS) 微型端口驱动程序调用 KS DDIs 在正确的 IRQL 级别。

|              |     |
|--------------|-----|
| 驱动程序模型 | KS  |

|                                   |                                                                                                                                       |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| 使用此规则发现的错误检查 | [**Bug 检查 0xC4:驱动程序\_VERIFIER\_已检测\_冲突**](https://docs.microsoft.com/windows-hardware/drivers/debugger/bug-check-0xc4--driver-verifier-detected-violation) (0x00081009) |

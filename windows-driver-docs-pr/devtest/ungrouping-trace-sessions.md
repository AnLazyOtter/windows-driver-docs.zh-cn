# <a name="ungrouping-trace-sessions"></a>将跟踪会话取消分组


若要取消分组的跟踪会话组：

1.  [停止所有正在运行的跟踪会话](stopping-a-trace-session.md)组中。

2.  在中[跟踪会话列表](trace-session-list.md)，右键单击分组的会话中的任意单元格。

3.  单击**取消分组会话**。

如果取消停止的跟踪会话，他们开始再次运行。 若要删除的跟踪会话，停止它们，然后[将其删除](removing-a-trace-session.md)。

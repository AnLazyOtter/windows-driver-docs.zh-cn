# <a name="pdodeviceinitapi-rule-kmdf"></a>PdoDeviceInitAPI 规则 (kmdf)


**PdoDeviceInitAPI**规则指定[ **WdfPdoInitAllocate** ](https://msdn.microsoft.com/library/windows/hardware/ff548786)和所有其他设备对象初始化设置 DDIs [ **WDFDEVICE\_INIT** ](https://msdn.microsoft.com/library/windows/hardware/ff546951)结构的物理设备对象 (PDO) 必须在调用之前驱动程序调用[ **WdfDeviceCreate** ](https://msdn.microsoft.com/library/windows/hardware/ff545926)对 PDO。

|              |      |
|--------------|------|
| 驱动程序模型 | KMDF |

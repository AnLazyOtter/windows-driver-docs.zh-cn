# <a name="using-the-device-metadata-authoring-wizard"></a>使用设备元数据创作向导


\[本主题介绍提供在 Windows Driver Kit (WDK) 8 的设备元数据创作工具。 如果要开发 Windows 8.1 的设备体验，使用设备元数据创建向导附带[Microsoft Visual Studio 2013 和 Windows Driver Kit (WDK) 8.1](https://go.microsoft.com/fwlink/p/?LinkId=226411)。 有关详细信息，请参阅[Windows 8.1 设备体验](https://go.microsoft.com/fwlink/p/?linkid=325561)。 \]

## <a name="span-idgettingstartedspanspan-idgettingstartedspanspan-idgettingstartedspangetting-started"></a><span id="Getting_started"></span><span id="getting_started"></span><span id="GETTING_STARTED"></span>入门


启动后**创作**工具**欢迎使用元数据创建向导**屏幕出现。 选择以下选项之一：

-   若要创建新的设备元数据包，请选择**创建新的设备元数据包**，然后单击**下一步**。
-   若要更新现有的设备元数据包，请选择**更新现有设备元数据包**，单击**浏览**以选择该包，然后单击**下一步**。
    **请注意**  不能编辑 Device Stage 体验的工具。 但是，如果使用此工具更新包不会更改设备阶段文件夹。 有关设备阶段的详细信息，请参阅[Windows 7 设备体验](https://go.microsoft.com/fwlink/p/?linkid=221801)。

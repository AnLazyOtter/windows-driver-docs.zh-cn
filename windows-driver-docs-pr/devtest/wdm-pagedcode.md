# <a name="pagedcode-rule-wdm"></a>PagedCode 规则 (wdm)


**PagedCode**规则指定驱动程序调用[ **PAGED\_代码**](https://docs.microsoft.com/windows-hardware/drivers/kernel/mm-bad-pointer)宏仅当它在执行时**IRQL &lt;= APC\_级别**。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

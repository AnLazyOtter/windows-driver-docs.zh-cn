# <a name="multremovelock-rule-wdm"></a>MultRemoveLock 规则（wdm）


**MultRemoveLock**规则验证只通过一个唯一的删除锁定来调用[**IoAcquireRemoveLock**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-ioacquireremovelock) 。 这是一个警告规则。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

# <a name="pnpcpu-general-notes"></a>PNPCPU 常规说明


PNPCPU 目录应包含：

-   BUS.INF

-   BUSENUM.SYS

-   DELTA.CAT

-   PNPCPU.EXE

-   PNPCPU.SYS

-   处理器。INF

虽然是必需的没有特定的路径的设置所需运行 PNPCPU 工具中，从包含所有上面列出的文件的目录运行 Pnpcpu.exe。

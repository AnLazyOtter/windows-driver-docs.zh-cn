# <a name="spinlock-rule-wdm"></a>旋转锁规则 (wdm)


**SpinLock**规则指定的在调用[ **KeAcquireSpinLock**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-keacquirespinlock)，驱动程序调用[ **KeReleaseSpinLock**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-kereleasespinlock)之前对后续调用**KeAcquireSpinLock**或设置为[ **KeAcquireSpinLockRaiseToDpc**](https://docs.microsoft.com/previous-versions/windows/hardware/drivers/ff551928(v=vs.85))。

如果它们是获取和释放锁不同资源的，允许嵌套的调用。 获取或释放的相同资源的锁嵌套的调用违反此规则。

此规则还指定驱动程序已使用[ **KeReleaseSpinLock** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-kereleasespinlock)解除调度例程之前的所有数值调节钮锁定或取消例程结束。

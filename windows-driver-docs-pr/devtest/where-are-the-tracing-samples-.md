# <a name="where-are-the-tracing-samples"></a>跟踪示例位于何处？


[TraceDrv](https://go.microsoft.com/fwlink/p/?LinkId=617726)是设计为演示 WPP 软件跟踪的示例驱动程序。 TraceDrv 现已推出[Windows 驱动程序示例](https://go.microsoft.com/fwlink/p/?LinkId=616507 )GitHub 上的存储库。

TraceDrv 示例还包括 TraceCtl，启动 TraceDrv 并使其生成跟踪消息的应用程序。

[Toaster](https://go.microsoft.com/fwlink/p/?linkid=256195)，在 WDK 中的常规示例驱动程序还针对检测[WPP 软件跟踪](wpp-software-tracing.md)。 此外，还可以从[Windows 硬件开发示例](https://go.microsoft.com/fwlink/p/?LinkId=618052)。

[Eventdrv](https://go.microsoft.com/fwlink/p/?LinkId=617724)是一个示例，演示如何实现[事件跟踪 Windows (ETW)](event-tracing-for-windows--etw-.md)驱动程序中。 ETW 内核模式 API 中引入了 Windows Vista 和更早的操作系统中不支持。

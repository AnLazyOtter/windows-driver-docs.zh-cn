# <a name="irqlrtlpassive-rule-wdm"></a>IrqlRtlPassive 规则 (wdm)


IrqlRtlPassive 规则指定驱动程序调用[ **RtlDeleteRegistryValue** ](https://msdn.microsoft.com/library/windows/hardware/ff561829)仅当执行在 IRQL = 被动\_级别。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

|                                   |                                                                                                                                       |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| 使用此规则发现的错误检查 | [**Bug 检查 0xC4:驱动程序\_VERIFIER\_已检测\_冲突**](https://msdn.microsoft.com/library/windows/hardware/ff560187) (0x0002001E) |

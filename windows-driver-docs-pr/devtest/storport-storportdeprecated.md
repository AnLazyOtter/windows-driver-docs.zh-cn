# <a name="storportdeprecated-rule-storport"></a>StorPortDeprecated 规则 (storport)


此规则验证，该驱动程序不会调用这些不推荐使用的例程之一：**StorPortValidateRange**或**StorPortLogError**。

|              |          |
|--------------|----------|
| 驱动程序模型 | Storport |

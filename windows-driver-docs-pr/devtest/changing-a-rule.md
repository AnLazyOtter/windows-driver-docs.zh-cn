# <a name="changing-a-rule"></a>更改规则


若要更改筛选器中的规则：

1.  在中[跟踪会话列表](trace-session-list.md)，在会话中，右键单击任一单元，然后单击**管理筛选器**。

2.  在中**管理筛选器**对话框中**当前规则**区域中，选择你想要更改的规则。

3.  若要更改在其中应用规则的顺序，请单击**移**或**下移**按钮。

4.  若要更改的规则，元素在**为已修改或已添加的规则**区域中，更改[的规则元素](filter-rule-elements.md)。

5.  单击**修改规则**。

如果你已保存中的规则[工作区](using-traceview-workspaces.md)，你必须[重新保存工作区](saving-or-resaving-a-workspace.md)后更改该规则。 否则，原始规则仍在工作区中。

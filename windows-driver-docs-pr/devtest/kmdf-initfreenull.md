# <a name="initfreenull-rule-kmdf"></a>InitFreeNull 规则（kmdf）


**InitFreeNull**规则指定 DDIS 接收 PWDFDEVICE\_init 作为参数，不能使用指向[WDFDEVICE\_INIT](https://docs.microsoft.com/windows-hardware/drivers/wdf/wdfdevice_init)结构的**NULL**指针调用。

框架提供的方法初始化[WDFDEVICE\_INIT](https://docs.microsoft.com/windows-hardware/drivers/wdf/wdfdevice_init)结构。 当驱动程序调用[**WdfDeviceCreate**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdevicecreate)来创建功能设备对象（FDO）或物理设备对象（PDO）时，如果成功，则**WdfDeviceCreate**方法会将第一个参数设置为**NULL** 。

如果驱动程序在调用设备对象初始化方法或[**WdfDeviceCreate**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdevicecreate)时遇到错误，则驱动程序必须调用 WdfDeviceInitFree。 成功调用 WdfDeviceInitFree 后，应将[WDFDEVICE\_INIT](https://docs.microsoft.com/windows-hardware/drivers/wdf/wdfdevice_init)结构的指针设置为**NULL** （PWDFDEVICE\_init =**null**）。

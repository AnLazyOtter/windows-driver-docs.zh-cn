# <a name="deleting-a-workspace"></a>删除工作区

删除工作区时，你可以从 TraceView 删除所有保存的跟踪会话属性和配置设置。

有关停止工作区会话或删除从显示的信息，请参阅[停止和删除工作区会话](stopping-and-removing-a-workspace-session.md)。

### <a name="span-idtodeleteaworkspacespanspan-idtodeleteaworkspacespanto-delete-a-workspace"></a><span id="to_delete_a_workspace"></span><span id="TO_DELETE_A_WORKSPACE"></span>若要删除工作区

1.  上**文件**菜单上，单击**删除工作区**。

2.  从**保存工作区**列表中，选择工作区的名称。

3.  单击 **“确定”**。

### <a name="span-idcommentspanspan-idcommentspancomment"></a><span id="comment"></span><span id="COMMENT"></span>注释

可以一次删除只有一个工作区。

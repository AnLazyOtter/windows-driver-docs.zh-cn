# <a name="wdf-ddi-fault-injection-tester-wdffitester"></a>WDF DDI 错误注入测试人员 (WdfFiTester)


WdfFiTester 使用 WMI 接口来配置错误注入到 KMDF 设备驱动程序接口 (DDI) 函数调用。 WdfFiTester 测试 KMDF 驱动程序能够正确地处理这些插入的失败。

本部分包括以下主题：

[WdfFiTester 概述](wdffitester-overview.md)

[WdfTester 安装](wdftester-installation.md)

[启用故障注入](enabling-fault-injection.md)

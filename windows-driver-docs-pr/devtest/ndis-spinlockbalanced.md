# <a name="spinlockbalanced-rule-ndis"></a>SpinLockBalanced 规则（ndis）


**SpinLockBalanced**规则验证对获取旋转锁的函数的调用次数是否等于释放同一旋转锁的函数的调用次数。

|              |      |
|--------------|------|
| 驱动程序型号 | 以此 |

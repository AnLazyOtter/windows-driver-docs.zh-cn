# <a name="using-sdv-and-the-sample-drivers"></a>使用 SDV 和示例驱动程序


若要帮助你开始使用 Static Driver Verifier (SDV) 可以运行该工具的 SDV FailDriver 示例。 示例驱动程序包含用于显示的功能和功能的 SDV 有意代码错误。 这些示例驱动程序不起作用，并且不应作为实际的驱动程序开发项目的示例。 SDV FailDriver 示例驱动程序中有[Windows 驱动程序示例](https://go.microsoft.com/fwlink/p/?LinkId=616507)GitHub 上的存储库。

-   [SDV-FailDriver-KMDF](https://go.microsoft.com/fwlink/p/?LinkId=617993)
-   [SDV-FailDriver-WDM](https://go.microsoft.com/fwlink/p/?LinkId=617999)
-   [SDV-FailDriver-NDIS](https://go.microsoft.com/fwlink/p/?LinkId=617995)
-   [SDV-FailDriver-STORPORT](https://go.microsoft.com/fwlink/p/?LinkId=617997)

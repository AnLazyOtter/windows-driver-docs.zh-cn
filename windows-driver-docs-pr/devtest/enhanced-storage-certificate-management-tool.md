# <a name="enhanced-storage-certificate-management-tool"></a>增强的存储证书管理工具


增强型存储证书管理工具 (EhStorCertMgrCmd.exe) 可管理符合 IEEE 1667 标准的 USB 存储设备上的证书。 此工具可通过以下组：

-   开发人员的 USB 存储设备进行测试和验证的 IEEE 1667 标准遵从性。

-   IT 管理员提供和管理的证书存储的 IEEE 1667 合规 USB 存储设备在企业环境中使用。

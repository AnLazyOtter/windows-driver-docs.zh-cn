# <a name="trace-consumer"></a>跟踪使用者


*跟踪使用者*是应用程序和接收来自实时跟踪会话的跟踪消息的工具和[跟踪日志](trace-log.md)，并设置消息格式，以便它们是可读。

[Tracefmt](tracefmt.md)是跟踪使用者。 [TraceView](traceview.md)既[跟踪控制器](trace-controller.md)和跟踪使用者。

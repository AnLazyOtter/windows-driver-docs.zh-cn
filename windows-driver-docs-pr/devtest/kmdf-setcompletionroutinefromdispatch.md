# <a name="setcompletionroutinefromdispatch-rule-kmdf"></a>SetCompletionRoutineFromDispatch 规则 (kmdf)


**SetCompletionRoutineFromDispatch**规则将验证该驱动程序未指定完成例程上从 IRP 其[ *EvtDeviceWdmIrpDispatch* ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdfdevice/nc-wdfdevice-evt_wdfdevice_wdm_irp_dispatch)回调函数。

|              |      |
|--------------|------|
| 驱动程序模型 | KMDF |

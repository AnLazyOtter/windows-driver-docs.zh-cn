# <a name="c28645"></a>C28645


警告 C28645:使用不再建议使用问号消息符号调用了 MessageBox

不再建议的模式对话框中的问号消息图标使用，因为它不清楚地表示特定类型的消息和的消息表述问题无法应用于任何消息类型。 此外，用户可以将问号消息符号与帮助信息混淆。

因此，不要在消息框中使用此问号消息符号。 系统继续支持它包含的内容仅为向后兼容性。

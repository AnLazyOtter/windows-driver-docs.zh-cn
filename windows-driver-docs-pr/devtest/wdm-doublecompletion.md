# <a name="doublecompletion-rule-wdm"></a>DoubleCompletion 规则 (wdm)


**DoubleCompletion (WDM)** 规则指定驱动程序必须调用[ **IoCompleteRequest** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-iocompleterequest)两次以相同的 IRP 例程。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

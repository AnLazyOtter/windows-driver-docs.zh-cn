# <a name="c28730"></a>C28730


警告 C28730:可能分配\\\\0 直接向指针。

此警告意味着可能拼写错误： **nul**字符要分配给指针; 它可能是这种情况，字符作为字符串终止符，并应分配给内存的指针指向。

**请注意**  Windows Driver Kit (WDK) 8 还包括其他调试工具：WSDDebug\_client.exe 和 WSDDebug\_host.exe。 如果您打开 Visual Studio 命令提示符窗口，可以在 %windowssdkdir%中找到这些工具\\bin\\*&lt;arch&gt;* 目录。 （这些工具还包含 Microsoft Windows SDK 中。）这些工具是软设备和客户端，可用于对设备或应用程序进行故障排除。 详细了解 WSDDebug\_客户端和 WSDDebug\_托管，请参阅[设备上的 Web 服务](https://go.microsoft.com/fwlink/p/?linkid=81407)Windows SDK 的一部分。

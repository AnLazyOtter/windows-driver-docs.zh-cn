# <a name="using-the-tracing-flags-and-level-selection-dialog-box"></a>使用跟踪标志和级别选择对话框


在中**跟踪标志和级别选择**对话框中， **G** (GUID) 图标表示的跟踪提供程序。 此图标的文件名称旁边将显示[跟踪消息控件 (.tmc) 文件](trace-message-control-file.md)跟踪提供程序。 若要选择提供程序的跟踪级别和跟踪标志，找到适当**G**图标，并执行以下：

-   若要显示或更改提供程序的跟踪级别，请执行以下操作：
    -   双击**级别**项的提供程序，然后选择你想要提供程序的级别。 （可以选择只有一个级别。）
    -   右键单击**级别**项，依次指向**级别**，然后单击你想要提供程序的级别。
-   若要显示或更改的提供程序的标志，请执行以下操作：
    -   双击**标志**项提供程序，然后选择或清除所需的标志的复选框。 （可以选择多个标志）。
    -   右键单击**标志**项，然后依次**设置所有标志**或**清除所有标志**。
-   若要保存所做的更改，请单击**确定**。

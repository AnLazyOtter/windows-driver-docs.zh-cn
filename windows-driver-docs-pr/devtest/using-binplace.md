# <a name="using-binplace"></a>使用 BinPlace


## <span id="ddk_using_binplace_tools"></span><span id="DDK_USING_BINPLACE_TOOLS"></span>


通过组合使用 BinPlace 命令行、 各种环境变量和名为的文件控制 BinPlace*位置文件*。 *命令行*是在命令提示符窗口中键入的实际行或批处理文件中包含的。 *环境变量*进行常量全局设置中 Microsoft Windows 或本地命令提示符窗口中。 *位置文件*是一个文本文件，其中提供了有关在何处放置的文件移动或创建 BinPlace 信息。

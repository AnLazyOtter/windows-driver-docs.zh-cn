# <a name="c28721"></a>C28721


警告 C28721:不推荐使用的性能计数器体系结构

使用已弃用的性能计数器体系结构： &lt;*症状*&gt;。

该代码使用旧的性能计数器体系结构来公开应用程序或组件的性能指标。 适用于 Windows Vista 引入了新的性能计数器体系结构。 应该迁移到新的体系结构除非让应用程序 （必须在 Windows Vista 和 Windows 的早期版本运行） 的同时进行低级别要求。

# <a name="irqlkesetevent-rule-wdm"></a>IrqlKeSetEvent 规则（wdm）


**IrqlKeSetEvent**规则指定在*Wait*设置为**FALSE***时，* [**KESETEVENT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-kesetevent)例程仅在 irql &lt;= 调度\_级别进行调用，而在 irql &lt;= APC\_级别设置为**TRUE**。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

|                                   |                                                                                                                                       |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| 使用此规则发现的错误检查 | [**Bug 检查0xC4：检测到\_冲突的驱动程序\_验证程序\_** ](https://docs.microsoft.com/windows-hardware/drivers/debugger/bug-check-0xc4--driver-verifier-detected-violation) （0x00020016） |

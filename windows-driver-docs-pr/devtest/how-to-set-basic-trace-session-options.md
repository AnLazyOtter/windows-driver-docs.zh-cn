# <a name="how-to-set-basic-trace-session-options"></a>如何设置基本跟踪会话选项


使用此过程创建跟踪会话时设置基本跟踪会话选项。

1.  上**文件**菜单上，单击**新建日志会话**。

2.  单击**添加提供程序**，然后选择一个提供程序。 如有必要指定 TMF 文件或 TMF 文件路径。

3.  若要添加其他提供程序，请单击**添加提供程序**。 否则，请单击**下一步**。

4.  上**日志会话选项**页上，设置会话选项的基本的跟踪。

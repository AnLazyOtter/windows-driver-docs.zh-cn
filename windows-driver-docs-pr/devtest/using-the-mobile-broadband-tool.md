# <a name="using-the-mobile-broadband-metadata-authoring-wizard"></a>使用创作向导移动宽带元数据


\[本主题介绍提供在 Windows Driver Kit (WDK) 8 的设备元数据创作工具。 如果要开发 Windows 8.1 的设备体验，使用设备元数据创建向导附带[Microsoft Visual Studio 2013 和 Windows Driver Kit (WDK) 8.1](https://go.microsoft.com/fwlink/p/?LinkId=226411)。 有关详细信息，请参阅[Windows 8.1 设备体验](https://go.microsoft.com/fwlink/p/?linkid=325561)。 \]

启动后**移动宽带**工具**欢迎使用元数据创建向导**屏幕出现。 选择以下选项之一：

-   若要创建新的元数据包，请选择**创建新的设备元数据包**，然后单击**下一步**。
-   若要更新现有元数据的包，请选择**更新现有设备元数据包**，单击**浏览**以选择该包，然后单击**下一步**。

有关开发移动运营商应用程序的详细信息，请参阅[移动宽带：UWP 设备应用](https://go.microsoft.com/fwlink/p/?linkid=241446)。

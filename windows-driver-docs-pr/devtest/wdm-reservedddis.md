# <a name="reservedddis-rule-wdm"></a>ReservedDDIs 规则 (wdm)


**ReservedDDIs**规则验证驱动程序不会调用任何保留的函数。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

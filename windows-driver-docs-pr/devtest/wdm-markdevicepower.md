# <a name="markdevicepower-rule-wdm"></a>MarkDevicePower 规则 (wdm)


**MarkDevicePower**规则指定 IRP\_MJ\_与 IRP POWER\_MN\_设置\_都会被挂起，转到 S0 SystemPowerState IRP 的电源。

此规则仅适用于 FDO 和 FIDO 驱动程序。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

# <a name="opening-a-workspace"></a>打开工作区

当跟踪会话打开一个工作区时，TraceView 启动新的跟踪会话与保存的工作区中的属性。 当您打开跟踪日志显示的工作区时，显示完全一样配置它。

### <a name="span-idtoopenaworkspacespanspan-idtoopenaworkspacespanto-open-a-workspace"></a><span id="to_open_a_workspace"></span><span id="TO_OPEN_A_WORKSPACE"></span>若要打开的工作区

1.  上**文件**菜单上，单击**打开工作区**。

2.  从**保存工作区**列表中，选择工作区的名称。

    或者，在**新的工作区**框中，键入工作区的名称。

3.  单击 **“确定”**。

由于工作区中保存的路径和文件名与跟踪会话相关联，这些文件是重复使用，并且覆盖每次打开工作区。 若要保留您的文件，移动或重命名它们后每个跟踪会话。

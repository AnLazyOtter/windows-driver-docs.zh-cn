# <a name="c28736"></a>C28736

警告 C28736:已禁止的 API 参数用法

此警告指示正在使用旧的事件日志或其他跟踪 API 或 Perflib 使用者 API，使用已禁止自变量。 这些 Api 和自变量已替换为 Windows 的事件跟踪模型，从而提高性能和安全性。

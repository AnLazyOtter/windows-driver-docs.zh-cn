# <a name="changing-the-text-size-in-the-defect-viewer"></a>更改缺陷查看器中的文本大小


若要从更改缺陷查看器窗口中的文本的大小**视图**菜单中，选择**较小**或**较大**，或者按 F1 （较小者） 或 F2 （更大） 的密钥。 您可以按其重复来增加或减少的文本大小，直到它达到其最小值或最大大小。

此功能会影响仅缺陷查看器。 它不会更改结果窗格中的文本的大小。

下面的屏幕截图显示了如何更改在缺陷查看器窗口中的文本大小。

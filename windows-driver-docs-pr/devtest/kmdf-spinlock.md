# <a name="spinlock-rule-kmdf"></a>旋转锁规则 (kmdf)


**Spinlock**规则指定的调用[ **KeAcquireSpinLock** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-keacquirespinlock)或者[ **KeAcquireSpinLockRaiseToDpc** ](https://docs.microsoft.com/previous-versions/windows/hardware/drivers/ff551928(v=vs.85))并[ **KeReleaseSpinlock** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-kereleasespinlock)严格的分支结构中使用。

|              |      |
|--------------|------|
| 驱动程序模型 | KMDF |

# <a name="source-code-pane-actions"></a>源代码窗格中的操作


您可以执行以下操作中的**源代码**窗格：

-   **读取的源代码文件**

    若要读取的源代码文件，单击文件选项卡，并使用鼠标或键盘浏览该文件。 在你选择行或移动突出显示**源代码**窗格中中的相应代码元素[跟踪树窗格](trace-tree-pane.md)突出显示了和中的值[状态窗格](state-pane.md)发生更改以反映在该点在规则验证过程中的变量的值。

-   **切换源代码文件**

    若要查看当前未显示源代码文件，请单击该文件的选项卡。 该文件将移至文件堆栈的顶部。 单击后一个选项卡，可以使用的向右箭头和向左箭头键以将文件放在堆栈的顶部。

-   **将复制的源代码文件**

    可以从任何选项卡中复制文本**源代码**窗格和粘贴到任何应用程序都支持丰富文本格式 (RTF) 文件，例如 Microsoft Word 或写字板。

    若要将复制在源代码窗格中的任何选项卡显示的代码：

    1.  在任意位置单击**源代码**窗格选项卡。
    2.  从**编辑**菜单中，选择**全**（或按 CTRL + A）。
    3.  从**编辑**菜单中，选择**副本**（或按 CTRL + C）。

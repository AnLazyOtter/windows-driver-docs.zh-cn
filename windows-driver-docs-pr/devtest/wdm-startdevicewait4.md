# <a name="startdevicewait4-rule-wdm"></a>StartDeviceWait4 规则 (wdm)


**StartDeviceWait4**规则指定驱动程序不应调用[ **KeWaitForSingleObject** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-kewaitforsingleobject)开始设备 IRP 的上下文中。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

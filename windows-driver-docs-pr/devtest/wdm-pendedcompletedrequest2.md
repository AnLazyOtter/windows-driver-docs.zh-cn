# <a name="pendedcompletedrequest2-rule-wdm"></a>PendedCompletedRequest2 规则 (wdm)


**PendedCompletedRequest2**规则指定在调用后等待是必需[ **IoCallDriver** ](https://msdn.microsoft.com/library/windows/hardware/ff548336)或者[ **PoCallDriver**](https://msdn.microsoft.com/library/windows/hardware/ff559654)因为调度例程无法完成挂起的 IRP。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

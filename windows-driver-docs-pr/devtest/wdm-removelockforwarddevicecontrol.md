# <a name="removelockforwarddevicecontrol-rule-wdm"></a>RemoveLockForwardDeviceControl 规则 (wdm)


**RemoveLockForwardDeviceControl**规则验证的调用[ **IoAcquireRemoveLock** ](https://msdn.microsoft.com/library/windows/hardware/ff548204)并[ **IoReleaseRemoveLock**](https://msdn.microsoft.com/library/windows/hardware/ff549560)驱动程序使用时正确使用[ **IoCallDriver** ](https://msdn.microsoft.com/library/windows/hardware/ff548336)转发到另一个设备 IRP。

|              |     |
|--------------|-----|
| 驱动程序模型 | WDM |

# <a name="describe-your-service-in-the-mobile-broadband-metadata-authoring-wizard"></a>在移动宽带元数据创作向导中描述服务

有关您在 Windows 中，例如型号和制造商的服务，可以提供描述性信息。

### <a name="to-provide-descriptive-information-about-a-service"></a>若要提供有关服务的描述性信息

1. 单击**说明**选项卡。
2. 填写以下字段。
    - **服务名称**。 此可选字段不在 Windows 8 中使用。
    - **服务提供商**。 显示 Windows 连接管理器中的运算符名称。
    - **数量**。 指定唯一标识你的服务的 GUID。 此 GUID 用于标识该运算符时使用运算符 XML 预配。 如果更新设备元数据包，此 GUID 应保持不变。
        **请注意**这是与体验 ID 和设备元数据包的文件名称不同的值。 选择服务编号 GUID 的详细信息，请参阅[提供移动宽带的元数据指南](https://go.microsoft.com/fwlink/p/?linkid=242064)。

- **说明 1**。 此可选字段不在 Windows 8 中使用。
- **说明 2**。 此可选字段不在 Windows 8 中使用。

有关元数据属性的详细信息，请参阅[服务的元数据包架构引用适用于 Windows 8 的](https://go.microsoft.com/fwlink/p/?LinkId=226755)。

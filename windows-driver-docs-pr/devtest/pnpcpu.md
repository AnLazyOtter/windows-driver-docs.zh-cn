# <a name="pnpcpu"></a>PNPCPU


PNPCPU (Pnpcpu.exe) 是模拟的正在运行的 Windows Server 2008 实例的处理器热添加一个命令行工具。

驱动程序和应用程序开发人员可以使用此工具来测试，其驱动程序或应用程序不会失败时在系统运行时添加处理器。 在其中的驱动程序或应用程序已扩展为使用热添加处理器的情况下功能，PNPCPU 可用于确认正确处理所有相关即插即用通知。

PNPCPU 随附在 Windows Vista WDK 中及更高版本，并在 Windows Server 2008 上运行。 若要使用此工具，您必须是 Administrators 组的成员。

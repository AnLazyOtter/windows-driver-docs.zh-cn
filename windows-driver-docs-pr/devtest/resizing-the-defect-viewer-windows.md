# <a name="resizing-the-defect-viewer-windows"></a>调整缺陷查看器窗口的大小

要调整大小的缺陷查看器窗口，直到该图标将更改，将光标置于窗口边框上方，然后将边框拖动到所需的位置，如以下屏幕截图中所示。

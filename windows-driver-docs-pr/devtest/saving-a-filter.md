# <a name="saving-a-filter"></a>保存筛选器


没有特定过程保存筛选器。 相反，你可以跟踪会话保存在工作区中。 筛选器另存为工作区部分。 若要保存工作区：

1.  在中[跟踪会话列表](trace-session-list.md)，在会话中，右键单击任一单元，然后单击**保存工作区**。

2.  底部的页上，在**工作区**中，为工作区中，键入一个名称，单击**确定**。

在保存筛选器时在工作区中，注意不要覆盖你想要保存的文件。 由于跟踪日志的路径和文件名称保存在工作区中，如果打开的工作区，将覆盖现有的跟踪日志文件。 有关详细信息，请参阅[使用 TraceView 工作区](using-traceview-workspaces.md)。

# <a name="c28158"></a>C28158


警告 C28158:未保存 IRQL

一个 **\_IRQL\_保存\_** 批注指定的当前函数会将 IRQL 值保存在指定的变量，但在其中的 IRQL 值不保存在的中的至少一个路径变量。

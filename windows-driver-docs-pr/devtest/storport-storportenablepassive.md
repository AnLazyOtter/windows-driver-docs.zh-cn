# <a name="storportenablepassive-rule-storport"></a>StorPortEnablePassive 规则 (storport)


此规则验证**StorPortEnablePassiveInitialization**不从调用任何 StorPort 微型端口驱动程序例程以外**HwInitialize**。

|              |          |
|--------------|----------|
| 驱动程序模型 | Storport |

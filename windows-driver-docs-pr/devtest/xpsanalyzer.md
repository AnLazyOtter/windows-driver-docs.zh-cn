# <a name="xpsanalyzer"></a>XpsAnalyzer

XpsAnalyzer (XpsAnalyzer.exe) 是一个命令行工具，用于分析 XML 纸张规范 (XPS) 文件与 XPS 1.0 规范的兼容性。

此工具将执行使用 XPS 规则中所述的一组其分析[XpsAnalyzer 规则](xpsanalyzer-rules.md)。 XpsAnalyzer 会生成一个报告，描述的结构和基于这些规则的 XPS 文件的内容。 报表 XML 和 HTM 格式生成，并且可以选择生成为 SQL 文件。

有关 XPS 和 XPS 文件的结构的详细信息，请下载[XML 纸张规范](http://download.microsoft.com/download/1/6/a/16acc601-1b7a-42ad-8d4e-4f0aa156ec3e/XPS_1_0.exe)。

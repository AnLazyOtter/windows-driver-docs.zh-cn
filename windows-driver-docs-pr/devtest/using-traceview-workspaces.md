# <a name="using-traceview-workspaces"></a>使用 TraceView 工作区

TraceView 工作区是一组跟踪会话属性和可以保存并重复使用的跟踪日志显示属性。 对于工作区，可以显示常用的日志或在一个快速的步骤中启动仔细配置的跟踪会话。

本部分包括：

[保存或重新保存工作区](saving-or-resaving-a-workspace.md)

[打开一个工作区](opening-a-workspace.md)

[停止和删除工作区会话](stopping-and-removing-a-workspace-session.md)

[正在删除工作区](deleting-a-workspace.md)

[TraceView 工作区的限制](limitations-of-traceview-workspaces.md)

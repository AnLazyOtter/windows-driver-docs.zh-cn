# <a name="storportnotification2-rule-storport"></a>StorPortNotification2 规则 (storport)


此规则验证的调用**StorPortNotification**只允许使用 （即记录） 的通知类型。

允许的通知类型包括：

**BufferOverrunDetected**
**BusChangeDetected**
**LinkDown**
**LinkUp** 
**QueryTickCount**
**RequestComplete**
**RequestTimerCall** 
 **ResetDetected**
**WMIEvent**
**WMIReregister**

|              |          |
|--------------|----------|
| 驱动程序模型 | Storport |

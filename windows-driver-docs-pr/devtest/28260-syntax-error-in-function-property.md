# <a name="c28260"></a>C28260


警告 C28260:在函数内的属性进行分析时找到批注中的语法错误

此警告意味着批注中而不所分析的代码存在错误。 从通常被编码为宏的低级别批注中提取的消息文本中所述的文本。 通常情况下，但是，语法错误是在其中一个宏的参数表达式中时。 语法错误的常见位置是包含批注的函数的标头文件。

# <a name="doublekesetevent-rule-storport"></a>DoubleKeSetEvent 规则 (storport)


此规则验证**KeSetEvent**两次在同一个事件对象上不会对其进行调用。 如果同一事件对象传递给该例程，该驱动程序未能通过该规则。

|              |          |
|--------------|----------|
| 驱动程序模型 | Storport |

# <a name="stampinf"></a>Stampinf


Stampinf (Stampinf.exe) 是一种工具，可用于生成该驱动程序时更新常见 INF 和 INX 文件指令。 有关设置项目的 Stampinf 选项的信息，请参阅[驱动程序项目的 Stampinf 属性](https://docs.microsoft.com/windows-hardware/drivers/develop/stampinf-properties-for-driver-projects)。 有关将 Stampinf 支持添加到项目文件的信息，请参阅[WDK 和 Visual Studio 构建环境](wdk-and-visual-studio-build-environment.md)并[Stampinf 任务](stampinf-task.md)。

INX 文件类似于 INF 文件和 KMDF 和 UMDF 驱动程序用于支持多个平台和 KMDF 和 UMDF 版本。 有关 INX 文件的信息，请参阅[使用 INX 文件转换为创建 INF 文件](https://msdn.microsoft.com/library/windows/hardware/ff545473)。

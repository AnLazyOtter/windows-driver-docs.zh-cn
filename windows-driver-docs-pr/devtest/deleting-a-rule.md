# <a name="deleting-a-rule"></a>正在删除规则


若要从筛选器中删除一条规则：

1.  在中[跟踪会话列表](trace-session-list.md)，在会话中，右键单击任一单元，然后单击**管理筛选器**。

2.  在中**管理筛选器**对话框中**当前规则**区域中，选择你想要删除的规则。

3.  单击**删除规则**。

如果你已保存中的规则[工作区](using-traceview-workspaces.md)，你必须[重新保存工作区](saving-or-resaving-a-workspace.md)后删除规则。 否则，该规则不会删除从工作区。

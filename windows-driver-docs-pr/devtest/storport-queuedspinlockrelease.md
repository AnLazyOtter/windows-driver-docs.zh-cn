# <a name="queuedspinlockrelease-rule-storport"></a>QueuedSpinLockRelease 规则 (storport)


此规则将验证该驱动程序不会调用**KeReleaseInStackQueuedSpinLock**而不必首先获取通过锁**KeAcquireInStackQueuedSpinLock**。

|              |          |
|--------------|----------|
| 驱动程序模型 | Storport |

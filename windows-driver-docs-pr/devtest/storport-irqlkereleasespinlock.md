# <a name="irqlkereleasespinlock-rule-storport"></a>IrqlKeReleaseSpinLock 规则 (storport)


此规则验证**KeReleaseSpinLock**在调用**IRQL = 调度\_级别**仅。 它还必须设置为以前的 IRQL 级别的 IRQL。 此调用通常会通过调用前面**KeAcquireSpinLock**。

|--------------|----------|
| 驱动程序模型 | Storport |

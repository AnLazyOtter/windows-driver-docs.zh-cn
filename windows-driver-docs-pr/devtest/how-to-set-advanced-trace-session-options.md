# <a name="how-to-set-advanced-trace-session-options"></a>如何设置会话选项的高级的跟踪


使用以下过程创建跟踪会话时设置高级的跟踪会话选项。

1.  上**文件**菜单上，单击**新建日志会话**。

2.  单击**添加提供程序**，然后选择一个提供程序。 如有必要指定 TMF 文件或 TMF 文件路径。

3.  若要添加更多提供程序，请单击**添加提供程序**。 此步骤可选。

4.  单击“下一步” 。

5.  上**日志会话选项**页上，如果需要将基本的跟踪设置会话选项。

6.  单击**高级日志会话选项**按钮。

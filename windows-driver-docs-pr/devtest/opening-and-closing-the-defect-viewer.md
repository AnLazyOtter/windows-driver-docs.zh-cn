# <a name="opening-and-closing-the-trace-viewer"></a>打开和关闭跟踪查看器


您可以打开跟踪查看器中的任何规则**结果**报告缺陷的选项卡。

若要打开跟踪查看器：

-   在结果选项卡中找到的规则，感兴趣，并单击**缺陷**。

-   在此期间 SDV 检索它需要的信息，短暂延迟后跟踪查看器打开。 然后，在更多扩展延迟，它定位并调整窗口的大小。 1 分钟或更长的延迟被视为正常。

可以一次显示一个跟踪查看器 （适用于一个规则冲突）。 若要打开不同的规则冲突跟踪查看器，必须先关闭当前的规则冲突跟踪查看器。

若要关闭缺陷查看器，请执行以下任一操作：

-   从**文件**菜单中，选择**退出**。

-   单击**关闭**(**X**) 中静态驱动程序验证工具报告的最右侧角的按钮。

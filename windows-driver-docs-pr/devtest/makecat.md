# <a name="makecat"></a>MakeCat


MakeCat (Makecat.exe) 是一个命令行[CryptoAPI](https://go.microsoft.com/fwlink/p/?linkid=136391)工具，用于创建[编录文件](https://msdn.microsoft.com/library/windows/hardware/ff537872)有关[驱动程序包](https://msdn.microsoft.com/library/windows/hardware/ff544840)。

有关 MakeCat 工具和其命令行自变量的详细信息，请参阅[使用 MakeCat](https://go.microsoft.com/fwlink/p/?linkid=70086)网站。

有关如何使用 MakeCat 工具的详细信息，请参阅[版本签名的驱动程序包创建编录文件](https://msdn.microsoft.com/library/windows/hardware/ff540172)。

**请注意**  必须使用 MakeCat 工具只是为了创建目录文件的情况下使用 INF 文件不安装的驱动程序包。 如果要安装驱动程序包，请使用 INF 文件，使用[ **Inf2Cat** ](inf2cat.md)工具创建目录文件。 Inf2Cat 自动包含在包的 INF 文件中引用的驱动程序包中的所有文件。 有关如何使用 Inf2Cat 工具的详细信息，请参阅[使用 Inf2Cat 创建编录文件](https://msdn.microsoft.com/library/windows/hardware/ff553618)。

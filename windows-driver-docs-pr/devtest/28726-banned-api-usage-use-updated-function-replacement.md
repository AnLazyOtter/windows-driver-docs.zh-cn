# <a name="c28726"></a>C28726


警告 C28726:已禁止的 API 用法

此警告表明，正在使用的函数已被禁或已弃用，并且具有更为可靠和安全替换。

若要查找替换，搜索名称中不推荐使用的函数[Windows 驱动程序开发](https://msdn.microsoft.com/library/windows/hardware/ff557573)文档。 文档列出了不推荐使用的函数的名称，并建议替换项。

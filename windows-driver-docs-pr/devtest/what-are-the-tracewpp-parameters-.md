# <a name="what-are-the-tracewpp-parameters"></a>有哪些 TraceWPP 参数？


TraceWPP 工具 (TraceWpp.exe) 是预处理器 WPP。 此工具的参数的详细信息，请参阅[WPP 预处理器](wpp-preprocessor.md)。

若要列出所有 TraceWPP 参数，请使用以下命令：

```
TraceWpp -?
```

# <a name="what-is-an-autologger"></a>什么是自动记录器？


*自动记录器*是[跟踪会话](trace-session.md)记录从用户模式和内核模式事件[跟踪提供程序](trace-provider.md)在启动过程中。 此功能允许你跟踪的跟踪提供程序的操作，当 Windows 正在启动。

仅在 Windows Vista 和更高版本的 Windows 上支持自动记录器会话。 在这些系统中，您可以将记录到全局记录器跟踪会话，如中所述[跟踪期间启动](tracing-during-boot.md)，但 autologgers 是首选的方法。

有关 autologgers 和早期版本的 Windows 中的启动跟踪的方法的详细信息，请参阅[跟踪期间启动](tracing-during-boot.md)。

# <a name="initfreedevicecreatetype2-rule-kmdf"></a>InitFreeDeviceCreateType2 规则 (kmdf)


InitFreeDeviceCreateType2 规则指定一个驱动程序必须调用[ **WdfDeviceCreate** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdfdevice/nf-wdfdevice-wdfdevicecreate)它将调用后[ **WdfDeviceInitFree** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdfdevice/nf-wdfdevice-wdfdeviceinitfree).

|              |      |
|--------------|------|
| 驱动程序模型 | KMDF |

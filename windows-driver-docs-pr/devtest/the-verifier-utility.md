# <a name="the-verifier-utility"></a>验证程序实用工具


## <span id="ddk_the_verifier_utility_tools"></span><span id="DDK_THE_VERIFIER_UTILITY_TOOLS"></span>


验证程序实用工具，Verifier.exe，激活并监视驱动程序验证程序。 此实用工具位于 %windir%\\Windows system32 目录。

有两种方法用于验证程序实用工具。 在 Windows Vista 和更高版本的 Windows 上，所需的管理特权。

-   **命令行。** 若要在命令行使用驱动程序验证程序，请打开命令提示符窗口 (**以管理员身份运行**)，类型**verifier**跟至少一个命令行参数。

-   **驱动程序验证程序管理器。** 若要打开驱动程序验证程序管理器图形用户界面，请打开命令提示符窗口 (**以管理员身份运行**)，类型**verifier** （不带任何参数）。

**请注意**  驱动程序验证程序管理器在 Windows XP 及更高版本稍有不同版本的驱动程序验证程序管理器在 Windows 2000 中。

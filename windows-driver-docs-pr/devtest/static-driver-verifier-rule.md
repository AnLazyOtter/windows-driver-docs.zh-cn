# <a name="static-driver-verifier-rule"></a>Static Driver Verifier 规则


验证规则定义所必需的驱动程序和操作系统的内核接口之间正确交互。 规则是用规范语言的接口检查 （slic 表），一种简单的语言，由 Microsoft Research 开发明确实现此目的。

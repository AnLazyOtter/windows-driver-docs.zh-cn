# <a name="download-failure-cabs"></a>下载失败的 Cab

使用此方法，可以使用从作为[获取报告数据](get-report-data.md) API的一部分接收的报告中检索到的 applicationId 和 cabIdHash 为驱动程序下载失败的 cab

- [下载 OEM 硬件错误的 CAB 文件](https://docs.microsoft.com/windows/uwp/monetize/download-the-cab-file-for-an-oem-hardware-error)

- [下载 Windows 10 驱动程序错误的 CAB 文件](https://docs.microsoft.com/windows/uwp/monetize/download-the-cab-file-for-a-windows-10-driver-error)

此外，你可以使用[批量下载驱动程序的 CAB 文件](bulk-download-failure-cabs.md)下载可用于失败哈希的所有文件

# <a name="create-a-mobile-broadband-experience"></a>创建移动宽带体验

移动宽带体验可帮助你创建用于关联 UWP 移动宽带应用与硬件保护装置的服务元数据包。

本节包括下列主题：

- [开发人员模式和预览模式](https://msdn.microsoft.com/library/windows/hardware/dn282082.aspx)

- [服务图标要求](https://msdn.microsoft.com/library/windows/hardware/dn236416.aspx)

- [服务编号](https://msdn.microsoft.com/library/windows/hardware/dn236413.aspx)

- [受信任的证书](https://msdn.microsoft.com/library/windows/hardware/dn486876.aspx)

你还可获取有关服务元数据包的详细信息，以及如何使用以下链接将其提交至合作伙伴中心：

- [用于创建服务元数据的开发人员指南](https://msdn.microsoft.com/library/windows/hardware/dn973080) - 这是一个分步指南，可引导你完成将服务元数据包提交至合作伙伴中心的过程。

- [服务元数据包架构参考](https://msdn.microsoft.com/library/windows/hardware/dn973175) - 针对服务元数据包中所含 XML 架构的参考。

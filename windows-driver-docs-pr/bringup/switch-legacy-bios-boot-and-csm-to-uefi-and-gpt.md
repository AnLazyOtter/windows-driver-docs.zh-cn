# <a name="switch-legacy-bios-boot-and-csm-to-uefi-and-gpt"></a>切换到 UEFI 和 GPT 的传统 BIOS 引导和 CSM


固件启动选项; 之间切换旧版 BIOS 和 UEFI （标记可能与此不同固件之间），则不需要进行升级。 

请务必注意，当对固件和文件系统进行更改，就可以使问题复杂化。 

推荐的方案是升级系统"按原样"。

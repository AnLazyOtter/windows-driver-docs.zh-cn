# <a name="device-guard-and-credential-guard"></a>Device Guard 和 Credential Guard


Device Guard 是一个 Windows 安全功能的组合的企业相关的硬件、 固件和软件安全功能。 当一起配置时，它将锁定设备，以便它仅可以运行受信任的应用程序。

凭据保护使用基于虚拟化的安全性来隔离密钥，以便只有特权系统软件可以访问它们。

Device Guard 和 Credential Guard 是可选功能，如果实现并已启用，可以减少公开设备上启用恶意软件通过要求额外的保护程序的攻击面。

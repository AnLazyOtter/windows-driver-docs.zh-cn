# <a name="wi-fi-support-in-firmware"></a>在固件中的 Wi-fi 支持 


Wi-fi 支持最初定义在 UEFI 规范 2.5 中，使用 UEFI 规范 2.6 中的相关。

Windows 操作系统和 Windows 预安装环境 (WinPE) 当前不具有能够充分利用这些功能的 UEFI 的支持。 

发送电子邮件至<SAUEFI@Microsoft.com>如果您有问题或疑虑，或者如果你的公司有此功能感兴趣。

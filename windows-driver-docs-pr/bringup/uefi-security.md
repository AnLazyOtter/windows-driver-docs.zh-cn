# <a name="uefi-security"></a>UEFI 安全性


Windows 操作系统取决于的完整性和固件和硬件组件的配置以提供安全的工作环境。

本部分旨在作为指南提供了指向可执行和验证固件的 Windows 操作系统的安全功能的信息。

以下部分介绍用于启用更安全的固件和更安全的操作系统的固件组件。 这将涵盖突出显示的组件，并提供指针和现有的联机资源的链接 （如果有）。

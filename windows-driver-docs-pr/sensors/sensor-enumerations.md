# <a name="sensor-enumerations"></a>传感器枚举


以下枚举用于指示不同状态或值的各种传感器功能。

## <a name="in-this-section"></a>本节内容

|主题|描述|
|---|---|
|PROXIMITY_TYPE|此枚举表示的邻近邻近感应传感器可以检测到的类型。|
|[ACTIVITY_STATE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-activity_state)|此枚举表示活动检测传感器报告的活动状态。|
|[PEDOMETER_STEP_TYPE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-pedometer_step_type)|此枚举表示由计步器报告的步骤类型。|
|[PEDOMETER_STEP_TYPE_COUNT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-pedometer_step_type_count)|此枚举表示计步器可以检测到的步骤类型的数目。|
|[MAGNETOMETER_ACCURACY](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsclassextension/ne-sensorsclassextension-magnetometeraccuracy)|此枚举表示磁力仪的准确性状态。|
|SENSOR_DATA_TYPE|此枚举描述受支持的传感器的数据类型。|
|[SENSOR_STATE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-sensor_state)|此枚举表示传感器的有效状态。|
|SENSOR_TYPE|此枚举包含的客户端驱动程序使用来描述传感器类型的值。|
|COORDINATE_AXIS|此枚举表示三维欧几里得坐标轴的六个不同的方向。|

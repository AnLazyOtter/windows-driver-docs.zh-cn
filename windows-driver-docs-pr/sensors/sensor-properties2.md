# <a name="sensor-properties"></a>传感器属性


通用传感器驱动程序模型支持属性的特征化 Windows 设备安装的驱动程序的使用。

每个属性表示由*属性键*，用于标识的属性类别和属性标识符的 GUID 值。 本主题包含有关属性的合作伙伴可用于标识其传感器驱动程序的信息。

### <a name="in-this-section"></a>本部分内容

|主题|描述|
|---|---|
|[通用传感器属性](common-sensor-properties.md)|本主题介绍常见的所有传感器的传感器属性。|
|[枚举属性](enumeration-properties.md)|本主题介绍中即插即用驱动程序存储区提供了静态传感器属性。|
|[其他传感器属性](other-sensor-properties.md)|除了[通用传感器属性](common-sensor-properties.md)并[枚举属性](enumeration-properties.md)，有其他特定于某些类型的传感器的传感器属性。 本部分讨论了这些其他属性。|
|[数据字段属性](data-field-properties.md)|本主题介绍用于仅数据字段的传感器属性。|

# <a name="use-the-sensors-diagnostic-tool-to-test-your-driver-and-device"></a>使用传感器诊断工具来测试您的驱动程序和设备


传感器诊断工具 (SDT) 随 Windows 驱动程序工具包，并可以使用此工具来测试驱动程序、 固件和硬件功能。

>[!NOTE]
> 传感器诊断工具现已弃用适用于 Windows 10。 请 SensorInfo 应用从 Microsoft Store 中，使用的所有传感器测试和诊断。

 

您的设备和驱动程序运行后，可以使用此工具来测试：

-   数据检索
-   事件处理
-   报告间隔
-   更改敏感度
-   属性检索
-   属性设置

此屏幕截图显示 SDT 运行后 SpbAccelerometer 示例驱动程序和 ADXL345 传感器所安装：

![传感器诊断工具](images/sdt2.png)

有关使用 SDT 的详细信息，请参阅[传感器驱动程序文档](https://docs.microsoft.com/windows-hardware/drivers/sensors/)。

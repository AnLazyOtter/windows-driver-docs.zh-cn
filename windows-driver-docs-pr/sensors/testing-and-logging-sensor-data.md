# <a name="testing-sensor-property-support"></a>测试传感器属性支持


使用传感器诊断工具来测试是否驱动程序和固件支持属性检索。 该工具将调用在传感器 API，以确定是否支持检索属性的属性。
 

## <a name="configuring-the-sensor-diagnostic-tool-to-retrieve-sensor-properties"></a>配置传感器诊断工具来检索传感器属性


若要配置诊断工具，以便它可以检索加速感应器的属性，请执行以下操作：

1.  展开左侧的传感器窗格中加速感应器的节点并检查**已连接**框。
2.  单击左侧的传感器窗格中的加速感应器节点。

右窗格中的属性部分包含您的传感器的更新后的属性数据。 此数据对应于你的设备支持的属性。

[测试传感器事件](testing-sensor-events.md)主题介绍如何测试是否驱动程序和固件支持变化敏感度和当前报告间隔。 请参阅以下各节，了解如何使用该工具来测试支持的属性时更改这些设置。

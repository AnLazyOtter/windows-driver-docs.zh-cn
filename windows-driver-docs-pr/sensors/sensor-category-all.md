# <a name="sensorcategoryall"></a>传感器\_类别\_所有


传感器\_类别\_所有类别都表示的所有平台定义的传感器类别集。

## <a name="platform-defined-property-keys"></a>平台定义的属性键

此类别包括以下平台定义的数据字段。

|数据类型|在任务栏的搜索框中键入|含义|
|--|--|--|
|SENSOR_DATA_TYPE_TIMESTAMP|VT_FILETIME|报表所需的所有数据。 将标记数据报表创建的每个数据报表使用时间。 使用协调世界时 (UTC)。|
 

>[!IMPORTANT]
> 每个平台定义常见的数据类型**PROPERTYKEY**基于一种常见**GUID**名为传感器\_数据\_类型\_常见\_GUID。 由于它是保留的基值，请勿使用此**GUID**来定义你自己属性的密钥。

# <a name="time-stamp-helper"></a>时间戳帮助程序


时间戳帮助程序函数由 v2 传感器驱动程序，并且与传感器设备驱动程序软件接口 (DDSI) 一起使用。

**GetPerformanceTime**

通过传感器 DDSI 的使用情况

-   检索时间 （毫秒）。

备注

-   用于计算的进程或度量持续时间。

|--------------------------|------------------------|
| 最低受支持的客户端 | Windows 8.1            |
| 最低受支持的服务器 | Windows Server 2012 R2 |
| Header                   | Sensorsutils.h         |

# <a name="sensor-helper-functions"></a>传感器帮助器函数

本部分提供有关使用传感器结构的结构进行初始化和要检索其相关信息的帮助器函数的信息。

## <a name="in-this-section"></a>本节内容

|主题|描述|
|---|---|
|[SENSOR_COLLECTION_LIST_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_collection_list_init)|此函数将初始化 SENSOR_COLLECTION_LIST 结构。|
|[SENSOR_COLLECTION_LIST_SIZE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_collection_list_size)|此函数返回 SENSOR_COLLECTION_LIST 结构的大小。|
|[SENSOR_COLLECTION_LIST_CALCULATE_MAX_COUNT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_collection_list_calculate_max_count)|此函数计算 SENSOR_VALUE_PAIR SENSOR_COLLECTION_LIST 结构中的元素的数。|
|[SENSOR_PROPERTY_LIST_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_property_list_init)|此函数将初始化 SENSOR_PROPERTY_LIST 结构。|
|[SENSOR_PROPERTY_LIST_SIZE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_property_list_size)|此函数返回的属性列表的大小。|
|[SENSOR_PROPERTY_LIST_CALCULATE_MAX_COUNT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_property_list_calculate_max_count)|此函数可计算 PROPERTYKEY 元素数。|
|[SENSOR_CONFIG_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorscx/nf-sensorscx-sensor_config_init)|此函数将初始化 SENSOR_CONFIG 结构。|
|[SENSOR_CONTROLLER_CONFIG_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorscx/nf-sensorscx-sensor_controller_config_init)|此函数将初始化 SENSOR_CONTROLLER_CONFIG 结构。|

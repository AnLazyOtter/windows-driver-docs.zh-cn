# <a name="proximity-sensor-property"></a>邻近传感器属性



此属性是可选的枚举属性。

|属性键|在任务栏的搜索框中键入|访问 （R/O，R/W） |必需/可选|描述|
|--|--|--|--|--|
|DEVPKEY_Sensor_ProximityType|VT_UI4|R/O|可选|描述被检测到邻近的类型。 它可以是 HumanProximity 或 ObjectProximity。 有关详细信息，请参阅 ProximityType 枚举。|

 

## <a name="requirements"></a>要求


**标头：** Sensorsdef.h

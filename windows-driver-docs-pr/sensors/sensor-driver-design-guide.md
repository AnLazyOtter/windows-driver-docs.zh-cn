# <a name="sensor-driver-design-guide"></a>传感器驱动程序设计指南


本文档介绍如何编写与 Windows 中的传感器和位置平台进行交互的设备驱动程序。 Windows 驱动程序工具包提供了一个基本示例，名为 SensorsGeolocationDriverSample，以指导你完成实现传感器硬件的设备驱动程序的第一个步骤。 此外，Windows 驱动程序工具包提供了更完整的示例中，调用演示了支持的实际硬件所需的代码 SensorsHIDClassDriver。

在本部分中的主题介绍如何生成来启用传感器设备以使用 Windows 中的传感器和位置平台的驱动程序。

# <a name="sample-wdtf-scenarios"></a>示例 WDTF 方案


可以访问提供 WDK WDTF 基于方案脚本。 要查找这些和其他示例，请打开 Windows 资源管理器并导航到测试\\示例\\WDTF WDK 安装目录 (例如，c:\\Program Files (x86)\\Windows 工具包\\8.0\\测试\\示例\\WDTF\\测试)。

**请注意**  如果你打算创建您自己的测试的这些脚本的副本，请确保使用每个新版本的 WDK 更新您的脚本。 这些示例脚本可能会更改每个版本。 对于最准确地进行测试，请确保将任何这些脚本的版本合并到您的脚本。

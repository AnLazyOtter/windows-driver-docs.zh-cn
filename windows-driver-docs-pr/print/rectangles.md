# <a name="rectangles"></a>矩形





若要描述 GPD 文件中的矩形，请使用 RECT 关键字。 格式为：

**RECT** ( *left* , *top* , *right* , *bottom* )

其中*左*，*顶部*，*右*，并*底部*表示[数值](numeric-values.md)中,[掌握单位](master-units.md)。

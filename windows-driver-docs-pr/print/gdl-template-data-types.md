# <a name="gdl-template-data-types"></a>GDL 模板数据类型


*GDL 模板数据类型*分为[基元](gdl-primitive-data-types.md)数据类型和[复合](gdl-compound-data-types.md)数据类型。

数据类型已分配给属性模板后，分析器筛选器将尝试解析是此模板的实例，为该数据类型使用的语法规则定义的任何 GDL 项中显示的值。 您可以定义一些通过语法[模板指令](gdl-template-directives.md)。 如果分析成功，该数据类型将映射到一个或多个 XML 基元类型，然后为该属性的子元素发出在快照中。

模板具有用于特殊规则[继承](data-type-template-inheritance.md)。

有关模板的数据类型的详细信息，请参阅以下各节：

[GDL 基元数据类型](gdl-primitive-data-types.md)

[GDL 复合数据类型](gdl-compound-data-types.md)

[使用模板来定义数据类型](defining-data-types-by-using-templates.md)

[数据类型模板继承](data-type-template-inheritance.md)

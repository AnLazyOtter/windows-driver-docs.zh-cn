# <a name="attribute-types"></a>属性类型





定义多种类型的属性，组成：

-   [常规属性](general-attributes.md)

    常规属性是与打印机设备相关联。

-   [命令属性](command-attributes.md)

    命令属性是与打印机命令相关联。

-   [功能属性](feature-attributes.md)

    功能属性都与特定的打印机功能。

-   [选项属性](option-attributes.md)

    选项属性相关联，属于特定功能的选项。

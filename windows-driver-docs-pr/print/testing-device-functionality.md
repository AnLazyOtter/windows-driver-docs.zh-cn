# <a name="testing-device-functionality"></a>测试设备功能


完成设备安装和插功能的所有测试后，验证设备在按预期执行的所有函数。 尽可能打印测试页面和任意数量的应用程序和文档类型。

您还应执行以下附加功能测试。

1.  **用户的访问权限：** 验证具有不同级别的系统访问权限的用户可以打印到设备，包括用户身份登录到远程使用远程桌面连接应用程序的系统。

2.  **特殊功能：** 验证你的设备支持的任何特殊功能。 例如，如果打印机支持双向 (bidi) 通信，验证从打印机的双向请求正常运行，如正在取消从设备和双工打印打印作业。

3.  **作业控制：** 打印作业可以进行暂停、 取消、 恢复后，验证，以及重新启动。 此外验证多个打印作业可以是启动、 打印、 取消，并且同时重新启动。

# <a name="writing-a-network-print-provider"></a>编写网络打印提供程序





有时，一个供应商可能想要提供[打印提供程序](print-providers.md)以支持新的网络体系结构。 一种可能的方法提供新的打印提供程序功能是创建全新的打印提供程序替换[本地打印提供程序](local-print-provider.md)。 实际上，但是，这是第一挑战性且可能不必要的任务。 一种替代方法是创建与本地打印提供程序结合使用的部分的打印提供程序的工作。

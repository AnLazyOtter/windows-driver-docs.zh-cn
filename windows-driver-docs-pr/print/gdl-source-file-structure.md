# <a name="gdl-source-file-structure"></a>GDL 源文件结构


可以是 GDL 条目[特性](gdl-attributes.md)或[构造](gdl-constructs.md)。 *特性*是[关键字](gdl-keywords.md)-[值](gdl-values.md)对。 一个*构造*是一个对象，可以包含零个或多个 GDL 条目 （这可能是属性或构造或这两者的组合） 和本身与之关联的关键字值对。

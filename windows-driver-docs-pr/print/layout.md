# <a name="layout"></a>布局


架构路径：\\Printer.Layout

节点类型： 属性

布局属性包含打印作业的每个页面的外观的相关的信息。 这包括可用介质的类型以及如何将有关每个工作表的信息。

布局属性是父级的以下属性：

[NumberUp](numberup2.md)

[方向](orientation2.md)

[解决方法](resolutions.md)

[InputBins](inputbins2.md)

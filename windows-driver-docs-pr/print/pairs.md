# <a name="pairs"></a>对





若要将一对值分配到属性，请使用对关键字。 格式为：

**PAIR** ( *Value1* , *Value2* )

其中*Value1*并*Value2*是[数值](numeric-values.md)。 例如，按指定的游标原点位置[掌握单位](master-units.md)，如下所示：

```cpp
*CursorOrigin: PAIR(120, 100)
```

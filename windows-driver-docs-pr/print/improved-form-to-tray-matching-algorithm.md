# <a name="improved-form-to-tray-matching-algorithm"></a>改进的送纸器格式匹配算法


Windows Vista 中的送纸器格式匹配算法进行了改进的方式相同[匹配的函数的窗体](improved-form-matching-algorithm.md)。 在 Windows Vista 中，支持的本地化的名称已增强，可提供更加精确和独立于语言的送纸器窗体关系查找。

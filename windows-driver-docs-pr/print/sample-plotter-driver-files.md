# <a name="sample-plotter-driver-files"></a>绘图仪驱动程序文件示例





Windows Driver Kit (WDK) 包括以下 MSPlot 示例代码：

-   有关源代码[绘图器驱动程序用户界面](plotter-driver-user-interface.md)DLL (plotui.dll)

-   有关源代码[绘图器驱动程序呈现器](plotter-driver-renderer.md)(plotter.dll)

-   Plotgpc.exe，生成.pcd 文件的源代码。

-   示例微型驱动程序文本文件，，该文件可以用作输入 plotgpc.exe。

示例位于包含 WDK 示例的目录树的子目录中。 绘图器驱动程序呈现器 (plotter.dll) 可以编译为用户模式或内核模式 DLL。 有关详细信息，请参阅[选择用户模式或内核模式](choosing-user-mode-or-kernel-mode.md)。

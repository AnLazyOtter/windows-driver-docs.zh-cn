# <a name="printer-features"></a>打印机功能





打印机功能是可以由 Unidrv 驱动程序控制的功能。 通过列出功能，其特征 GPD 文件中的，您通知 Unidrv 驱动程序在打印机支持的功能。

每个打印机功能可以分配给一个或多个状态，并[打印机选项](printer-options.md)用于定义了可能的状态。 例如，如果您的打印机接受信纸尺寸和法律大小页，GPD 文件应指定 PaperSize 功能，以及号和法律选项。

打印机属性页上列出的所有功能和其关联的选项，或与打印机相关的文档属性表。 有关这些属性表的详细信息，请参阅[Unidrv 用户界面](unidrv-user-interface.md)。

本部分介绍两个 GPD 语言支持[标准功能](standard-features.md)并[自定义功能](customized-features.md)。 在本部分中的其他主题包括[功能条目格式](feature-entry-format.md)，[功能属性](feature-attributes.md)，并[功能冲突优先级](feature-conflict-priority.md)。

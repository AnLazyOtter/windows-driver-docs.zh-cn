# <a name="deviceinfo-section"></a>DeviceInfo 部分


DeviceInfo 部分组成[DeviceInfo](deviceinfo.md)属性并提供有关整体的设备的信息的多个对子值。

```cpp
Printer
 DeviceInfo
 FriendlyName
 Manufacturer
 ModelName
 Location
    FirmwareVersion
    IEEE1284DeviceID
    NetworkingInfo
 Comment
```

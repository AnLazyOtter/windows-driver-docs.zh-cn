# <a name="macros"></a>宏





GPD 语言支持两种类型的宏：

-   [阻止宏](block-macros.md)，这样便可以指定一组可以完全 GPD 文件中的任意位置插入的 GPD 文件条目。

-   [值宏](value-macros.md)，这样便可以指定可以插入 GPD 文件中的单独任意位置的值集。

# <a name="option-attributes"></a>选项属性





指定时[打印机选项](printer-options.md)，特性用于 Unidrv 提供以下信息：

-   表示一个选项的显示名称、 用户提示和帮助消息的文本字符串。

-   一个布尔值，该值指示是否将始终存在或是可安装的选项。

-   要选择一个选项将导致该打印机命令。

-   不同类型的特定于选项的信息。

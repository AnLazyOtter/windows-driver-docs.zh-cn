# <a name="serviceid"></a>ServiceID


`ServiceID`命令返回的服务标识符 (ID) 中的函数发现密钥 PNP 窗体\_PNPX\_ServiceID。 输出缓冲区包含以 NULL 结尾的字符串表示的 URI。

有关函数发现密钥的详细信息，请参阅 Microsoft Windows SDK 文档。

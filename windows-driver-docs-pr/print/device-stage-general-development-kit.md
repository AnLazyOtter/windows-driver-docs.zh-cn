# <a name="device-stage-general-development-kit"></a>设备阶段常规开发工具包


设备阶段常规开发工具包包含以下信息：

-   **Windows 7 设备阶段设计指南**-如何创建 Device Stage 体验，以反映你的公司的公司和产品品牌。

-   **Windows 7 设备阶段开发指南**-的所需知道创建 Device Stage 体验的常见步骤，Device Stage 开发过程的概述。

-   **Windows 7 设备阶段参考指南**-引用用于开发的指南和组装 Device Stage 体验，包括高级开发主题。

-   **Windows 7 设备阶段 XML 架构**-架构文件可用于编辑 Device Stage 体验 XML 文件。

-   **Windows 7 设备阶段开发示例**-一系列示例图形和工具来帮助开发 Device Stage 体验。

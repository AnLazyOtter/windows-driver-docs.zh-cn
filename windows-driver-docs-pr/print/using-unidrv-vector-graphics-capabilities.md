# <a name="using-unidrv-vector-graphics-capabilities"></a>使用 Unidrv 矢量图形功能





Microsoft Windows XP 及更高版本，Unidrv 提供两种类型的向量图形打印机支持：HP/2 单色 (PCL-5e) 或 PCL XL 单色 (PCL 6)。 Microsoft Windows Server 2003 及更高版本，Unidrv 还支持 PCL XL 颜色 (PCL 6)。 此外，还有 pseudovector 图形支持的图形不具备向量呈现功能的打印机。

[HP/2 单色 (PCL 5e) 向量图形支持](hp-gl-2-monochrome--pcl-5e--vector-graphics-support.md)

[PCL XL (PCL 6) 向量图形支持](pcl-xl--pcl-6--vector-graphics-support.md)

[伪矢量图形支持](pseudo-vector-graphics-support.md)

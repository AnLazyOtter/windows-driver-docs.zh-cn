# <a name="updating-web-page-information"></a>更新 Web 页面信息





打印服务器不支持服务器推送更新有关打印服务器的 Web 页的信息。 客户端请求操作 （假定用户的浏览器可以识别的 HTML 指定"刷新"参数） 支持，并且可以包含在自定义页面。 （有关客户端请求的详细信息，请参阅相应的 HTML 文档。）

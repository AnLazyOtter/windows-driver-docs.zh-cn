# <a name="managing-a-port"></a>管理端口





端口管理活动组成[添加一个端口](adding-a-port.md)，[配置的端口](configuring-a-port.md)，并[删除端口](deleting-a-port.md)请求响应来自应用程序 （如打印文件夹）。 这些端口管理活动由端口监视器，不是语言监视器，并且需要管理员权限。 有特殊规则[将端口配置信息存储](storing-port-configuration-information.md)。

一个附加的可选的管理活动涉及[设置端口超时值](setting-port-time-out-values.md)。

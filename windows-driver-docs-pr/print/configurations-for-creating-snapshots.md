# <a name="configurations-for-creating-snapshots"></a>用于创建快照的配置


配置是没有什么比参数和分配给每个参数的当前值的列表。 GDL 分析器接口需要的配置为在 XML 中，但为了简化，下面的伪代码示例显示了典型表示的配置使用的参数值对的列表。

```cpp
Today: Sunday
Weather: Cloudy
```

# <a name="range-template-data-type-issues"></a>范围模板数据类型问题


\*ArraySize 指令接受一个整数范围。 (一个范围是由连字符 （-） 分隔的 （可选） 括在圆括号和方括号的两个整数\[例如， \*ArraySize:\[1-15\]\]). 可以使用 GPD 通配符 (\*) 来表示任何整数值。 例如，若要定义任何大小的数组，可以使用\*ArraySize: \*。

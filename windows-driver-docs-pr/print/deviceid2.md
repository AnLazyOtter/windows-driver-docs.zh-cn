# <a name="deviceid"></a>DeviceID


**DeviceID**命令返回的设备标识符 (ID) 中的功能发现键，窗体**主键\_PNPX\_GlobalIdentity**。 此键返回根设备的全局唯一 ID (UUID)。 输出缓冲区中包含一个以 NULL 结尾的字符串，表示 UUID。

有关函数发现密钥的详细信息，请参阅 Microsoft Windows SDK 文档。

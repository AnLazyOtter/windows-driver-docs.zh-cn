# <a name="memory"></a>内存


架构路径：\\Printer.Configuration.Memory

节点类型： 属性

说明： 安装在设备中的内存值项

内存属性包含两个对子值：**大小**并**PS**。

### <a name="span-idsizespanspan-idsizespan-size"></a><span id="size"></span><span id="SIZE"></span> 大小

架构路径：\\Printer.Configuration.Memory:Size

节点类型： 值

数据类型： BIDI\_INT

说明： 安装在设备的千字节 (KB) 中的物理内存量。

### <a name="span-idpsspanspan-idpsspan-ps"></a><span id="ps"></span><span id="PS"></span> PS

架构路径：\\Printer.Configuration.Memory:PS

节点类型： 值

数据类型： BIDI\_INT

说明： 的内存量，以千字节 (KB)，可用于设备的 Postscript 解释程序。 此数量应早于所安装物理内存量。

# <a name="unicode-representations-in-xml-snapshots"></a>XML 快照中的 Unicode 表示形式


GDL 分析器可以通过在它生成的快照和架构中使用 utf-8 或 utf-16 表示 Unicode。 用户可以选择此选项标志通过 Unicode 中相应的接口函数。 UTF-8 编码快照可以加载到任何文本编辑器中以便于查看，通常小于 UTF 16 版本。

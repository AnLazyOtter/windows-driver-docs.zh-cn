# <a name="numeric-values"></a>数字值





所有的数字值，指定为条目的值或 GPD 文件中的参数值必须是整数。 不允许小数点，除了文本字符串内。

数字值假定为正数，除非前面有减号。

数字值假定为十进制，除非前面有 0x，在这种情况下它们是无符号十六进制值。

星号字符 (\*) 可以用于指示无限值或"不关心"值，如果特定 GPD 文件条目的上下文中适用。

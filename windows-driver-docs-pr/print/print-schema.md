# <a name="print-schema"></a>打印架构


从根本上打印架构为基础的打印票证和打印功能的技术。 打印架构定义的组织和 PrintTicket 和 PrintCapabilities 文档，这些技术使用的内容。

打印架构设计映射到特定关键字的公共功能，并且还支持专用扩展，以实现供应商进行创新。 打印票证框架和打印功能框架是整个打印架构框架的子集。

通过使用常用的架构来描述打印功能和属性，应用程序可以访问无需连接到特定的打印驱动程序或应用程序的打印功能。 公用架构还可以减少多义性的名称和功能，通过提供一组一致的名称和定义用于打印机功能和特性。

Microsoft Windows SDK 文档中记录了打印架构。

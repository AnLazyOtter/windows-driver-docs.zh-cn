# <a name="symbolic-names"></a>符号名称





符号名称是用作标识符的字符串。 符号名称具有以下特征：

-   第一个字符必须是大写或小写字母。

-   后的第一个字符的名称可以包括大写和小写字母、 数字和下划线字符 ( \_ )。

-   名称是区分大小写。

-   在名称中的字符数不受限制。

GPD 语言预定义了大量的符号名称。 预定义的名称存在[打印机属性](printer-attributes.md)，[命令名](command-names.md)，[标准功能](standard-features.md)，[标准选项](standard-options.md)，和[标准变量](standard-variables.md)。

您可以定义符号名称[自定义功能](customized-features.md)，[自定义选项](customized-options.md)，并[宏](macros.md)。

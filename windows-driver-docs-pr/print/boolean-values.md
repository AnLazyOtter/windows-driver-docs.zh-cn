# <a name="boolean-values"></a>布尔值





布尔值定义为相等的预定义常量的任一 **，则返回 TRUE**或**FALSE**。 按照约定，所有的符号名称[打印机属性](printer-attributes.md)的接受布尔值结尾为问号字符，如以下示例所示：

```cpp
*RotateRasterData?: FALSE
```

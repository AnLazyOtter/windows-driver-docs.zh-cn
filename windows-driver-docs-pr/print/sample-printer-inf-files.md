# <a name="sample-printer-inf-files"></a>示例打印机 INF 文件





示例打印机 INF 文件提供与 Windows Driver Kit (WDK)，位于\\src\\打印\\inf\\driver5 子目录。 此文件是说明了用于安装 Unidrv 和 Pscript 微型驱动程序的 INF 文件布局的模板。 将驱动程序文件的名称替换为占位符文件名称。

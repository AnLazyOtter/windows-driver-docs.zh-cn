# <a name="print-driver-manifests"></a>打印驱动程序清单





打印驱动程序程序包可能包含两种类型的清单：

-   **应用程序清单**作为安全功能是用于所有 Windows Vista 驱动程序。 应用程序清单的详细信息，请参阅用于编写设备安装应用程序的指导原则。

-   **清单主题**是用于控制打印应用程序 UI 的外观的打印驱动程序的选择。

有关对主题清单的详细信息，请参阅[使用主题清单](using-theme-manifests.md)。

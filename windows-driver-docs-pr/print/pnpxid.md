# <a name="pnpxid"></a>PnPXID


`PnPXID`命令返回 Plug and Play (PnP) 标识符 (ID) 中的主键的函数发现密钥形式\_PNPX\_id。 此键返回服务或设备的唯一的标识符。 输出缓冲区包含以 NULL 结尾的字符串表示的标识符。

有关函数发现密钥的详细信息，请参阅 Microsoft Windows SDK 文档。

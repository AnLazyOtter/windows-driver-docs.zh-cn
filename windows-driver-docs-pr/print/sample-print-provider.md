# <a name="sample-print-provider"></a>示例打印提供程序





**警告**  从 Windows 10 开始，支持第三方打印提供商的 Api 已弃用。 Microsoft 不建议到第三方打印提供商的任何投资。 此外，在 Windows 8 和更高版本的 v4 打印驱动程序模型是可用的产品，第三方打印提供商不可能创建，或管理使用 v4 打印驱动程序的队列。

 

使用 Windows Driver Kit (WDK) 中包含示例打印提供程序。 该示例不完全正常运行的打印提供程序;它是包含的主干实现的模板[函数定义的打印提供商](functions-defined-by-print-providers.md)。 示例文件都位于\\Src\\打印\\Pp WDK 的子目录。

# <a name="installing-driver-specific-files"></a>安装驱动程序特定的文件





文件与相关联的打印机驱动程序通过包括打印机的 INF 文件中。 有关详细信息，请参阅[打印机 INF 文件](printer-inf-files.md)。

若要提供指向并打印支持 Windows 95/98/98 客户端连接到 Windows 2000 和更高版本的服务器，请参阅中的"先前名称"数据部分的说明[打印机 INF 文件的数据部分](printer-inf-file-data-sections.md)。

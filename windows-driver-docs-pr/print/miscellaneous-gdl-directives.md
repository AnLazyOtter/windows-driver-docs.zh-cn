# <a name="miscellaneous-gdl-directives"></a>其他 GDL 指令


GDL 提供了两个其他指令：

-   \*IgnoreBlock 使 GDL 分析器忽略值和构造的内容。

-   \*%（星号加上百分号） 将导致分析器忽略该行的其余部分。 用于添加注释，可以使用此指令。

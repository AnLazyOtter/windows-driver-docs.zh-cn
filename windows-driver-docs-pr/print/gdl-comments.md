# <a name="gdl-comments"></a>GDL 注释


一个*注释*可以出现的任何位置的[非文本空白](gdl-whitespace-characters.md)可以出现。 注释开头星号和百分比符号 (\*%)并遇到 linebreak 序列或继续符 linebreak 时结束。

整个注释将被视为非文本空格。 终止 linebreak 序列不是注释的一部分。 注释可以出现在嵌套的上下文中，但在注释中识别任何上下文。

# <a name="consumables"></a>Consumables


架构路径：\\Printer.Consumables

节点类型： 属性

消费品属性包含有关所有设备中可使用提供的信息。 这包括打印过程中使用的项，但不包含可打印的媒体。

消费品属性是父[\[名称\]](-name-.md)属性。

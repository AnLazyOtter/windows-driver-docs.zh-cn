# <a name="uwp-device-apps-for-printers"></a>适用于打印机的 UWP 设备应用


UWP 应用的打印机的设备应用程序允许 Ihv 突出显示自定义的打印机首选项和通知支持通过其设备的特殊功能。

UWP 的设备应用程序是由 Microsoft Store 分发，而且为用户且已选择要下载应用程序自动安装。 UWP 应用的打印机的设备应用程序可能会提供有关在 Windows 上的任何打印机。

使用 v4 打印驱动程序安装的打印机可以提供自定义的打印机首选项体验支持 printTaskSettings 协定或通过支持 backgroundTasks 协定支持打印机通知。 有关详细信息，请参阅[开发 UWP 设备应用用于打印](https://docs.microsoft.com/windows-hardware/drivers/devapps/windows-store-device-apps-for-printers)。

# <a name="printer-forms-support"></a>打印机窗体支持


Microsoft 的打印机驱动程序可以添加自定义窗体以支持在打印机中的特殊窗体和纸张大小。 您可以通过调用这些窗体添加到系统打印机窗体数据库**AddForm**从每个新的窗体的打印机驱动程序。 Unidrv 打印机驱动程序还可以指定中的其他窗体[GPD 文件](introduction-to-gpd-files.md)打印机驱动程序。

有关如何将窗体添加到窗体数据库的详细信息，请参阅[将窗体添加到窗体数据库](adding-forms-to-the-forms-database.md)。

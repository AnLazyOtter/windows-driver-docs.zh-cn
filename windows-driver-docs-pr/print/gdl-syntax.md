# <a name="gdl-syntax"></a>GDL 语法


GDL 分析器处理 GDL 条目的流。 每个条目由 linebreak 序列或构造分隔符分隔。 GDL 流可能位于单个文件中，也可能会通过一系列逻辑上等同于一个流，其中包含单个文件中的文件来构造。

有两种类型的 GDL 条目： 属性和构造。 *GDL 属性*是简单的关键字和值对。 *构造 GDL*包含一系列数据条目调用后跟 GDL 特性*构造*。 构造一系列分隔的一系列*构造分隔符。*

空白字符可以出现在之间 GDL 条目和注释可以出现在 GDL 的流。

GDL 使用字符串来表示数据。

# <a name="device-stage-document-device-development-kit"></a>设备阶段文档设备开发工具包

设备阶段文档设备开发人员工具包包含以下信息：

- **Windows 7 设备阶段文档设备开发指南**-详细的有关如何将设备标识、 添加兼容的任务，请使用状态和其他高级创作主题的文档的设备特定开发主题的文档。

- **Windows 7 设备阶段规划工作表**-文档设备规划工作表的列表所需和可选值时，需要规划，构建 Device Stage 体验。

- **记录设备教程**-分步指南如何编辑文档设备模板创建新的设备的自定义设备阶段体验。

- **记录设备模板**-Device Stage 模板适用于文档设备。

# <a name="angle"></a>角度


架构路径：\\Printer.Finishing.Staple.Angle

节点类型： 属性

说明： 此属性包含所有适用于应用的目的是输出页面的装订方向的值项。

Angle 属性包含两个对子值：CurrentValue 和支持。

### <a name="span-idcurrentvaluespanspan-idcurrentvaluespan-currentvalue"></a><span id="currentvalue"></span><span id="CURRENTVALUE"></span> CurrentValue

架构路径：\\Printer.Finishing.Staple.Angle:CurrentValue

节点类型： 值

数据类型： BIDI\_INT

说明： staples 要应用的当前 （默认值） 方向。

允许使用以下值：

水平

垂直

倾斜

unknown

### <a name="span-idsupportedspanspan-idsupportedspan-supported"></a><span id="supported"></span><span id="SUPPORTED"></span> 支持

架构路径：\\Printer.Finishing.Staple.Angle:Supported

节点类型： 值

数据类型： BIDI\_字符串

说明： 一个以逗号分隔的列表进行装订角度支持的所有值。

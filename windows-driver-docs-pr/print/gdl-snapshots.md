# <a name="gdl-snapshots"></a>GDL 快照


如果没有任何[GDL 架构](gdl-schemas.md)，则[GDL 分析器](gdl-parser.md)可以创建[XML 快照](xml-structure-of-gdl-snapshots.md)指定的源数据的[GDL 配置](gdl-configurations.md)。 如果没有架构，分析器将验证仅源文件的语法正确无误。 它不能确定如果缺少必要的数据或树的结构符合客户端的预期。 无架构，由 GDL 分析器生成的快照包含的原始值，在源 GDL 文件中定义。

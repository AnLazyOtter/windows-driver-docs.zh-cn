# <a name="device-stage-for-document-devices"></a>Device Stage 文档设备


Windows 设备体验是一系列功能，可改进用户发现和使用连接到其 PC 的设备的方式。 系统和平台功能，可定义如何设备制造商提供其设备和相关的应用程序和服务-在 Windows 中的支持这些用户功能。

设备阶段提供了丰富的平台用于开发和分发特定于设备的体验。 使用设备阶段，可以创建与品牌、 功能和服务在设备--匹配所有通过定义少量的 XML 文件和图形的体验。 设备阶段才支持有限数量的设备类。

本文档介绍适用于文档设备的设备类特定功能。

有关最新信息，请参阅[Windows 硬件开发人员中心](https://go.microsoft.com/fwlink/p/?linkid=154165)。

### <a name="document-device-class"></a>设备的文档类

文档设备包括打印机、 扫描仪和多功能打印机 (MFP)。 除了提供设备阶段平台的基础知识，Windows 7 还包括各种打印和扫描特定的功能，很容易就能包含 Device Stage 体验中。

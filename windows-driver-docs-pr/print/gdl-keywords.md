# <a name="gdl-keywords"></a>GDL 关键字


一个*GDL 关键字*是一个令牌，从集中的字符组成\[A-Z、 a-z、 0-9 \*， \_，？\]。 按照约定，大多数关键字开头有星号 (\*)。 关键字的长度没有限制。

GDL 关键字是必需的任何 GDL 条目。 关键字后跟一个可选[值](gdl-values.md)。 如果提供一个值，则必须由冒号 （:） 分隔的关键字和值。 可选[空白字符](gdl-whitespace-characters.md)还可以显示关键字和值之间。

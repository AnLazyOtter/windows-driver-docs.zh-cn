# <a name="compressing-raster-data"></a>压缩光栅数据





Unidrv 支持光栅数据压缩，使用以下三种格式：

-   标记图像文件格式 (TIFF)，版本 4.0

-   增量行压缩 （金）

-   东亚运行长度编码 (FE RLE)

这些方法中定义*HP PCL 5 打印机语言技术参考*。 （此资源可能不会在某些语言和国家/地区中可用。）

可以在 GPD 文件以指示其这些压缩设置您的打印机支持的格式中包括的条目。 你还可以提供您自己的压缩算法。

有关光栅数据压缩的详细信息，请参阅以下主题：

[使用 Unidrv 支持压缩](using-unidrv-supported-compression.md)

[使用自定义的压缩](using-customized-compression.md)

# <a name="plug-and-play-for-printers"></a>打印机即插即用





本部分介绍在 Microsoft Windows 2000 及更高版本的打印机插的设计。 打印插设计是类似于 Windows 95 设计的。 Windows 2000 和更高版本的操作系统版本支持插到 LPT 端口、 USB 端口、 红外端口和并行端口连接的打印机。

有三个系统组件负责插打印机安装：

1.  [打印机安装和插管理器](printer-installation-and-the-plug-and-play-manager.md)

2.  [Plug and Play 总线枚举器](plug-and-play-bus-enumerators.md)

3.  [安装自定义即插即用打印机驱动程序](installing-a-custom-plug-and-play-printer-driver.md)

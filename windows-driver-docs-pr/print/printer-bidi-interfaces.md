# <a name="printer-bidi-interfaces"></a>打印机 Bidi 接口

这些接口使 V4 打印驱动程序模型建立和维护与 USB 打印机的双向通信。 这些接口才可通过在打印后台处理程序中运行的 JavaScript 代码访问。 

有关后台处理程序的详细信息，请参阅[打印后台处理程序体系结构](https://docs.microsoft.com/windows-hardware/drivers/print/print-spooler-architecture)并[打印后台处理程序组件](https://docs.microsoft.com/windows-hardware/drivers/print/print-spooler-components)。

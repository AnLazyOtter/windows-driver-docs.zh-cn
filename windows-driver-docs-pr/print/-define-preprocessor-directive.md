# <a name="define-preprocessor-directive"></a>\#定义预处理器指令

```GDL
#Define: symbol
```

\#定义添加*符号*预处理器符号字典的值。 *符号*可以是任何值令牌并且是必需的。 可以多次添加相同的符号。

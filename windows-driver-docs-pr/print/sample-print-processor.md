# <a name="sample-print-processor"></a>示例打印处理器





源代码 Genprint.dll，接受示例打印处理器[EMF 数据](emf-data-type.md)，[原始数据](raw-data-type.md)，并[文本数据](text-data-type.md)作为输入，包括 Windows Driver Kit (WDK) 中。 代码位于\\Src\\打印\\Genprint WDK 中的子目录。

**请注意**  编译此打印处理器时，设置 Unicode 标志\#定义 UNICODE。

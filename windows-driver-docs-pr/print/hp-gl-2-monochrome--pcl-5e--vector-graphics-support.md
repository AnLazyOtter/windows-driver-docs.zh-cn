# <a name="hp-gl2-monochrome-pcl-5e-vector-graphics-support"></a>HP/2 单色 (PCL-5e) 矢量图形支持





HP/2 单色 (PCL 5e) 向量图形中的支持运行 Windows XP 及更高版本的 Unidrv 允许打印机驱动程序，以创建作业的数据是最适合于设备的格式： 矢量和光栅图形的图形。 这通常会导致更少的系统开销，更低的输出数据传输和更快的打印吞吐量。

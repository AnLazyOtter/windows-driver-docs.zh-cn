# <a name="configuration"></a>配置


架构路径：\\Printer.Configuration

节点类型： 属性

配置属性包含有关如何配置设备信息： 内存、 是否已安装硬盘，和是否安装了双面打印单元的量。 仅当物理设备的已修改配置，通常通过实际硬件中的更改时，此数据发生更改。

配置属性是所有的属性和配置部分中的值的标头。

配置属性是父级的以下属性：

[内存](memory.md)

[HardDisk](harddisk.md)

[DuplexUnit](duplexunit.md)

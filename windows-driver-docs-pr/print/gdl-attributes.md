# <a name="gdl-attributes"></a>GDL 属性


GDL 属性是一个关键字和值对。 *值*是一种数据，并*关键字*是只需一个标记，用于标识数据的一部分。 只要遵守 GDL 语法规则允许任意关键字和数据对。

下面的代码示例显示了 GDL 属性。

```cpp
*Weight: 330 pounds
```

在前面的示例中，"\*权重"是关键字，并且"330 磅"字符串值。

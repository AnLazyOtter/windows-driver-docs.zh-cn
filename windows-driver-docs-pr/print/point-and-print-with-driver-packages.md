# <a name="point-and-print-with-driver-packages"></a>点选打印与驱动程序包


连接到打印服务器的打印客户端可以使用点和打印复制安装整个驱动程序包。 使用的优点点和包使用打印的：

-   打印客户端上安装的驱动程序的所有可运行的组件。

-   在打印客户端将检查驱动程序签名和驱动程序的完整性。

-   指向并打印更可信和托管环境中的管理员可以更好地控制。

点和打印过程则是 Windows Vista 中不同于早期版本的 Windows 由于驱动程序包的简介。 Windows Vista 使用包安装作为驱动程序安装的首选方法。 在版本的 Windows 早于 Windows Vista，指向并打印可能仅是基本的打印功能上安装客户端。

驱动程序软件包安装需要驱动程序存储区，这在早于 Windows Vista 的 Windows 版本上不可用。 当正在运行 Windows 的早期版本的客户端连接到 Windows Vista 的打印服务器、 打印服务器使用指向和打印，而包在客户端计算机上安装打印机。

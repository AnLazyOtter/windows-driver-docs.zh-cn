# <a name="gdl-directives-for-macros"></a>用于宏的 GDL 指令


GDL 具有定义宏的指令。 引用对宏是展开的行中获取由分析器解释的条目。

GDL 包含以下宏指令：

-   **\*宏**定义一组值宏。

-   **\*BlockMacro**定义块宏。

-   **\*InsertBlock**引用块宏

-   **=** （等号） 引用值登录时它后接值宏的名称。

宏指令的详细信息，请参阅[GDL 宏](gdl-macros.md)。
